Sequence Embedded Wallet es una solución de billetera no custodial que permite a los usuarios autenticarse con proveedores sociales, correo electrónico o como invitado. Está disponible a través de nuestros diversos SDKs como [Web](/sdk/headless-wallet/quickstart), [Unity](/sdk/unity/overview) y [Unreal](/sdk/unreal/introduction).

Estas billeteras sin interfaz se pueden crear bajo demanda para cada usuario y le permiten integrar una aplicación con una blockchain compatible con Ethereum, mientras usa construcciones familiares como APIs RPC. Además, no hay ventanas modales emergentes necesarias, confirmaciones del usuario y otras distracciones, por lo que puede proporcionar una experiencia de juego verdaderamente nativa y fluida.

Sequence Embedded Wallet ofrece billeteras realmente no custodiales y el sistema garantiza que cada billetera creada solo puede ser accedida con el consentimiento del propietario de la billetera. Esto es posible gracias al poder de [AWS Nitro Enclaves](https://aws.amazon.com/ec2/nitro/nitro-enclaves/) para manejar datos sensibles en un entorno seguro al que nadie puede acceder - ni siquiera Sequence.

Estas también se comparten entre experiencias, si se desea, por lo que sus usuarios tienen una dirección de billetera consistente en todos sus juegos y plataformas, sin importar qué inicio de sesión social estén usando.
