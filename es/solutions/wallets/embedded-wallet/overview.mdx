Sequence Embedded Wallet es una solución de wallet no custodiada que permite a los usuarios autenticarse con proveedores sociales, correo electrónico o como invitados. Está disponible a través de nuestros diferentes SDKs como [Web](/sdk/headless-wallet/quickstart), [Unity](/sdk/unity/overview) y [Unreal](/sdk/unreal/introduction).

Estas wallets headless pueden crearse bajo demanda para cada usuario y le permiten integrar una aplicación con una blockchain compatible con Ethereum, utilizando construcciones familiares como APIs RPC. Además, no hay modales emergentes necesarios, confirmaciones visibles para el usuario ni otras distracciones, por lo que puede ofrecer una experiencia de juego realmente nativa y fluida.

Sequence Embedded Wallet ofrece wallets realmente no custodiadas y el sistema garantiza que cada wallet creada solo puede ser accedida con el consentimiento del propietario de la wallet. Esto es posible gracias al uso de [AWS Nitro Enclaves](https://aws.amazon.com/ec2/nitro/nitro-enclaves/) para manejar datos sensibles en un entorno seguro al que nadie puede acceder, ni siquiera Sequence.

Estas también pueden compartirse entre experiencias, si así lo desea, para que sus usuarios tengan una dirección de wallet consistente en sus juegos y plataformas, sin importar qué inicio de sesión con redes sociales estén usando.

Cuando esté listo para comenzar la implementación, siga la [guía de inicio rápido](/sdk/headless-wallet/quickstart) o instale el SDK desde su gestor de paquetes favorito:

<CodeGroup>
  ```bash pnpm
  pnpm install @0xsequence/waas
  ```

  ```bash npm
  npm install @0xsequence/waas
  ```

  ```bash yarn
  yarn add @0xsequence/waas
  ```
</CodeGroup>
