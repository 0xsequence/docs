Agregar una smart wallet potenciada por Sequence a su dApp en React es un proceso sencillo. Esta guía describe los tres pasos principales para pasar de un proyecto nuevo a una wallet totalmente integrada.

## 🔑 Paso 1: Obtenga la clave de acceso de su proyecto
Primero, necesita una clave para conectar su dApp con la infraestructura de Sequence. Esta clave identifica su proyecto y habilita nuestros servicios.

**Acción:** Vaya a [sequence.build](https://sequence.build), regístrese o inicie sesión y cree un nuevo proyecto.

**Resultado:** Podrá obtener su `projectAccessKey` único.

## 📦 Paso 2: Instale el paquete principal del SDK `@0xsequence/connect`
Luego, agregue el paquete principal del Web SDK de Sequence a las dependencias de su proyecto. Este paquete contiene todos los hooks, componentes y el modal de wallet preconstruido necesarios.

**Acción:** En la terminal de su proyecto, ejecute el comando de instalación:

```bash
npm install @0xsequence/connect
```

**Resultado:** El SDK de Sequence ahora es una dependencia en su proyecto, listo para usarse.

## 🔗 Paso 3: Integre con su app React
Finalmente, envuelva su aplicación con nuestro proveedor de wallet. Esto hace que todas las funciones de conexión estén disponibles para cualquier componente dentro de su dApp.

**Acción:** Siga nuestra [Guía oficial de integración del Web SDK](/sdk/web/getting-started) para una guía detallada sobre cómo integrar.

**Resultado:** La wallet de Sequence ya está activa en su dApp. Puede usar de inmediato hooks de wagmi como `useConnect()` para permitir que los usuarios inicien sesión con inicio de sesión con redes sociales o correo electrónico e interactúen con la blockchain.

Vaya a la [Guía de integración del Web SDK](/sdk/web/getting-started) para una guía detallada sobre cómo integrar el Web SDK en una aplicación React/Next.js.
