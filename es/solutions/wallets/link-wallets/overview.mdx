# Resumen de Vinculación de Wallets
La Vinculación de Wallets permite que un desarrollador cree una asociación verificable entre dos wallets de un mismo usuario. Generalmente, esto es un vínculo entre una wallet embebida para un juego o aplicación y una wallet externa, como Metamask. Esto resuelve varios problemas para los juegos, tales como:
- Validar que un usuario posee un NFT, pero sin necesidad de mover su activo valioso para acceder a funciones exclusivas o recibir recompensas.
- Migración sencilla de fondos o liquidez desde una wallet externa hacia la wallet embebida del juego.
- Verificar la cantidad de tokens ERC20 que posee un usuario para otorgar derechos de gobernanza.

Usando los SDKs de Sequence, la sesión de la Embedded Wallet puede iniciarse desde casi cualquier plataforma en móvil, web, PC o consola, para que los usuarios disfruten de una experiencia de juego fluida. Sin embargo, cuando se desee, el usuario puede ser dirigido a una sesión de navegador externa para vincular wallets adicionales.
