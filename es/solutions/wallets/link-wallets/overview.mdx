# Visión General de la Vinculación de Billeteras

La Vinculación de Billeteras permite a un desarrollador crear una asociación verificable entre dos billeteras de un mismo usuario. Generalmente, esto es un vínculo entre una billetera integrada para un juego o aplicación y una billetera externa, como Metamask. Esto resuelve varios problemas para los juegos como:

* Validar que un usuario posee un NFT, pero no quiere mover su valioso activo para control de acceso basado en tokens o distribuciones de recompensas.
* Fácil migración para el flujo de fondos o liquidez desde una billetera externa a la billetera integrada para un juego.
* Verificar la cantidad de tokens ERC20 que posee un usuario para otorgar derechos de gobernanza.

Usando los SDK de Sequence, la sesión de la billetera integrada puede ser iniciada desde casi cualquier plataforma como Móvil, Web, PC o Consola para que los usuarios puedan disfrutar de una experiencia de juego sin problemas. Sin embargo, cuando se desee, un usuario es dirigido a una sesión de navegador externa para vincular billeteras adicionales.
