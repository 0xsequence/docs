---

Configurar y personalizar Ecosystem Wallet para su marca a través de Sequence Builder es extremadamente sencillo.

<Steps>
  <Step title="Configure su Embedded Wallet">
    Configure embedded wallet a través de [Sequence Builder](https://sequence.build/project/default/embedded-wallet) en su proyecto. Para más detalles sobre cada configuración, consulte nuestra [documentación](/solutions/builder/embedded-wallet) sobre embedded wallet.
  </Step>

  <Step title="Configure Web SDK con su marca">
    [Personalice la experiencia de inicio de sesión](https://sequence.build/project/default/embedded-wallet/edit) para la marca de su ecosistema en Web SDK.
  </Step>

  <Step title="Comparta claves con socios">
    Para habilitar a los socios del ecosistema, simplemente comparta las claves de embedded wallet (WaaS y Project Access Key) con socios de confianza, así como el diseño para Web SDK. Todo lo que deben hacer es integrar [Web SDK](/sdk/web/getting-started) en su aplicación usando las claves proporcionadas.
  </Step>
</Steps>

<Warning>
  Tenga en cuenta que solo recomendamos compartir claves con socios de confianza para garantizar la seguridad de los usuarios.
</Warning>
