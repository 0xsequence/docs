Sequence es la mejor manera de incorporar jugadores a su juego habilitado para blockchain. Tenemos dos opciones principales para comenzar: la [Embedded Wallet](/sdk/headless-wallet/quickstart) y la [Ecosystem Wallet](/solutions/wallets/ecosystem/overview). Ambas son completamente no custodial.

La [Embedded Wallet](/sdk/headless-wallet/quickstart) está diseñada para una experiencia de juego tradicional. Sus ventajas son:
- Autenticar usuarios con proveedores sociales comunes o correo electrónico.
- Soporte para Unity, Unreal, Web y Móvil.
- Pop-ups, modales y firma de transacciones opcionales para ofrecer una experiencia de usuario fluida.
- Totalmente no custodial, wallet de contrato inteligente.
- Inicio de sesión listo para usar con el [Web SDK](/sdk/web/overview) o cree su propio proceso.

Por otro lado, la [Ecosystem Wallet](/solutions/wallets/ecosystem/overview) está dirigida a cadenas y aplicaciones que desean construir todo un ecosistema.

Nuestra biblioteca [Web SDK](/sdk/web/overview) es una interfaz personalizable escrita en Typescript, diseñada para usarse en el navegador. Es ideal cuando desea admitir usuarios con wallets existentes como Metamask, ofrecer una experiencia lista para usar, y también llegar a millones de nuevos usuarios que prefieren la comodidad del inicio de sesión con redes sociales.

Por último, ofrecemos la [Vinculación de Wallets](/sdk/web/getting-started) para su ecosistema, que permite a los desarrolladores crear una asociación verificable entre dos wallets de un mismo usuario. Generalmente, esto es un vínculo entre una wallet embebida para un juego o aplicación y una wallet externa, como Metamask, para que pueda consultar fácilmente todas las wallets asociadas a un solo usuario, a través de juegos, ecosistemas y blockchains.
