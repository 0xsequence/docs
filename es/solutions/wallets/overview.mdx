Sequence es la mejor manera de incorporar usuarios a su aplicación habilitada para blockchain. Tenemos dos opciones principales para comenzar: la [Embedded Wallet](/sdk/headless-wallet/quickstart) y la [Ecosystem Wallet](/solutions/wallets/ecosystem/overview). Ambas son completamente no-custodiales.

La [Embedded Wallet](/sdk/headless-wallet/quickstart) ofrece una experiencia whitelabel/in-app fluida. Sus ventajas son:
- Autentique usuarios con proveedores de inicio de sesión con redes sociales populares o correo electrónico.
- Soporte para Web, Mobile, Unity y Unreal.
- Pop-ups, modales y firma de transacciones opcionales para ofrecer una experiencia de usuario fluida.
- Totalmente no-custodial, wallet de smart contract con abstracción de cuentas.
- Inicio de sesión listo para usar con [Web SDK](/sdk/web/overview) o cree sus propios flujos.

Por otro lado, la [Ecosystem Wallet](/solutions/wallets/ecosystem/overview) está dirigida a cadenas y aplicaciones que desean construir todo un ecosistema.

Nuestra biblioteca [Web SDK](/sdk/web/overview) es una interfaz personalizable escrita en Typescript, diseñada para usarse en el navegador. Es ideal cuando desea admitir usuarios con wallets existentes como Metamask, ofrecer una experiencia lista para usar, y también llegar a millones de nuevos usuarios que prefieren la comodidad del inicio de sesión con redes sociales.

Por último, ofrecemos [Wallet Linking](/sdk/web/getting-started) para su ecosistema, lo que permite a los desarrolladores crear una asociación verificable entre dos wallets de un mismo usuario. Generalmente, esto es un enlace entre una embedded wallet de una aplicación y una wallet externa, como Metamask, para que pueda consultar fácilmente todas las wallets asociadas a un usuario, a través de aplicaciones, ecosistemas y blockchains.
