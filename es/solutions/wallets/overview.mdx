Sequence es la mejor manera de incorporar jugadores a tu juego habilitado para blockchain. Tenemos dos opciones principales para comenzar: la [Embedded Wallet](/sdk/headless-wallet/quickstart) y la [Ecosystem Wallet](/solutions/wallets/ecosystem/overview). Ambas son completamente no custodiales.

La [Embedded Wallet](/sdk/headless-wallet/quickstart) está diseñada para una experiencia de juego tradicional. Las ventajas son:

* Autentifica usuarios con proveedores sociales comunes o correo electrónico.
* Soporte para Unity, Unreal, Web y Móvil.
* Ventanas emergentes opcionales, modales y firma de transacciones para una experiencia de usuario fluida.
* Billetera de contratos inteligentes totalmente no custodial.
* Inicio de sesión listo para usar con [Web SDK](/sdk/web/overview) o construye tu propio flujo.

Por otro lado, la [Ecosystem Wallet](/solutions/wallets/ecosystem/overview) atiende a cadenas y aplicaciones que quieren construir un ecosistema completo.

Nuestra [Web SDK](/sdk/web/overview) es una biblioteca de interfaz de usuario personalizable escrita en Typescript, diseñada para una experiencia en navegador. Esto es excelente cuando quieres dar soporte a usuarios con billeteras existentes como Metamask, tener una experiencia lista para usar, así como millones de potenciales nuevos usuarios que prefieren la conveniencia del inicio de sesión social.

Por último, proporcionamos [Wallet Linking](/sdk/web/getting-started) para tu ecosistema que permite la capacidad de que un desarrollador cree una asociación verificable entre dos billeteras de un solo usuario. Generalmente, esto es un enlace entre una billetera integrada para un juego o aplicación y una billetera externa, como Metamask, para que puedas consultar fácilmente todas las billeteras asociadas a un solo usuario - a través de juegos, ecosistemas y blockchains.
