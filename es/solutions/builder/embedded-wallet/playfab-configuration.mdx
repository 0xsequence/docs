sidebarTitle: Configuración de PlayFab
---

Para integrar la autenticación de PlayFab en la instancia de Embedded Wallet, debe configurar un juego en el panel de [Microsoft PlayFab Developer](https://developer.playfab.com/), comenzando con los siguientes pasos:

<Steps>
  <Step title="Registrarse / Iniciar sesión">
    Primero, acceda a la página de inicio de sesión [aquí](https://developer.playfab.com/):

    <Frame>
      ![Página de inicio de sesión de PlayFab](/images/guides/playfab/playfab_developer_sign_in.png)
    </Frame>
  </Step>

  <Step title="Crear un nuevo estudio">
    Después de iniciar sesión, vaya a [este enlace para crear un estudio](https://developer.playfab.com/en-us/r/studio/new) que generará un `Title ID` para usted.
  </Step>

  <Step title="Obtener un Title ID">
    Después de crear un estudio, puede obtener el `Title ID` para incluirlo en el SDK en [esta página](https://developer.playfab.com/en-us/my-games)

    <Frame>
      ![Obtener un Title ID](/images/guides/playfab/playfab_developer_retrieve_title_id.png)
    </Frame>

    Luego agregue el Title ID a su [configuración de Embedded Wallet](https://sequence.build/project/default/wallet/embedded-wallet) como un nuevo proveedor de inicio de sesión para completar la configuración.

    ¡La configuración está lista! Por último, consulte nuestra implementación de referencia para web para ver cómo [obtener el ticket de Playfab](/sdk/headless-wallet/authentication#authenticating-with-playfab) y pasarlo a Sequence para autenticar al usuario.
  </Step>
</Steps>
