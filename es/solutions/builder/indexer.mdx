Sequence Builder equipa a los desarrolladores de juegos con un indexador que recopila datos de las siguientes redes: Ethereum (y Sepolia), Optimism, Base, Gnosis Chain, Polygon (y Polygon Mumbai), Polygon zkEVM, Arbitrum One, Arbitrum Nova, Gnosis Chain, BNB Smart Chain (y Smart Chain Testnet), Avalanche (y Avalanche Testnet), Oasys Homeverse (y Oasys Homeverse Testnet).

#### ¿Qué datos puedo consultar para mi juego?

Builder elimina el estrés de recopilar datos on-chain para tu juego. Simplemente selecciona los datos que deseas del indexador, y este genera el código necesario en fragmentos.

Obtén diferentes tipos de datos del indexador:

* tokens y coleccionables que posee cualquier billetera
* suministro total para cualquier token o coleccionable
* el historial de transacciones y transferencias para cualquier dirección de billetera
* metadatos del contrato para cualquier token o colección
* metadatos para cualquier coleccionable
* eventos de cambio de saldo para cualquier token o colección

#### ¿Cuánto código tendré que escribir para consultar?

Sequence Indexer ofrece varias funciones que mejoran tus capacidades de datos on-chain sin escribir una línea de código, proporcionando fragmentos de código útiles de los casos de uso más comunes. Cuenta con una interfaz fácil de usar que puedes utilizar para obtener datos importantes, como los tokens que posee una billetera específica bajo un contrato específico, por ejemplo.

<Tip>
  También puedes aprovechar nuestro Indexer programáticamente a través de nuestras [APIs](/api-references/indexer/overview) o SDKs como [Unity](/sdk/unity/power/read-from-blockchain).
</Tip>
