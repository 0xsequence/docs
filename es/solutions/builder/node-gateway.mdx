Sequence Builder ofrece un Blockchain RPC completo que le proporciona toda la información necesaria para gestionar su infraestructura de nodos, desde la compatibilidad con su módulo Web3 favorito hasta los requisitos de datos en tiempo real y alto rendimiento para su juego.

##### ¿Qué tipo de rendimiento puedo esperar para mi juego?
Sequence Node Gateway está diseñado para potenciar la infraestructura de su juego. Node Gateway agrega múltiples proveedores de nodos, cambiando automáticamente entre ellos para asegurar la disponibilidad de datos correctos y sincronizados, dándole la tranquilidad de que sus aplicaciones descentralizadas funcionarán sin problemas y con baja latencia para sus jugadores.

No tiene que preocuparse por los picos de tráfico: nuestra infraestructura de nodos es robusta, ofrece mayor escalabilidad y tolerancia a fallos. Además, proporcionamos detección y recuperación automática ante fallas de nodos.

##### ¿Qué pueden hacer para simplificarme el proceso de integración?
Hemos diseñado nuestro panel para generar automáticamente claves de acceso API y URLs para cada red que tenga habilitada en la configuración. Esto facilita importar la URL en su módulo Web3 npm preferido. Además, nuestro panel le permite monitorear fácilmente el uso de cómputo y el consumo de créditos.
