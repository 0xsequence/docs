Sequence Builder ofrece un RPC de Blockchain completo que te proporciona toda la información necesaria para gestionar tu infraestructura de nodos, desde la compatibilidad con tu módulo Web3 favorito hasta tus requisitos de datos performantes en tiempo real para tu juego.

##### ¿Qué tipo de rendimiento puedo esperar para mi juego?

Sequence Node Gateway está diseñado para potenciar tu infraestructura de juegos. Node Gateway agrega múltiples proveedores de nodos, alternando automáticamente entre ellos para asegurar la disponibilidad de datos correctos y sincronizados — dándote la seguridad de que tus aplicaciones descentralizadas funcionan sin problemas con baja latencia para tus jugadores.

No te preocupes por el tráfico pico – nuestra infraestructura de nodos es robusta, ofreciendo mayor escalabilidad y tolerancia a fallos. Además, proporcionamos detección y recuperación automática de fallos de nodos.

##### ¿Qué pueden hacer para simplificar el proceso de integración para mí?

Hemos diseñado nuestro panel de control para generar automáticamente claves de acceso API y URLs para cada red que tengas habilitada en la configuración. Esto facilita la importación de la URL en tu módulo npm Web3 preferido. Además, nuestro panel de control facilita el monitoreo de tu uso de computación y consumo de créditos.
