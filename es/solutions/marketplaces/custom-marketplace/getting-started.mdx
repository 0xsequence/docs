Utilice los protocolos de marketplace de Sequence para construir un marketplace que permita [Shopify-like](https://www.shopify.com/) control sobre el desarrollo de código personalizado con el código fuente de Sequence.

El [código base que puedes clonar](https://github.com/0xsequence-demos/marketplace-boilerplate) desde CLI y comenzar inmediatamente permite a los usuarios realizar ofertas y listados de elementos coleccionables. Personaliza en qué red están tus colecciones usando [esta guía](/solutions/collectibles/contracts/deploy-an-item-collection) para desplegar una colección usando el [Sequence Builder](https://sequence.build). Agrega tus colecciones a tu marketplace siguiendo [esta guía](/solutions/marketplaces/white-label-marketplace/guide) desde el paso 4, `Add a collectible to Marketplace`, en adelante.

## Inicio Rápido: Boilerplate de Marketplace

Para comenzar rápidamente con un comando de una línea en tu terminal bash, usa el siguiente comando y sigue las indicaciones para inicializar el entorno:

```shell
npx sequence-cli marketplace create-marketplace-boilerplate
```
