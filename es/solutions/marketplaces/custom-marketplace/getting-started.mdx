Utilice los protocolos de marketplace de Sequence para construir un marketplace que le permita un control tipo [Shopify](https://www.shopify.com/) sobre el desarrollo de código personalizado con el código fuente de Sequence.

La [base de código que puede clonar](https://github.com/0xsequence/marketplace-boilerplate) desde la CLI y comenzar de inmediato permite a los usuarios realizar ofertas y listados de objetos coleccionables. Personalice en qué red están sus colecciones usando [esta guía](/solutions/collectibles/contracts/deploy-an-item-collection) para desplegar una colección con el [Sequence Builder](https://sequence.build). Agregue sus colecciones a su marketplace siguiendo [esta guía](/solutions/marketplaces/white-label-marketplace/guide) desde el paso 4, `Add a collectible to Marketplace`, en adelante.

## Guía rápida: Marketplace Boilerplate
Para comenzar rápidamente con un solo comando en su terminal bash, use el siguiente comando y siga las indicaciones para inicializar el entorno:

```shell
npx sequence-cli marketplace create-marketplace-boilerplate
```
