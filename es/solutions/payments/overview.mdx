## Opciones de un vistazo

<CardGroup>
  <Card title="Trails" href="/solutions/payments/trails">
    Pagos cross-chain de un solo clic desde cualquier wallet, cualquier token. Orquesta los swaps/bridges necesarios y ejecuta la llamada final en la cadena de destino.
  </Card>

  <Card title="Shop" href="/solutions/payments/shop/overview">
    Tienda de ventas primarias preconstruida respaldada por contratos auditados. Configure artículos, precios, límites y horarios con poco código.
  </Card>

  <Card title="Marketplace" href="/solutions/payments/marketplace/overview">
    Marketplace white-label más APIs para listados, ofertas y pujas. Agregue liquidez externa manteniendo la experiencia en su app.
  </Card>

  <Card title="Checkout SDK" href="/solutions/payments/checkout-sdk">
    Checkout con tarjeta de crédito/débito listo para usar en compras de NFTs a través de socios de pago compatibles. Funciona junto a flujos de wallet y Trails.
  </Card>
</CardGroup>

## Guía de decisión

- ¿Necesita **pagos cross-chain de un solo clic**? Use <a href="/solutions/payments/trails">Trails</a>.
- ¿Quiere una **tienda configurable para ventas primarias**? Use <a href="/solutions/payments/shop/overview">Shop</a>.
- ¿Necesita **ventas secundarias o flujos de trading personalizados**? Use <a href="/solutions/payments/marketplace/overview">Marketplace</a>.
- ¿Quiere **pagos con tarjeta para NFTs** a través de socios? Use <a href="/solutions/payments/checkout-sdk">Checkout SDK</a>.
