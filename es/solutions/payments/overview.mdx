## Sequence Pay

Sequence Pay aborda la complejidad de manejar pagos en múltiples monedas, intercambios, on-ramping y transacciones en juegos blockchain, marketplaces, launchpads y más. La solución introduce métodos simplificados y flujos de interfaz de usuario en Web SDK para web, integraciones directamente con motores de juego a través de nuestros SDKs de Unity y Unreal, así como APIs para Sequence Smart Swaps.

* [Smart Swap](/sdk/web/guides/smart-swaps) con Sequence aprovechando las monedas en las carteras de los jugadores a otra moneda en la misma cadena. Los desarrolladores pueden definir la moneda objetivo y Sequence se encargará de todo, incluyendo la UI y el flujo a través de Web SDK así como un conjunto de [APIs](/api-references/infrastructure/overview). El poder viene en conjunto con el aprovechamiento de las Sequence Wallets donde podemos detectar automáticamente la moneda correcta para intercambiar y agrupar automáticamente las transacciones para simplificar la experiencia de usuario. ¡Di adiós a las transacciones separadas de `approve` y `transfer` transacciones!
* Compra un token ERC721 o ERC1155 a través de [Sequence Checkout](/sdk/web/guides/checkout) usando un contrato de ventas primarias o secundarias como un marketplace con las siguientes opciones de pago:
  * Comprar con cualquier criptomoneda en la cartera.
  * Transferir fondos desde otra cartera a una cartera Sequence y comprar.
  * Pagar usando una tarjeta de crédito o débito que detectará inteligentemente el proveedor correcto para cada región, cadena y moneda.
  * Pagar con otra criptomoneda en una cartera haciendo un intercambio y compra automatizados.
* [On-ramp](/sdk/web/guides/on-ramp) con Sequence para comprar criptomonedas con tarjeta de crédito o débito usando un proveedor como Transak en [Unity](/sdk/unity/monetization/onboard-user-funds), [Unreal](/sdk/unreal/onboard-user-funds) o [Web](/sdk/web/guides/on-ramp).
