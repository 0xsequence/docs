Sequence Marketplace ofrece a los desarrolladores un marketplace boilerplate con una base sólida, permitiendo a los usuarios comprar, listar, hacer ofertas y aceptar ofertas desde el inicio. Alternativamente, puede usar nuestro Marketplace SDK para integrarlo en su aplicación existente, o usar directamente nuestras APIs de Marketplace para construir su propio marketplace personalizado.

## Guía rápida: Marketplace Boilerplate
Para comenzar rápidamente con un solo comando en su terminal bash, use el siguiente comando y siga las indicaciones para inicializar el entorno:

```shell
npx sequence-cli marketplace create-marketplace-boilerplate
```
