Esta página describe todas las opciones para lanzar y operar un marketplace con Sequence. Elija el enfoque que se adapte a su cronograma y nivel de control deseado.

<CardGroup>
  <Card title="Marketplace alojado sin código" href="/solutions/payments/marketplace/hosted">
    Lance un marketplace white-label desde Builder sin programar. Configure comisiones, regalías (ERC-2981), colecciones y temas; luego publique.
  </Card>

  <Card title="Marketplace personalizado (Boilerplate)" href="/solutions/payments/marketplace/custom">
    Comience desde un boilerplate listo para producción. Mantenga control total sobre rutas, UI y lógica de negocio usando los contratos y servicios de Sequence.
  </Card>

  <Card title="Marketplace SDK" href="/sdk/web/marketplace-sdk/overview">
    Integre acciones de marketplace (listar, comprar, ofertas/pujas, cancelar, aceptar) directamente en su app. Cree flujos personalizados sin mantener una UI completa de marketplace.
  </Card>

  <Card title="APIs de Marketplace (listados agregados)" href="/api-references/marketplace/overview">
    Consulte listados normalizados agregados de su Sequence Marketplace más fuentes externas como OpenSea y Magic Eden, en las principales cadenas EVM.
  </Card>
</CardGroup>

## Cuándo usar cada opción

- **¿Quiere prototipar o lanzar de inmediato** sin recursos de ingeniería? → use el <a href="/solutions/payments/marketplace/hosted">Marketplace alojado sin código</a>.
- **¿Quiere control sobre la experiencia y reglas de negocio** pero no desea construir todo desde cero? → use el <a href="/solutions/payments/marketplace/custom">Marketplace personalizado (Boilerplate)</a>.
- **Si ya tiene una app y solo necesita acciones de marketplace** (por ejemplo, listar/comprar en su UI existente) → use el <a href="/sdk/web/marketplace-sdk/overview">Marketplace SDK</a>.
- **Necesita búsqueda/exploración a gran escala con liquidez externa** → use el <a href="/api-references/marketplace/overview">Marketplace APIs (Listados Agregados)</a>.

<Info>
  La agregación actualmente apunta a <strong>las principales cadenas EVM</strong> y normaliza resultados entre fuentes. La cobertura varía según la cadena y el marketplace; consulte la referencia de la API para campos y filtros exactos.
</Info>

## Capacidades

- <strong>Listados y Ofertas</strong>: precio fijo, ofertas/pujas, aceptar/cancelar.
- <strong>Regalías y Comisiones</strong>: aplique regalías ERC-2981 y comisiones de plataforma por colección o marketplace.
- <strong>Agregación</strong>: combine listados de Sequence con liquidez externa (por ejemplo, OpenSea, Magic Eden) mediante las APIs de Marketplace.
- <strong>Observabilidad</strong>: use el <a href="/solutions/indexer/overview">Indexer</a> para ventas, transferencias y analítica.
