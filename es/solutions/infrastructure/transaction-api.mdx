La **Transaction API** es un relayer unificado que envía transacciones en cadenas EVM con **patrocinio de gas**, **abstracción de comisiones**, **agrupamiento** y **procesamiento en paralelo**. Administra los nonces, estima el gas óptimo y reenvía transacciones cuando es necesario, para que usted pueda enfocarse en la lógica de su negocio.

<Info>
  Para detalles de los endpoints, consulte la referencia API: [Transactions API](/api-references/transactions/overview). Para uso con SDK, vea [TypeScript SDK](/sdk/typescript/overview) y [Go SDK](/sdk/go/overview). Un ejemplo completo de servidor está en [Guía para construir un servidor de relaying de transacciones](/guides/building-relaying-server).
</Info>

## Capacidades principales

- **Patrocinio de gas** — patrocine interacciones de usuarios a nivel de proyecto, contrato, token o dirección.
- **Abstracción de comisiones** — los usuarios pueden pagar comisiones en ERC-20 soportados (por ejemplo, USDC), o usted puede patrocinar totalmente.
- **Agrupamiento** — agrupe varias llamadas en una sola transacción onchain.
- **Procesamiento en paralelo** — envíe múltiples transacciones independientes de manera concurrente.
- **Compatible con cualquier contrato EVM** — no se requieren cambios en el contrato.

## Cuándo usarlo

- Quiere un envío de transacciones de nivel producción con **reintentos, precios de gas correctos y gestión de nonces**.
- Necesita **gas gratis** o **denominado en tokens** para los usuarios.
- Está construyendo **acciones disparadas desde el servidor** (por ejemplo, minteos, recompensas) o **meta-transacciones disparadas desde el cliente**.

## Ejemplo mínimo de cliente (TypeScript)

```ts
import { Session } from '@0xsequence/auth'
import { ethers } from 'ethers'

const chainId = 8453 // Base mainnet, for example

// Create a Sequence session for the current user or service signer
const session = await Session.singleSigner({ signer: myEOA, projectAccessKey: '<ACCESS_KEY>' })
const signer = session.account.getSigner(chainId)

// Call any contract function
const erc20 = new ethers.Contract('0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48', ['function transfer(address to, uint256 amount)'], signer)
const tx = await erc20.transfer('0xRecipient', 10n ** 6n) // 1 USDC
await tx.wait()
```

## Patrocinio de gas y tokens de comisión
Utilice **Builder** para configurar el patrocinio de gas para su proyecto. También puede hacer que el usuario final pague al relayer directamente incluyendo una transacción de **pago de comisión** en el paquete. Vea los tokens de comisión aceptados mediante `/rpc/Relayer/FeeTokens` en el relayer de la red. Más detalles disponibles en la [referencia API](/api-references/transactions/overview).
