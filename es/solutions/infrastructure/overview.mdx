Esta sección cubre los componentes principales que puede combinar o usar juntos como un backend escalable completo para aplicaciones onchain.

<CardGroup>
  <Card title="Indexer" href="/solutions/indexer/overview">
    Saldos, transferencias, precios y eventos de contratos en tiempo real a través de cadenas EVM. Lecturas de baja latencia con webhooks y suscripciones a eventos.
  </Card>

  <Card title="API de transacciones" href="/solutions/infrastructure/transaction-api">
    Relayer/dispatcher unificado para enviar transacciones con patrocinio de gas, agrupamiento y nonces en paralelo. Compatible con cualquier contrato EVM.
  </Card>

  <Card title="Blockchain RPC" href="/solutions/infrastructure/blockchain-rpc">
    Acceso a nodos de alta disponibilidad mediante el Sequence Node Gateway. RPC verificado y enrutado automáticamente entre proveedores con configuración mínima.
  </Card>

  <Card title="Sidekick" href="/solutions/infrastructure/sidekick">
    Backend dockerizado para gestión segura de claves y endpoints estandarizados de lectura/escritura (por ejemplo, mint, transfer) usando la infraestructura de Sequence.
  </Card>
</CardGroup>

## Cómo encajan estos componentes

- **Lea rápido con Indexer**; suscríbase a cambios mediante **Webhooks** o **Suscripciones a eventos**.
- **Escriba de forma confiable con Transaction API** usando patrocinio de gas, precios óptimos de gas y reenvío automático.
- **Llame contratos o librerías directamente** a través de **Blockchain RPC** cuando necesite acceso directo al nodo.
- **Ejecute flujos del lado del servidor** con **Sidekick** para custodia segura de claves y endpoints simplificados, o construya su propio servidor con nuestros SDKs.
