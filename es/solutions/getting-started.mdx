Registrarse y crear un proyecto es fácil de hacer y debería tomarte menos de unos minutos. Te ayudaremos a registrarte en Builder y luego crear tu primer proyecto.

¡Empecemos!

## Regístrate en Sequence Builder

<Steps>
  <Step title="Go to sequence.build">
    Ve a [Sequence.Build](https://sequence.build/) y haz clic en el botón `Start Building` o `Sign in` para comenzar.

    <Frame>
      ![Sequence builder landing page](/images/builder/builder_landing_page_signup_project.png)
    </Frame>
  </Step>

  <Step title="Choose a sign in method">
    Cuando inicias sesión por primera vez en Builder, se crea una billetera para ti.

    Sin embargo, es posible que ya tengas una billetera Sequence. Si has iniciado sesión con esa billetera, verás una opción para usarla.

    Sin una billetera, se te darán algunas opciones para iniciar sesión. Puedes usar cualquiera de los inicios de sesión sociales, o simplemente puedes usar tu correo electrónico para comenzar.

    <Frame>
      ![Sequence builder choose signin](/images/builder/builder_choose_signin_signup_project.png)
    </Frame>
  </Step>

  <Step title="Sign in to sequence.build">
    Después de elegir cómo te gustaría iniciar sesión, se te presentarán los términos de uso. Para leer los `Terms of Use`, `Privacy Policy`, y `Master Services Agreement`, haz clic en los enlaces respectivos para cada uno.

    <Frame>
      ![Sequence builder accept terms](/images/builder/builder_accept_terms_signup_project.png)
    </Frame>

    Una vez que hayas leído cada uno, haz clic para continuar.

    ¡Ahora has iniciado sesión en Builder!
  </Step>

  <Step title="Create a Project">
    Todo lo que haces en Builder está centrado en cada proyecto - así que comencemos uno.

    La forma más fácil de comenzar un proyecto es simplemente hacer clic en `+ Create Project`.

    <Frame>
      ![Sequence builder dashboard](/images/builder/builder_dashboard_signup_project.png)
    </Frame>
  </Step>

  <Step title="Enter the project details">
    Aparecerá un modal que te permitirá ingresar detalles sobre tu proyecto. Proporciona un nombre para el proyecto y, si lo deseas, agrega un avatar para el proyecto.

    Luego, selecciona la(s) red(es) en la(s) que te gustaría comenzar a desplegar. Esto siempre se puede cambiar más tarde.

    <Frame>
      ![Sequence builder enter details](/images/builder/builder_enter_details_signup_project.png)
    </Frame>
  </Step>

  <Step title="Create the project!">
    ¡Ahora presiona el botón `Create new Project`!

    <Frame>
      ![Sequence builder project created](/images/builder/builder_project_created_signup_project.png)
    </Frame>

    Una vez que termines, deberías estar viendo el panel de control de tu proyecto.
  </Step>

  <Step title="Claim an API Access Key">
    Finalmente, para comenzar a construir necesitarás reclamar una clave de acceso desde Builder, lo cual se puede lograr navegando a tu `project` en la esquina superior izquierda, seleccionando `Settings` en la navegación lateral, y seleccionando la tarjeta `API Access Keys`

    <Frame>
      ![open project settings](/images/builder/builder_open_access_key_settings.png)
    </Frame>

    Dentro de `settings` del `API Access Keys` selecciona la `dev` clave de acceso, que típicamente se usa para entornos de desarrollo, mientras que la clave `prod` se usa para entornos de producción para aplicaciones frente a tus usuarios.

    <Frame>
      ![settings api keys](/images/builder/builder_settings_api_keys.png)
    </Frame>
  </Step>

  <Step title="(Optional for Development) Create a Service Account">
    Para integrar algunos servicios del stack de Sequence, se debe crear una Cuenta de Servicio para realizar actualizaciones o leer del estado - incluyendo [Collections API](/guides/metadata-guide) o [Webhooks](/api-references/indexer/examples/webhook-listener) - una `Secret API Key` es requerida.

    Se puede navegar a la sección de Claves de Acceso API seleccionando el proyecto, accediendo a la página de navegación lateral de `Settings`, y seleccionando la tarjeta `API Access Keys`

    <Frame>
      ![open project settings](/images/builder/builder_open_access_key_settings.png)
    </Frame>

    Y desplázate hasta el final, donde encontrarás la sección `Secret API Access Key(s)`

    Donde seleccionarías `+ Add Service Account`

    <Frame>
      ![secret API keys](/images/builder/builder_service_accounts_create_secret_api_key.png)
    </Frame>

    Y crear una clave `Read`, o una clave `Write` secreta de acceso API.

    Para realizar actualizaciones en el sistema Sequence, selecciona `Write` y `+ Add Service Account` para incluir en tu integración.

    <Frame>
      ![create a write key](/images/builder/builder_service_accounts_create_write_key.png)
    </Frame>

    Copia el token JWT y guárdalo en un lugar seguro, o en tu proyecto

    <Warning>
      Advertencia: No tendrás acceso al `Secret API Access Key` generado en el futuro desde la interfaz del Builder, asegúrate de guardarlo en un lugar seguro.
    </Warning>
  </Step>
</Steps>
