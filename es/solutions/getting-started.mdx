Registrarse y crear un proyecto es muy sencillo y debería tomarle menos de unos minutos. Le ayudaremos a registrarse en Builder y luego a crear su primer proyecto.

¡Comencemos!

## Regístrese en Sequence Builder

<Steps>
  <Step title="Vaya a sequence.build">
    Ingrese a [Sequence.Build](https://sequence.build/) y haga clic en el botón `Start Building` o `Sign in` para comenzar.

    <Frame>
      ![Página principal de Sequence builder](/images/builder/builder_landing_page_signup_project.png)
    </Frame>
  </Step>

  <Step title="Elija un método de inicio de sesión">
    Cuando inicia sesión por primera vez en Builder, se crea una wallet para usted.

    Sin embargo, es posible que ya tenga una wallet Sequence. Si inicia sesión con esa wallet, verá la opción de usarla.

    Si no tiene una wallet, se le presentarán varias opciones para iniciar sesión. Puede usar cualquiera de los inicios de sesión con redes sociales, o simplemente usar su correo electrónico para comenzar.

    <Frame>
      ![Sequence builder elegir inicio de sesión](/images/builder/builder_choose_signin_signup_project.png)
    </Frame>
  </Step>

  <Step title="Inicie sesión en sequence.build">
    Después de elegir cómo desea iniciar sesión, se le presentarán los términos de uso. Para leer los `Términos de Uso`, la `Política de Privacidad` y el `Acuerdo Maestro de Servicios`, haga clic en los enlaces correspondientes.

    <Frame>
      ![Sequence builder aceptar términos](/images/builder/builder_accept_terms_signup_project.png)
    </Frame>

    Una vez que haya leído cada uno, haga clic en el botón para continuar.

    ¡Ahora ha iniciado sesión en Builder!
  </Step>

  <Step title="Crear un proyecto">
    Todo lo que haga en Builder gira en torno a cada proyecto, así que vamos a crear uno.

    La forma más sencilla de iniciar un proyecto es simplemente hacer clic en `+ Create Project`.

    <Frame>
      ![Panel de Builder](/images/builder/builder_dashboard_signup_project.png)
    </Frame>
  </Step>

  <Step title="Ingrese los detalles del proyecto">
    Aparecerá una ventana emergente donde podrá ingresar los detalles de su proyecto.

    Luego, seleccione la(s) red(es) en la(s) que desea comenzar a desplegar. Esto siempre se puede cambiar más adelante.

    <Frame>
      ![Sequence builder ingresar detalles](/images/builder/builder_enter_details_signup_project.png)
    </Frame>
  </Step>

  <Step title="¡Cree el proyecto!">
    ¡Ahora haga clic en el botón `Create new Project`!

    <Frame>
      ![Proyecto creado en Sequence builder](/images/builder/builder_project_created_signup_project.png)
    </Frame>

    Al finalizar, debería estar viendo el panel de su proyecto.
  </Step>

  <Step title="Solicite una clave de acceso API">
    Finalmente, para empezar a crear necesitará solicitar una clave de acceso desde Builder, lo cual puede hacer navegando a su `project` en la esquina superior izquierda, seleccionando `Settings` en la barra lateral y eligiendo la tarjeta `API Access Keys`.

    <Frame>
      ![Abrir configuración del proyecto](/images/builder/builder_open_access_key_settings.png)
    </Frame>

    Dentro de `settings` en `API Access Keys`, seleccione la clave de acceso `dev`, que normalmente se usa para entornos de desarrollo, mientras que la clave `prod` se utiliza para entornos de producción frente a sus usuarios.

    <Frame>
      ![Configuración de claves API](/images/builder/builder_settings_api_keys.png)
    </Frame>
  </Step>

  <Step title="(Opcional para desarrollo) Cree una cuenta de servicio">
    Para integrar algunos servicios del stack de Sequence, debe crear una cuenta de servicio para hacer actualizaciones o leer el estado, incluyendo [Collections API](/guides/metadata-guide) o [Webhooks](/api-references/indexer/examples/webhook-listener); se requiere una `Secret API Key`.

    Puede ir a la sección de API Access Keys seleccionando el proyecto, accediendo a la página de `Settings` en la barra lateral y eligiendo la tarjeta `API Access Keys`.

    <Frame>
      ![Abrir configuración del proyecto](/images/builder/builder_open_access_key_settings.png)
    </Frame>

    Desplácese hasta el final, donde encontrará la sección `Secret API Access Key(s)`.

    Allí debe seleccionar `+ Add Service Account`.

    <Frame>
      ![Claves API secretas](/images/builder/builder_service_accounts_create_secret_api_key.png)
    </Frame>

    Cree una clave de solo lectura (`Read`) o una clave secreta de acceso API de escritura (`Write`).

    Para hacer actualizaciones en el sistema Sequence, seleccione `Write` y luego `+ Add Service Account` para incluirla en su integración.

    <Frame>
      ![Crear una clave de escritura](/images/builder/builder_service_accounts_create_write_key.png)
    </Frame>

    Copie el token JWT y guárdelo en un lugar seguro, o en su proyecto.

    <Warning>
      Advertencia: No podrá acceder al `Secret API Access Key` generado en el futuro desde la interfaz de Builder, así que asegúrese de guardarlo en un lugar seguro.
    </Warning>
  </Step>
</Steps>
