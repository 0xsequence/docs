# Configuración del Sequence MCP Server
El Sequence MCP Server le permite aprovechar potentes agentes de IA como Claude directamente en su IDE preferido. Esta integración le permite recibir asistencia inteligente en tareas de desarrollo relacionadas con Sequence, desde consultar datos de blockchain hasta interactuar con contratos inteligentes.

## Requisitos previos
Antes de comenzar, asegúrese de tener:
- Node.js instalado en su sistema
- Un IDE compatible (Cursor o Claude Desktop)
- npm o yarn como gestor de paquetes

## Pasos de instalación

1.  Abra su terminal y ejecute el siguiente comando:
    ```bash
    npx @mintlify/mcp@latest add sequence-0fb8d9e6
    ```

2.  Cuando se le solicite, seleccione su cliente MCP preferido:
    - Cursor
    - Claude Desktop

3.  El proceso de instalación configurará automáticamente el Sequence MCP Server para la aplicación seleccionada.

## Uso del MCP Server
Una vez instalado, puede comenzar a usar la asistencia de IA en su IDE. Así es como funciona:
1. Abra su IDE (Cursor o Claude Desktop)
2. Comience a escribir su consulta relacionada con el desarrollo en Sequence
3. El agente de IA procesará su solicitud y le brindará asistencia relevante

### Ejemplos de consultas
Puede hacer preguntas como:
- "Cree un script para consultar balances y metadatos de tokens para la dirección 0x8e3E38fe7367dd3b52D1e281E4e8400447C8d8B9"
- "¿Cómo integro wallets integrados de Sequence?"
- "Muéstreme cómo obtener el historial de transacciones de un wallet"

Cuando su consulta esté siendo procesada, verá una indicación como esta:

```
> Calling MCP tool
```
