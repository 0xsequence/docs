La pila de Sequence Wallet utiliza un "contexto" diferente para cada versión de la billetera.
Este contexto es un conjunto de direcciones que se utilizan para interactuar con la billetera.
El contexto se utiliza para desplegar la billetera, interactuar con la billetera y para interactuar con los módulos de la billetera.

Para más información sobre el contexto de la billetera, consulta la versión específica de la billetera que estés usando:

* [Contexto de Billetera v1](/solutions/technical-references/internals/v1/wallet-context)
* [Contexto de Billetera v2](/solutions/technical-references/internals/v2/wallet-context)
