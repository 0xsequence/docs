La pila de Sequence Wallet utiliza un "contexto" diferente para cada versión de la wallet.
Este contexto es un conjunto de direcciones que se utilizan para interactuar con la wallet.
El contexto se usa para desplegar la wallet, interactuar con la wallet y con los módulos de la wallet.

Para más información sobre el wallet context, consulte la versión específica de la wallet que está utilizando:
- [Wallet Context v1](/solutions/technical-references/internals/v1/wallet-context)
- [Wallet Context v2](/solutions/technical-references/internals/v2/wallet-context)
