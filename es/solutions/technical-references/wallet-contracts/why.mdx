Las wallets de Ethereum pueden tomar dos formas diferentes: cuentas controladas externamente (EOAs) o wallets de smart contracts.
La wallet de Sequence está implementada como un smart contract, lo que permite que el sistema ofrezca seguridad y funcionalidad adicionales.

## Cuentas controladas externamente
Las cuentas controladas externamente son la forma más básica de wallet en Ethereum; las EOAs son cuentas compuestas por un solo par de claves ECDSA.

### Ventajas

- Fáciles de implementar
- Baratas de usar (en algunos escenarios)
- Fáciles de respaldar

### Ejemplos populares

- [MetaMask](https://metamask.io/)
- [Rainbow](https://rainbow.me/)
- [MyEtherWallet](https://www.myetherwallet.com/)
- [Trust](https://trustwallet.com/)

## Wallets de smart contract
Las wallets de smart contract son wallets implementadas como smart contracts; esto permite que las wallets implementen lógica personalizada dentro de los límites que soporta la máquina virtual subyacente.
Por esta razón, estos contratos pueden ofrecer funcionalidades que no están disponibles para las EOAs y también pueden actualizarse si están preparadas para ello.

### Ventajas

- Múltiples llaves
- Rotación de llaves
- Pago de comisiones usando tokens ERC20 (por ejemplo, USDC)
- Capacidad de actualización
- Soporte para recuperación social
- Meta-transacciones
- Espacio de diseño más flexible

## Examples

- [Sequence](https://sequence.app/)
- [Gnosis Safe](https://safe.global/)
- [Argent](https://www.argent.xyz/)

## Comisiones de despliegue por Smart Account AA (Account Abstraction)
Para desplegar wallets de smart contract en la blockchain, se requiere pagar una comisión que varía según el proveedor de AA.

De una lista de proveedores que puede encontrar [aquí](https://github.com/ScreamingHawk/aa-benchmarks), Sequence tiene la comisión total más baja para desplegar una cuenta.
