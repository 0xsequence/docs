Inicia tu integración con Sequence con nuestras demostraciones de código abierto, ejemplos y plantillas base para acelerar tu tiempo de llegada al mercado.

## Incorporación

### Plantilla base de Web SDK con Billetera Integrada para React

Ejemplo de plantilla base que muestra la billetera integrada con Web SDK en una aplicación simple de React.

Ejecutar con Sequence-CLI:

```bash
npx sequence-cli boilerplates create-embedded-wallet-react-starter
```
