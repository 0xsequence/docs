Comience su integración con Sequence con nuestros demos de código abierto, ejemplos y proyectos base para acelerar su salida al mercado.

## Incorporación

### Web SDK con Embedded Wallet para React Boilerplate
Proyecto base de ejemplo que muestra una Embedded Wallet integrada con Web SDK en una aplicación React sencilla.

Ejecute con Sequence-CLI:

```bash
npx sequence-cli boilerplates create-embedded-wallet-react-starter
```
