Inicie su integración con Sequence con nuestros demos, ejemplos y boilerplates de código abierto para acelerar su salida al mercado.

## Onboarding

### Web SDK con Embedded Wallet para React Boilerplate
Boilerplate de ejemplo que muestra el Embedded Wallet integrado con Web SDK en una app React sencilla.

Ejecute con Sequence-CLI:

```bash
npx sequence-cli boilerplates create-embedded-wallet-react-starter
```
