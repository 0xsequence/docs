## Primeros pasos

### Instalación del SDK
Ofrecemos SDKs de TypeScript y Unity para el sistema de autenticación de Embedded Wallet. Puedes instalar el SDK de TypeScript con:

```bash
pnpm install @0xsequence/waas
```

Para más información sobre el SDK de Unity de Embedded Wallet, consulta la [documentación del SDK de Unity](/sdk/unity/overview)

### Configuración del proyecto
Sequence Embedded Wallet está disponible actualmente solo como beta cerrada. Para empezar a usar los SDKs de Embedded Wallet, deberás ponerte en contacto con el equipo de Sequence y obtener credenciales de API. Una vez realizado este paso, recibirás dos claves:
- Clave de acceso al proyecto
- Clave de configuración de Embedded Wallet

### Configuración de la librería
Para comenzar a usar el SDK de Sequence Embedded Wallet, deberás crear una nueva instancia de la clase `waas`:

```typescript
import { SequenceWaaS } from '@0xsequence/waas'

const waas = new SequenceWaaS({
  projectAccessKey: `${process.env.PROJECT_ACCESS_KEY}`,
  waasConfigKey: `${process.env.WAAS_CONFIG_KEY}`,
  network: 'mumbai'
}, defaults.TEST)
```

Ten en cuenta que la librería está operativa, pero no puede usarse para interactuar con ninguna wallet hasta que te hayas autenticado **como usuario**.
