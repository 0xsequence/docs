## Primeros pasos

### Instalación del SDK
Ofrecemos SDKs para TypeScript y Unity para el sistema de autenticación Embedded Wallet. Instale el SDK de TypeScript con:

```bash
pnpm install @0xsequence/waas
```

Para más información sobre el SDK de Unity para Embedded Wallet, consulte la [documentación del SDK de Unity](/sdk/unity/overview)

### Configuración del proyecto
Actualmente, Sequence Embedded Wallet está disponible solo en beta cerrada. Para comenzar a usar los SDKs de Embedded Wallet, debe contactar al equipo de Sequence y obtener credenciales de API. Una vez hecho esto, se le proporcionarán dos claves:
- Clave de acceso al proyecto
- Clave de configuración de Embedded Wallet

### Configuración de la librería
Para comenzar a usar el SDK de Sequence Embedded Wallet, debe crear una nueva instancia de la clase `waas`:

```typescript
import { SequenceWaaS } from '@0xsequence/waas'

const waas = new SequenceWaaS({
  projectAccessKey: `${process.env.PROJECT_ACCESS_KEY}`,
  waasConfigKey: `${process.env.WAAS_CONFIG_KEY}`,
  network: 'mumbai'
}, defaults.TEST)
```

Tenga en cuenta que la librería está operativa, pero no se puede usar para interactuar con ningún wallet hasta que se haya autenticado **como usuario**.
