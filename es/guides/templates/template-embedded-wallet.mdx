## Primeros Pasos

### Instalación del SDK

Proporcionamos SDKs de TypeScript y Unity para el sistema de autenticación de Cartera Integrada. Puedes instalar el SDK de TypeScript con:

```bash
pnpm install @0xsequence/waas
```

Para más información sobre el SDK de Cartera Integrada de Unity, por favor consulta la[documentación del SDK de Unity](/sdk/unity/overview)

### Configuración del Proyecto

La Cartera Integrada de Sequence está actualmente disponible solo como beta cerrada. Para comenzar a usar los SDKs de Cartera Integrada, necesitarás contactar al equipo de Sequence y adquirir credenciales de API. Una vez hecho esto, se te proporcionarán dos claves:

* Clave de acceso al proyecto
* Clave de configuración de Cartera Integrada

### Configuración de la Biblioteca

Para comenzar a usar el SDK de Cartera Integrada de Sequence, necesitarás crear una nueva instancia de la clase`waas`:

```typescript
import { SequenceWaaS } from '@0xsequence/waas'

const waas = new SequenceWaaS({
  projectAccessKey: `${process.env.PROJECT_ACCESS_KEY}`,
  waasConfigKey: `${process.env.WAAS_CONFIG_KEY}`,
  network: 'mumbai'
}, defaults.TEST)
```

Ten en cuenta que la biblioteca está operativa, pero no puede ser utilizada para interactuar con ninguna cartera hasta que te hayas autenticado**como usuario**.
