title: Instalación
description: Documentación para la instalación del SDK de Unreal para la infraestructura de Sequence para juegos web3.
---

## Usando Epic Games Marketplace

1.  Visita el [Epic Games Marketplace](https://www.unrealengine.com/marketplace/en-US/product/sequence-embedded-wallets)

2.  Haz clic en "Free" para agregar el plugin a tus Assets adquiridos.

3.  Haz clic en "Open in Launcher" e instala el Plugin para tu versión del Engine.

4.  Abre el proyecto, ve a `Edit > Plugins` y habilita el `SequencePlugin`

5.  Reinicia el Unreal Editor para que los cambios tengan efecto

6.  Para encontrar la carpeta de contenido de `SequencePlugin` en el content drawer, habilita mostrar contenido del engine: `Content Drawer > Settings > Show Engine Content`

## Usando la última versión de GitHub

1.  Descarga el archivo .zip de la última versión desde [Release from GitHub.](https://github.com/0xsequence/sequence-unreal/releases)

2.  Ve al directorio raíz del proyecto de Unreal usando un sistema de archivos o línea de comandos y crea una carpeta `Plugins` si aún no la tienes.

3.  Descomprime el archivo descargado y copia la carpeta SequencePlugin en la carpeta Plugins de tu proyecto.

4.  Inicia el proyecto y permite que actualice la configuración de UProject.

5.  Para encontrar la carpeta de contenido de `SequencePlugin` en el content drawer, habilita mostrar contenido de plugins: `Content Drawer > Settings > Show Plugin Content`

## Usando el Boilerplate de Unreal de Sequence
Utiliza nuestro boilerplate de proyecto para comenzar rápidamente a construir el proyecto desde cero.
1.  [Descarga nuestro Boilerplate desde GitHub](https://github.com/0xsequence/unreal-embedded-wallet-boilerplate)

2.  Instale la versión 5.4 de Unreal Engine, si aún no la tiene.

3.  Abra el Boilerplate Project y visite nuestra [Guía de Unreal Boilerplate](/guides/unreal-ew-guide)

4.  Reinicia el Unreal Editor para que los cambios tengan efecto

5.  Para encontrar la carpeta de contenido de `SequencePlugin` en el content drawer, habilita mostrar contenido del engine: `Content Drawer > Settings > Show Engine Content`
