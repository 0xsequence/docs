## Usando el Boilerplate de Unreal de Sequence

1. [Descarga nuestro Boilerplate desde GitHub](https://github.com/0xsequence/unreal-embedded-wallet-boilerplate)

2. Instala Unreal Engine versión 5.4, si aún no lo tienes.

3. Abre el Proyecto Boilerplate y visita nuestra [Guía del Boilerplate de Unreal](/guides/unreal-ew-guide)

## Usando la última versión de GitHub

1. Descarga el archivo .zip de la última versión desde [Release desde GitHub.](https://github.com/0xsequence/sequence-unreal/releases)

2. Navega al directorio raíz de tu proyecto de Unreal usando un sistema de gestión de archivos o línea de comandos y crea una `Plugins` carpeta, si aún no tienes una.

3. Descomprime el archivo descargado y copia la `SequencePlugin` carpeta en la carpeta `Plugins` de tu proyecto.

4. Inicia tu proyecto, luego permite que actualice la Configuración del UProject.

5. Para encontrar la `SequencePlugin` carpeta de contenido en tu cajón de contenido, habilita mostrar contenido del plugin: `Content Drawer > Settings > Show Plugin Content`

## Usando el Marketplace de Epic Games

1. Visita el [Marketplace de Epic Games](https://www.unrealengine.com/marketplace/en-US/product/sequence-embedded-wallets)

2. Haz clic en "Gratis" para agregar el plugin a tus Activos comprados.

3. Haz clic en "Abrir en Launcher", e instala el Plugin para tu versión del Engine.

4. Abre tu proyecto, navega a `Edit > Plugins`, y habilita el `SequencePlugin`

5. Reinicia el Editor de Unreal para que los cambios surtan efecto

6. Para encontrar la `SequencePlugin` carpeta de contenido en tu cajón de contenido, habilita mostrar contenido del engine: `Content Drawer > Settings > Show Engine Content`

## Actualizando a Unreal 5.4

Antes de actualizar, por favor respalda cualquier dato de configuración almacenado en `PluginConfig/Config.h` o `Config/Config.h`. Una vez respaldado, elimina todo el directorio SequencePlugin. Reemplázalo con la nueva versión actualizada del plugin.

Si tu proyecto está actualmente en 5.2 o 5.3, entonces para esta versión querrás actualizar a 5.4.

1. Cierra el Editor de Unreal y tu Editor de Código
2. Haz clic derecho en tu archivo de Proyecto Unreal > "Cambiar Versión de Unreal Engine" (puede que necesites hacer clic en "Mostrar más opciones"), elige 5.4 y confirma
3. Abre tu editor de código y recompila tu código fuente

Como una forma de preparación para el futuro, recomendamos instalar Visual Studio 2022 de acuerdo con [estos documentos](https://dev.epicgames.com/documentation/en-us/unreal-engine/setting-up-visual-studio-development-environment-for-cplusplus-projects-in-unreal-engine) ya que Unreal 5.4 y versiones posteriores dejarán de dar soporte a Visual Studio 2019.
