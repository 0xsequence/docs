---
title: Quickstart
description: Documentación de inicio rápido para el SDK de Unreal de Sequence.
---

<Steps>
  <Step title="Install the Unreal SDK">
    Descarga e instala la última versión del SDK de Unreal de Sequence desde [GitHub](https://github.com/0xsequence/sequence-unreal/releases), o comienza un nuevo proyecto desde cero descargando nuestro [Unreal Project Boilerplate](https://github.com/0xsequence/unreal-embedded-wallet-boilerplate).
  </Step>

  <Step title="Configure the SDK">
    Inicia sesión y crea un proyecto en [Sequence Builder](https://sequence.build).

    Descarga tu archivo de configuración desde Builder como se muestra a continuación.

    Alternativamente, obtén tu `Project Access Key` y `WaaS Configuration Key` desde `Onboard > Embedded Wallet`,
    y añádelos a tu [Archivo de Configuración de Sequence del Proyecto](/sdk/unreal/configuration).

    <Frame>
      <img src="/images/unreal/builder_config_file.png" />
    </Frame>
  </Step>

  <Step title="Integrate E-Mail Login">
    Comienza estableciendo una sesión de wallet usando un OTP por correo electrónico. Ejecuta el `Start Email Login Async` nodo para enviar una
    contraseña de un solo uso a la dirección de correo electrónico especificada. Luego, usa el `Confirm Email Login with Code Async` nodo con la
    contraseña recibida para completar el inicio de sesión.

    Una vez que hayas integrado tus primeras funciones, puedes continuar con [proveedores de inicio de sesión adicionales](/sdk/unreal/authentication)
    como Google, Apple o PlayFab.

    <Frame caption="Email Otp">
      <img src="/images/unreal/email_otp.png" />
    </Frame>
  </Step>

  <Step title="Test Sequence's Features">
    El SDK de Unreal de Sequence incluye una variedad de Blueprints para ayudarte a comenzar rápidamente tu juego.
    Una vez que todo está configurado, puedes crear widgets para mostrar un Inventario, Vista de Wallet o Tienda en el Juego.
    Consulta [cómo integrar una Vista de Wallet](/sdk/unreal/bootstrap_game#my-wallet-view) usando solo tres nodos de Blueprint para ver y enviar fondos.

    <Frame caption="Feature Wallet Demo">
      <img src="/images/unreal/feature_wallet_demo.gif" />
    </Frame>
  </Step>
</Steps>
