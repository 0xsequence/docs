---
title: Inicio rápido
description: Documentación de inicio rápido para el SDK de Unreal de Sequence.
---

<Steps>
  <Step title="Instale el SDK de Unreal">
    Descargue el SDK desde el [Epic Games Marketplace](https://www.unrealengine.com/marketplace/en-US/product/sequence-embedded-wallets).
    Consulte nuestros [pasos de instalación](/sdk/unreal/installation) para más información. Alternativamente, use nuestro
    [boilerplate](https://github.com/0xsequence/unreal-embedded-wallet-boilerplate) para iniciar rápidamente un nuevo proyecto de Unreal desde cero.
  </Step>

  <Step title="Configure el SDK">
    Inicie sesión y cree un proyecto en [Sequence Builder](https://sequence.build).

    Descargue su archivo de configuración desde Builder como se muestra a continuación.

    Como alternativa, obtenga su `Project Access Key` y `WaaS Configuration Key` desde `Onboard > Embedded Wallet`,
    y agréguelos al [archivo de configuración de Sequence de su proyecto](/sdk/unreal/configuration).

    <Frame>
      <img src="/images/unreal/builder_config_file.png" />
    </Frame>
  </Step>

  <Step title="Integre inicio de sesión por correo electrónico">
    Comience estableciendo una sesión de wallet usando un OTP enviado por correo electrónico. Ejecute el nodo `Start Email Login Async` para enviar una contraseña de un solo uso al correo electrónico especificado. Luego, use el nodo `Confirm Email Login with Code Async` con la contraseña recibida para completar el inicio de sesión.

    Una vez que haya integrado sus primeras funciones, puede continuar con [proveedores de inicio de sesión adicionales](/sdk/unreal/authentication)
    como Google, Apple o PlayFab.

    <Frame caption="Email Otp">
      <img src="/images/unreal/email_otp.png" />
    </Frame>
  </Step>

  <Step title="Pruebe las funciones de Sequence">
    El SDK de Unreal de Sequence incluye varios boilerplate para ayudarle a iniciar su juego rápidamente.
    Una vez que todo esté configurado, puede crear widgets para mostrar un inventario, Wallet View o una tienda dentro del juego.
    Consulte [cómo integrar una Wallet View](/sdk/unreal/bootstrap_game#my-wallet-view) usando solo tres nodos de Blueprint para ver y enviar fondos.

    <Frame caption="Feature Wallet Demo">
      <img src="/images/unreal/feature_wallet_demo.gif" />
    </Frame>
  </Step>
</Steps>
