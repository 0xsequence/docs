Nuestro SDK de Unreal incluye un conjunto de subsistemas para una integración fluida con Blueprints. Puede usarlos en cualquiera de sus
Blueprints, como en el perfil de usuario de su juego o en la pantalla de inicio de sesión. Para manejar solicitudes asíncronas, asegúrese de
asignar listeners de eventos a las llamadas que realice.

## Sesiones de Sequence
Utilice el Wallet Subsystem de Sequence para crear y gestionar sesiones de usuario, como cuando inician sesión con correo electrónico o Google.

## Sequence Wallet
Utilice el Wallet Subsystem de Sequence para enviar transacciones o firmar mensajes. Asegúrese de que el usuario haya creado una sesión previamente.

<Frame caption="Subsystem Wallet">
  <img src="/images/unreal/subsystem_wallet.png" />
</Frame>

## Sequence Indexer
Utilice el Indexer Subsystem de Sequence para obtener información del usuario, como su saldo o los tokens que posee.

  <img src="/images/unreal/subsystem_indexer.png" />

## Sequence Pay
Utilice el Pay Subsystem de Sequence para acceder a la funcionalidad On-Ramp y permitir que los usuarios compren cripto con tarjetas de crédito.

## Sequence Utility
Utilice el Utility Subsystem de Sequence para funciones de conveniencia que encapsulan casos de uso comunes en nodos de Blueprint más pequeños.
