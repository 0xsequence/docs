title: Usando los Subsistemas de Sequence
sidebarTitle: Blueprint Subsystems

---

Nuestro SDK de Unreal incluye un conjunto de Subsistemas para una integración perfecta con Blueprint. Puedes usarlos en cualquiera de tus
Blueprints, como en el Perfil de Usuario o Pantalla de Inicio de Sesión de tu Juego. Para manejar solicitudes asíncronas, asegúrate de
asignar event listeners a las llamadas que realices.

## Sequence Sessions

Usa el Subsistema Wallet de Sequence para crear y gestionar sesiones de usuario, como cuando inician sesión con Email o Google.

  <img src="/images/unreal/subsystem_sessions.png" />

## Sequence Wallet

Usa el Subsistema Wallet de Sequence para enviar transacciones o firmar mensajes. Asegúrate de que el usuario haya creado una sesión previamente.

<Frame caption="Subsystem Wallet">
  <img src="/images/unreal/subsystem_wallet.png" />
</Frame>

## Sequence Indexer

Usa el Subsistema Indexer de Sequence para recuperar información del usuario, como su saldo o los tokens que posee.

  <img src="/images/unreal/subsystem_indexer.png" />

## Sequence Pay

Usa el Subsistema Pay de Sequence para acceder a la funcionalidad On-Ramp y permitir que los usuarios compren crypto usando tarjetas de crédito.

  <img src="/images/unreal/subsystem_pay.png" />

## Sequence Utility

Usa el Subsistema Utility de Sequence para funciones de conveniencia que encapsulan casos de uso comunes en nodos Blueprint más pequeños.
