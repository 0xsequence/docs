---
title: Configuración
description: Documentación para la configuración del SDK de Unreal para la pila
  de infraestructura Sequence para juegos web3.
---

<Steps>
  <Step title="Private Dependency Modules">
    En el archivo Build.cs de tu Proyecto ([dónde encontrarlo](https://forums.unrealengine.com/t/where-are-build-cs-and-target-cs/302665)) por favor verifica el módulo de Dependencia privada para que incluya "SequencePlugin":

    ```cpp
        public SequenceTest(ReadOnlyTargetRules Target) : base(Target)
        {
            PCHUsage = PCHUsageMode.UseExplicitOrSharedPCHs;
    		PublicDependencyModuleNames.AddRange(new string[] { "Core", "CoreUObject", "Engine", "InputCore" });
    		PrivateDependencyModuleNames.AddRange(new string[] { "SequencePlugin" });
    	}
    ```
  </Step>

  <Step title="Create a config file">
    Navega a `[YourProjectDirectory]/Config`, crea un archivo llamado `SequenceConfig.ini`, y agrega el siguiente contenido:

    ```
    [/Script/Sequence.Config]

    WaaSConfigKey = "Go to https://sequence.build/ under Onboard > Embedded Wallet"
    ProjectAccessKey = "Go to https://sequence.build/ under Settings > API Keys"
    GoogleClientID = "" 
    AppleClientID = "" 
    FacebookClientID = "" 
    DiscordClientID = ""
    PlayFabTitleID = ""
    RedirectUrl = "https://api.sequence.app"
    ```
  </Step>
</Steps>

<Warning>
  ¡No olvides [configurar los ajustes de tu billetera integrada en el Builder](/solutions/builder/embedded-wallet/configuration)!
</Warning>
