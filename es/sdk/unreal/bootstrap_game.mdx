El SDK de Unreal de Sequence incluye una variedad de funciones para ayudarle a iniciar su juego.
Puede crearlas como se muestra a continuación o duplicar los archivos `.uasset` de la carpeta `Plugins/SequencePlugin Content/Samples/`
y personalizarlos según sus necesidades.

## Inventario
Cree el widget `Sequence Inventory` para mostrar todos los ítems que posee un usuario de cualquier contrato ERC1155 o ERC721.

</Frame>

<Frame caption="Feature Inventory">

<Accordion title="Parámetros">
  **Chain (ENetwork Enum)**

  El tipo de cadena (Ethereum, Sepolia, ..) en la que está desplegado el contrato especificado.

  **Wallet Address (FString)**

  La dirección de wallet del usuario para el que desea listar los ítems del inventario.

  **Contract (FString)**

  La dirección de un contrato ERC1155 o ERC721 donde la wallet especificada posee ítems.
</Accordion>

## Venta primaria
Acelera el crecimiento del juego vendiendo ítems directamente a los jugadores. Crea el widget `Sequence Primary Sale` para mostrar una Tienda dentro del juego que utiliza ítems
de un contrato ERC1155, que los usuarios compran usando cualquier moneda ERC20 personalizada o existente.

[Contrato de Ventas Primarias](/solutions/collectibles/contracts/deploy-primary-sales-contract/): Aprenda cómo configurar y desplegar contratos para lanzar una venta primaria.

</Frame>

<Frame caption="Feature Sale">
  <img src="/images/unreal/feature_sale.png" />

<Accordion title="Parámetros">
  **Chain (ENetwork Enum)**

  El tipo de cadena (Ethereum, Sepolia, ..) en la que está desplegado el contrato especificado.

  **Token Contract Address (FString)**

  La dirección del contrato de token ERC1155 o ERC721 que desplegaste usando el Builder Console de Sequence.

  **Sale Contract Address (FString)**

  La dirección del contrato de venta que desplegaste usando el Builder Console de Sequence.

  **Payment Token (FString)**

  La dirección del token ERC20 que especificaste en la sección 'Mint Access' de tu contrato de venta.

  **Price (Integer)**

  El precio por cada token que especificaste en la sección 'Mint Access' de tu contrato de venta.

  **Tokens for Sale (Integer Array)**

  Una lista de ID de tokens (por ejemplo, \[0, 1, 2]) que se muestra en la interfaz de usuario.
</Accordion>

## Vista de mi wallet
Crea el widget `Sequence Profile` para mostrar un código QR para recibir fondos o enviar fondos a otras wallets.

<Frame caption="Feature Wallet Demo">
  <img src="/images/unreal/feature_wallet_demo.gif" />
</Frame>

<Frame caption="Feature Wallet">
  <img src="/images/unreal/feature_wallet.png" />
</Frame>

<Accordion title="Parámetros">
  **Chain (ENetwork Enum)**

  El tipo de cadena (Ethereum, Sepolia, ..) donde quieres ejecutar el juego.

  **Wallet Address (FString)**

  La dirección de wallet del usuario para quien deseas mostrar esta vista.

  **Enable Send (Boolean)**

  Indica si se habilita el botón para abrir una vista de envío. Establece esto en falso si quieres mostrar el perfil de otro usuario.
</Accordion>
