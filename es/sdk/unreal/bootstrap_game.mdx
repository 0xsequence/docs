El SDK de Unreal de Sequence incluye una variedad de características para ayudarte a iniciar tu juego. Puedes crearlas como se muestra a continuación o duplicar los `.uasset` archivos de la `Plugins/SequencePlugin Content/Samples/`
carpeta y personalizarlos según tus necesidades.

## Inventario

Crea el `Sequence Inventory` widget para mostrar todos los elementos que posee un usuario de cualquier contrato ERC1155 o ERC721.

</Frame>

<Frame caption="Feature Inventory">
  <img src="/images/unreal/feature_inventory.png" />
</Frame>

<Accordion title="Parameters">
  **Cadena (Enum ENetwork)**

  El tipo de cadena (Ethereum, Sepolia, ..) en la que está implementado el contrato especificado.

  **Dirección de Billetera (FString)**

  La dirección de la billetera del usuario para quien deseas listar los elementos del inventario.

  **Contrato (FString)**

  La dirección de un contrato ERC1155 o ERC721 donde la dirección de billetera especificada posee elementos.
</Accordion>

## Venta Primaria

Acelera el crecimiento de tu juego vendiendo elementos directamente a tus jugadores. Crea el `Sequence Primary Sale` widget para mostrar una Tienda En el Juego que utiliza elementos del juego de un contrato ERC1155, que los usuarios compran usando cualquier moneda ERC20 personalizada o existente.

[Contrato de Ventas Primarias](/solutions/collectibles/contracts/deploy-primary-sales-contract/): Aprende cómo configurar y desplegar contratos para lanzar una venta primaria.

<Frame caption="Feature Sale Demo">
  <img src="/images/unreal/feature_sale_demo.gif" />
</Frame>

<Frame caption="Feature Sale">
  <img src="/images/unreal/feature_sale.png" />
</Frame>

<Accordion title="Parameters">
  **Cadena (ENetwork Enum)**

  El tipo de cadena (Ethereum, Sepolia, ..) en la que está desplegado el contrato especificado.

  **Dirección del Contrato de Token (FString)**

  La dirección del contrato de token ERC1155 o ERC721 que desplegaste usando Sequence's Builder.

  **Dirección del Contrato de Venta (FString)**

  La dirección del contrato de venta que desplegaste usando Sequence's Builder.

  **Token de Pago (FString)**

  La dirección del token ERC20 que especificaste en la sección 'Mint Access' de tu contrato de Venta.

  **Precio (Integer)**

  El precio por cada token que especificaste en la sección 'Mint Access' de tu contrato de Venta.

  **Tokens en Venta (Integer Array)**

  Una lista de IDs de tokens (por ejemplo \[0, 1, 2]) que muestras en tu interfaz de usuario.
</Accordion>

## Vista de Mi Billetera

Crea el `Sequence Profile` widget para mostrar un código QR para recibir fondos o enviar fondos a otras billeteras.

<Frame caption="Feature Wallet Demo">
  <img src="/images/unreal/feature_wallet_demo.gif" />
</Frame>

<Frame caption="Feature Wallet">
  <img src="/images/unreal/feature_wallet.png" />
</Frame>

<Accordion title="Parameters">
  **Cadena (ENetwork Enum)**

  El tipo de cadena (Ethereum, Sepolia, ..) donde quieres ejecutar tu juego.

  **Dirección de Billetera (FString)**

  La dirección de la billetera del usuario para quien quieres mostrar esta vista.

  **Habilitar Envío (Boolean)**

  Indica si se debe habilitar el botón para abrir una vista de envío. Establece esto en falso si quieres mostrar el perfil de otro usuario.
</Accordion>
