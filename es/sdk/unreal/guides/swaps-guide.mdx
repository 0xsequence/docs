---
title: Guía rápida para Swaps
description: Guía rápida sobre cómo hacer Smart Swaps usando el SDK de Unreal de Sequence.
---

<Steps>
  <Step title="Obtener el precio de swap">
    Obtén el precio de un par de tokens determinado. Como alternativa, llama a `GetSwapPrices` para obtener una lista de opciones disponibles.
    Asegúrate de que un DEX como `UniSwap` tenga suficientes fondos para tus tokens.

    <Frame>
      <img src="/images/unreal/swap_price.png" />
    </Frame>
  </Step>

  <Step title="Obtener la cotización de swap">
    Cuando ya conoces el precio de un par de tokens, llama a `Get Swap Quote` para ese mismo par y obtén los datos de transacción necesarios para tu swap.

    <Frame>
      <img src="/images/unreal/swap_price.png" />
    </Frame>
  </Step>

  <Step title="Enviar transacción">
    Ahora que tiene los datos de su transacción de `Get Swap Quote`, construya un objeto de transacción y primero
    agregue el `Approve Data` y luego el `Transaction Data`. Asegúrese de que el orden de las llamadas a `Add Raw` sea el mismo.

    <Frame>
      <img src="/images/unreal/guides/send_swap_transaction.png" />
    </Frame>
  </Step>
</Steps>
