Si deseas que tus usuarios puedan pagar cosas en tu juego usando criptomonedas, encontrarás que nuestra integración con Transak es muy útil.
Transak permite a los usuarios comprar una variedad de criptomonedas en diferentes blockchains usando su tarjeta de crédito a través de una interfaz web fácil de usar. Aprende más sobre Transak[aquí](https://transak.com/).

## Generar URL de On-Ramp

Para generar un enlace para el on-ramping de Transak de tu usuario.

<Tabs>
  <Tab title="Blueprint">
    <Frame caption="Pay Onramp Link">
      <img src="/images/unreal/pay_onramp_link.png" />
    </Frame>
  </Tab>

  <Tab title="C++">
    ```cpp
    UTransakOnRamp* Transak = UTransakOnRamp::Init("UserWalletAddress");
    FString OnRampLink = Transak->GetTransakLink("USD", "100", "BTC", "network1,network2", false);
    // You can tailor the user experience by providing different default values for the parameters.
    ```
  </Tab>
</Tabs>

<Accordion title="Parameters">
  **Dirección de Billetera (FString)**

  La dirección del usuario que recibe la cantidad comprada de criptomoneda.

  **Moneda Fiat (FString)**

  La moneda fiat predeterminada mostrada en el modal de On-Ramp.

  **Cantidad Fiat (FString)**

  La cantidad fiat predeterminada mostrada en el modal de On-Ramp.

  **Criptomoneda (FString)**

  La criptomoneda que el usuario puede comprar.

  **Redes (FString)**

  Lista de redes que el usuario puede seleccionar, separadas por coma. Por ejemplo: 'ethereum, mainnet'.

  **Deshabilitar Formulario de Dirección de Billetera (Boolean)**

  Establece este valor en 'false' para permitir que los usuarios editen su dirección de billetera.
</Accordion>

<Accordion title="Return Types">
  **Tipo de Retorno (FString)**

  El enlace de On-Ramp utilizado para abrir en un WebView o Navegador Web externo.
</Accordion>

## Abrir URL de On-Ramp

Para generar un enlace para el on-ramping de Transak de tu usuario:

<Tabs>
  <Tab title="Blueprint">
    <Frame caption="Pay Open Onramp">
      <img src="/images/unreal/pay_open_onramp.png" />
    </Frame>
  </Tab>

  <Tab title="C++">
    ```cpp
    UTransakOnRamp* Transak = UTransakOnRamp::Init("UserWalletAddress");
    Transak->OpenTransakLink("USD", "100", "BTC", "network1,network2", false);
    // This will automatically open the generated Transak On-Ramp link in the default web browser.
    ```
  </Tab>
</Tabs>

<Accordion title="Parameters">

  La moneda fiat predeterminada mostrada en el modal On-Ramp.

  **Cantidad Fiat (FString)**

  La cantidad fiat predeterminada mostrada en el modal On-Ramp.

  **Criptomoneda (FString)**

  La criptomoneda que el usuario puede comprar.

  **Redes (FString)**

  Lista de redes que el usuario puede seleccionar, separadas por coma. Por ejemplo: 'ethereum, mainnet'.

  **Deshabilitar Formulario de Dirección de Billetera (Boolean)**

  Establezca este valor en 'false' para permitir que los usuarios editen su dirección de billetera.
</Accordion>

## Obtener Países Soportados

Para verificar los países soportados por Transak y conocer cómo están soportados, use el`GetSupportedCountries` método.

<Tabs>
  <Tab title="Blueprint">
    <Frame caption="Pay Get Supported Countries">
      <img src="/images/unreal/pay_get_supported_countries.png" />
    </Frame>
  </Tab>

  <Tab title="C++">
    ```cpp
    Transak->GetSupportedCountries(
        [](TArray<FSupportedCountry> SupportedCountries) {
        for (const FSupportedCountry& Country : SupportedCountries) {
        UE_LOG(LogTemp, Log, TEXT("Country: %s"), *Country.CountryName);
    }
    },
        [](FSequenceError Error) {
            UE_LOG(LogTemp, Error, TEXT("Encountered error fetching supported countries from Transak: %s"), *Error.ErrorMessage);
        }
    );
    ```
  </Tab>
</Tabs>

<Accordion title="Return Types">
  **Países Soportados (Array de FSupportedCountry)**

  Lista de objetos que incluyen información para cada país, como el nombre o código de moneda.
</Accordion>
