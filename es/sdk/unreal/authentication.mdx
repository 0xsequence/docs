## Inicio de Sesión Social

Para iniciar la autenticación basada en SSO en escritorio, necesitas navegar a un navegador para obtener el id\_token necesario.
En móvil, nuestro SDK maneja este proceso por ti usando plugins integrados.

En plataformas de escritorio, escucha el evento `Sign in Web View Required` y abre el `Sign In URL` retornado.
En plataformas móviles, escucha el evento `Id Token Received`.

Si tienes tu propia integración de PlayFab, puedes utilizarla. Una vez que hayas recibido tu SessionTicket de PlayFab, por favor pásalo para la autenticación con las APIs de Sequence usando el`PlayfabAuthenticateWithSessionTicket` método en tu`USequenceAthenticator` objeto.

## Federar Cuentas

Con las Cuentas Federadas, puedes asociar múltiples métodos de inicio de sesión con una sola cuenta y billetera.
Si tu usuario ha iniciado sesión como Invitado, definitivamente querrás empujarlos hacia la federación de su
cuenta para tener credenciales persistentes con las que puedan acceder a su Sequence Embedded Wallet en sesiones posteriores.
Mientras un usuario está autenticado con la API de Sequence, puedes agregar un método de inicio de sesión adicional usando la llamada apropiada de federar cuenta.

## Obtener Token de Id

Un caso de uso común es verificar la identidad del usuario en tu backend después de que el usuario haya completado el proceso de inicio de sesión en tu cliente de aplicación.
Obtén un Token de Id de Sequence de tus usuarios para enviarlo a tu backend para verificación usando cualquier biblioteca JWKS.
Lee nuestra[Guía de Verificación de Backend](/sdk/headless-wallet/verification) para más información.

<Tabs>
  <Tab title="Blueprint">
    <Frame>
      <img src="/images/unreal/get_id_token.png" />
    </Frame>
  </Tab>

  <Tab title="C++">
    ```cpp
    const TFunction<void (FSeqIdTokenResponse_Data)> OnSuccess = [this](const FSeqIdTokenResponse_Data& Data) { };
    const TFunction<void (FSequenceError)> OnFailure = [this](const FSequenceError& Err){ };

    const TOptional<USequenceWallet*> WalletOptional = USequenceWallet::Get();
    if (WalletOptional.IsSet() && WalletOptional.GetValue())
    {
        const USequenceWallet* Wallet = WalletOptional.GetValue();
        Wallet->GetIdToken(Nonce, OnSuccess, OnFailure);
    }
    ```
  </Tab>
</Tabs>

    const TSuccessCallback<TArray<FSession>> OnSuccess = [=](TArray<FSession> Response)
    {
        //Response is a list of Sessions
    };
    const FFailureCallback OnFailure = [=](const FSequenceError& Error)
    {
        UE_LOG(LogTemp,Display,TEXT("Error Message: %s"),*Error.Message);
    };
    const TOptional<USequenceWallet*> WalletOptional = USequenceWallet::Get();
    if (WalletOptional.IsSet() && WalletOptional.GetValue())
    {
        USequenceWallet * Api = WalletOptional.GetValue();
        Api->ListSessions(OnSuccess,OnFailure);
    }
