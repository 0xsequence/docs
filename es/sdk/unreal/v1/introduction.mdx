La versión V1 del SDK de Unreal de Sequence está obsoleta, considere [migrar a V2.](/sdk/unreal/v1/migration)

El SDK de Sequence Unreal Embedded Wallet ofrece integración completa de [Embedded Wallet](/solutions/wallets/developers/embedded-wallet/overview) de Sequence y del [Indexer](/api-references/indexer/overview) para juegos desarrollados con Unreal Framework. Contamos con soporte total para Blueprints, lo que permite una integración sencilla y fluida.

## Plataformas compatibles
Consulte nuestros [Requisitos de plataforma](/sdk/unreal/v1/platforms)
- Unreal Engine 5.5, 5.4, 5.3 y 5.2
- Android
- iOS
- PC Standalone
- Mac Standalone

## Métodos de inicio de sesión compatibles
Comience la integración usando nuestro [Session Subsystem para Blueprints](/sdk/unreal/v1/authentication).
- Correo electrónico con contraseña de un solo uso (OTP)
- Inicio de sesión único (Single Sign On) con Google y Apple
- Inicio de sesión y registro con PlayFab
- Inicio de sesión como invitado
