---
title: Guía rápida de Unreal v1
description: Documentación de inicio rápido para el SDK de Unreal de Sequence.
---

<Steps>
  <Step title="Instale el SDK de Unreal">
    Descargue e instale la última versión del Unreal SDK de Sequence desde [GitHub](https://github.com/0xsequence/sequence-unreal/releases), o inicie un proyecto nuevo desde cero
    descargando nuestro [Unreal Project Boilerplate](https://github.com/0xsequence/unreal-embedded-wallet-boilerplate).
  </Step>

  <Step title="Configure el SDK">
    Inicie sesión y cree un proyecto en [Sequence Builder](https://sequence.build).

    Obtenga su `Project Access Key` y `WaaS Configuration Key` desde `Onboard > Embedded Wallet`,
    y agréguelos a su [archivo de configuración de Sequence del proyecto](/sdk/unreal/v1/configuration).
  </Step>

  <Step title="Integre inicio de sesión por correo electrónico">
    Comience estableciendo una sesión de wallet usando un OTP enviado por correo electrónico. Ejecute el nodo `Start Email Login Async` para enviar una contraseña de un solo uso al correo electrónico especificado. Luego, use el nodo `Confirm Email Login with Code Async` con la contraseña recibida para completar el inicio de sesión.

    Una vez que haya integrado sus primeras funciones, puede continuar con [proveedores de inicio de sesión adicionales](/sdk/unreal/v1/authentication)
    como Google, Apple o PlayFab.

    <Frame caption="Email Otp">
      <img src="/images/unreal/v1/email_otp.png" />
    </Frame>
  </Step>

  <Step title="Pruebe las funciones de Sequence">
    El Unreal SDK de Sequence incluye una variedad de Blueprints para ayudarle a iniciar su juego rápidamente.
    Una vez que todo esté configurado, puede crear widgets para mostrar un inventario, la vista de wallet o una tienda dentro del juego.
    Consulte [cómo integrar una Wallet View](/sdk/unreal/v1/bootstrap_game#my-wallet-view) usando solo tres nodos de Blueprint para ver y enviar fondos.

    <Frame caption="Feature Wallet Demo">
      <img src="/images/unreal/v1/feature_wallet_demo.gif" />
    </Frame>
  </Step>
</Steps>
