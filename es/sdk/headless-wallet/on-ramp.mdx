Como el producto Embedded Wallet puede integrarse con [Web SDK](/sdk/web/overview), un desarrollador puede permitir a sus usuarios comprar criptomonedas con moneda fiduciaria con tarjeta de crédito o débito usando un componente react simplificado.

Los pasos de integración se encuentran [aquí](/sdk/web/guides/on-ramp)
