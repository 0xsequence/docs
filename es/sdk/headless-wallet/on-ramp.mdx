sidebarTitle: On-ramp
---

Ya que el producto de billetera integrada puede ser integrado con
[Web SDK](/sdk/web/overview), un desarrollador puede permitir que sus usuarios compren fiat a través de una tarjeta de crédito o débito con el uso de un componente react simplificado.


Los pasos de integración se pueden encontrar
[aquí](/sdk/web/guides/on-ramp)
