Para integración con Unreal o Unity, ve a [Unreal](/sdk/unreal/quickstart) o [Unity](/sdk/unity/installation).

<Steps>
  <Step title="Clone Starter Repo">
    Ejecuta lo siguiente desde tu terminal para clonar el repositorio inicial.

    <CodeGroup>
      ```bash Javascript
      npx sequence-cli boilerplates create-email-embedded-wallet-react-starter
      ```

      Selecciona usar Claves de Prueba.
    </CodeGroup>
  </Step>

  <Step title="Connect and Authenticate">
    Navega a `localhost:4444` e ingresa tu correo electrónico para probar la autenticación.
  </Step>

  <Step title="Add your embedded wallet keys">
    Visita Sequence Builder y [configura tu wallet integrada](/solutions/builder/embedded-wallet/configuration) para obtener claves específicas para tu proyecto y autenticar a tus usuarios. Simplemente reemplaza las claves de configuración existentes en el archivo .env.
  </Step>

  <Step title="Use Sequence Embedded Wallets!">
    Ahora que tienes una autenticación funcionando, puedes revisar nuestros ejemplos para las funciones más comunes:

    * [Firmar mensajes](/sdk/headless-wallet/use-wallets#sign-messages)
    * [Enviar transacciones](/sdk/headless-wallet/use-wallets#send-transactions)
    * [Llamar a contratos](/sdk/headless-wallet/use-wallets#call-any-contract)
    * [Verificar un usuario en tu backend](/sdk/headless-wallet/verification#prerequisite-deployment-of-a-parent-wallet)
  </Step>
</Steps>

<Tip>
  Estamos usando variables de ejemplo en el archivo .env, por lo que recomendamos crear un proyecto en [Sequence Builder](https://sequence.build/project/default/wallet/embedded-wallet) y [configurar tu propia configuración de billetera](/sdk/headless-wallet) y actualizar las variables de entorno para usar más proveedores sociales para autenticación, personalizar plantillas de correo electrónico y más.
</Tip>
