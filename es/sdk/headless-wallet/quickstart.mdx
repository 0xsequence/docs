Para integración con Unreal o Unity, vaya a [Unreal](/sdk/unreal/quickstart) o [Unity](/sdk/unity/installation).

<Steps>
  <Step title="Clonar repositorio inicial">
    Ejecute lo siguiente en su terminal para clonar el repositorio inicial.

    <CodeGroup>
      ```bash Javascript
      npx sequence-cli boilerplates create-email-embedded-wallet-react-starter
      ```

      Seleccione usar claves de prueba.
    </CodeGroup>
  </Step>

  <Step title="Conectar y autenticar">
    Vaya a `localhost:4444` e ingrese su correo electrónico para probar la autenticación.
  </Step>

  <Step title="Agregue sus claves de el Embedded Wallet">
    Visite Sequence Builder y [configure su Embedded Wallet](/solutions/builder/embedded-wallet/configuration) para obtener claves específicas de su proyecto y autenticar a sus usuarios. Simplemente reemplace las claves de configuración existentes en el archivo .env.
  </Step>

  <Step title="¡Use el Embedded Wallet de Sequence!">
    Ahora que tiene la autenticación funcionando, puede revisar nuestros ejemplos para las funciones más comunes:

    - [Firmar mensajes](/sdk/headless-wallet/use-wallets#sign-messages)
    - [Enviar transacciones](/sdk/headless-wallet/use-wallets#send-transactions)
    - [Llamar contratos](/sdk/headless-wallet/use-wallets#call-any-contract)
    - [Verificar un usuario en su backend](/sdk/headless-wallet/verification#prerequisite-deployment-of-a-parent-wallet)
  </Step>
</Steps>

<Tip>
  Estamos usando variables de ejemplo en el archivo .env, así que recomendamos crear un proyecto en [Sequence Builder](https://sequence.build/project/default/wallet/embedded-wallet) y [configurar su propia wallet](/sdk/headless-wallet) y actualizar las variables de entorno para usar más proveedores sociales para autenticación, personalizar plantillas de correo electrónico y más.
</Tip>
