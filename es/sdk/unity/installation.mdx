## Package Manager - Recomendado

### OpenUPM

1. [Instale OpenUPM](https://openupm.com/#get-started-with-cli-optional)
2. Abra una línea de comandos en la raíz de su proyecto de Unity
3. `openupm add xyz.0xsequence.waas-unity`

### o usando la interfaz de Package Manager

1. Asegúrese de tener Git 2.14.0 o superior instalado en su equipo
2. Abra el Package Manager (Ventana > Package Manager)
3. Haga clic en el ícono "+" en la ventana de Package Manager > "Add package from git URL..."
4. Pegue esta URL y haga clic en Add o presione Enter en su teclado `https://github.com/0xsequence/sequence-unity.git?path=/Packages/Sequence-Unity`

### Setup

1. Abra el Package Manager (Ventana > Package Manager)
2. Desde el Package Manager, haga clic en "Samples"
3. Importe "Setup" desde Samples
   ![Setup](/images/unity/unity-import-setup.png)

a) Esto importará una carpeta Resources con algunos prefabs de UI prediseñados para que los uses en tu proyecto.
4\. Importe `TMP Essentials` (si aún no lo ha hecho). Nota: Unity debería solicitarle esto automáticamente si intenta ejecutar o compilar con un objeto TextMeshPro en su escena.

![Importar TMPro](/images/unity/unity-import-tmpro.png)

### Ejemplos
Además de esta documentación, también le proporcionamos algunos ejemplos para ayudarle a usar nuestro SDK.

Puede encontrarlos de la siguiente manera:
1. Abrir el Package Manager
2. Buscar nuestro SDK "Sequence WaaS SDK"
3. Hacer clic en "Samples"
4. Hacer clic en "Import" junto a cualquier ejemplo que desee instalar. Esto creará una carpeta Samples bajo Assets e importará el ejemplo allí. El ejemplo será completamente editable ya que estará dentro de su carpeta Assets.

![Ejemplos](/images/unity/unity-package-manager-samples.png)

La segunda muestra, "Demo Scene", es una escena de ejemplo que muestra algunas de las funciones de nuestro SDK. Sirve como un complemento útil a la documentación. Tenga en cuenta que debe importar `Setup` para que esta demo funcione correctamente.

La tercera muestra, "Useful Scripts", incluye scripts y Prefabs útiles que facilitan la integración del SDK.

## Manual
Instalar mediante el Package Manager dejará el SDK en modo solo lectura. Nota: aún podrá modificar la interfaz de usuario al usar el Package Manager.

Para la mayoría de los casos de uso, esto es ideal. Sin embargo, entendemos que algunos usuarios avanzados querrán la flexibilidad de editar el SDK según lo necesiten.

Tenga en cuenta que si elige este camino, actualizar el SDK será más complicado porque cualquier cambio que realice será sobrescrito y estos cambios pueden pasar desapercibidos fácilmente en un sistema de control de versiones. En general, creemos que es más seguro importar el SDK en modo solo lectura mediante el Package Manager y escribir wrappers para extenderlo según sea necesario, pero le damos la libertad de trabajar con nuestro SDK como prefiera.
1. Agregue [Newtonsoft.json](https://docs.unity3d.com/Packages/com.unity.nuget.newtonsoft-json@3.0/manual/index.html) a su proyecto (si aún no está) mediante el package manager. Haga clic en el ícono de "+" en la ventana del Package Manager > "Add package by name..." y agregue `com.unity.nuget.newtonsoft-json`.
2. Descargue [la última versión del Sequence Unity SDK UnityPackage](https://github.com/0xsequence/sequence-unity/releases)
3. Arrastre el archivo `.unitypackage` a la ventana de su proyecto e impórtelo
4. Importe `TMP Essentials` (si aún no lo ha hecho). Nota: Unity debería solicitarle esto automáticamente si intenta ejecutar o compilar con un objeto TextMeshPro en su escena.
