# Introducción

El SDK de Sequence Unity Embedded Wallet proporciona integración completa de [Embedded Wallet](/sdk/headless-wallet) y [Indexer](/api-references/indexer/overview) para tus Juegos Unity, integrado con nuestra propia biblioteca SequenceEthereum construida específicamente para Unity. ¡Así es, no se requiere Nethereum!

Este SDK sigue el [Versionado Semántico](https://semver.org/) (`major.minor.patch`). Los cambios incompatibles siempre causarán un incremento de versión `major`, las nuevas características no incompatibles causarán un incremento de versión `minor`, y las correcciones de errores causarán un incremento de versión `patch`.

## Requisitos

Unity 2021.3.6f1 o posterior

* Android
* iOS
* PC standalone -> (Solo compilaciones Mono cuando se usa inicio de sesión social -> los comandos del sistema lanzan excepciones al hacer compilaciones IL2CPP -> ver [OpenIdAuthentication.PlatformSpecificSetup](https://github.com/0xsequence/sequence-unity/blob/master/Assets/Sequence/SequenceSDK/Authentication/OpenIdAuthenticator.cs#L231))
* Mac standalone -> (Solo compilaciones Mono cuando se usa inicio de sesión social -> en nuestras pruebas MacOS no detecta esquemas URL personalizados automáticamente a menos que ejecutes algunos comandos del sistema primero; estos comandos del sistema solo funcionan en Mono -> ver [OpenIdAuthentication.PlatformSpecificSetup](https://github.com/0xsequence/sequence-unity/blob/master/Assets/Sequence/SequenceSDK/Authentication/OpenIdAuthenticator.cs#L231))
* WebGL
