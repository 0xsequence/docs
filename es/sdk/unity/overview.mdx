# Introducción
El Sequence Unity Embedded Wallet SDK proporciona integración completa de Sequence [Wallet embebida](/sdk/headless-wallet) e [Indexer](/api-references/indexer/overview) para sus juegos en Unity, integrado con nuestra propia biblioteca SequenceEthereum diseñada especialmente para Unity. ¡Así es, no necesita Nethereum!

Este SDK sigue la [Versionado Semántico](https://semver.org/) (`major.minor.patch`). Los cambios incompatibles siempre incrementarán la versión `major`, las nuevas funciones compatibles incrementarán la versión `minor`, y las correcciones de errores incrementarán la versión `patch`.

## Requisitos
Unity 2021.3.6f1 o posterior
- Android
- iOS
- PC standalone -> (solo se permiten builds Mono cuando se utiliza inicio de sesión con redes sociales -> los comandos del sistema lanzan excepciones al hacer builds IL2CPP -> vea [OpenIdAuthentication.PlatformSpecificSetup](https://github.com/0xsequence/sequence-unity/blob/master/Assets/Sequence/SequenceSDK/Authentication/OpenIdAuthenticator.cs#L231))
- Mac standalone -> (solo se permiten builds Mono cuando se utiliza inicio de sesión con redes sociales -> en nuestras pruebas, MacOS no reconoce automáticamente los esquemas de URL personalizados a menos que ejecute algunos comandos del sistema primero; estos comandos solo funcionan en Mono -> vea [OpenIdAuthentication.PlatformSpecificSetup](https://github.com/0xsequence/sequence-unity/blob/master/Assets/Sequence/SequenceSDK/Authentication/OpenIdAuthenticator.cs#L231))
- WebGL
