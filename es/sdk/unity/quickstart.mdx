---
title: Quickstart
description: Documentación de inicio rápido para el SDK de Unity de Sequence.
---

<Steps>
  <Step title="Install the Unity SDK">
    Instala la última versión del SDK de Unity de Sequence desde [OpenUPM](/sdk/unity/installation#openupm),
    o usa [Unity's Package Manager UI](/sdk/unity/installation#or-using-package-manager-ui)

    y usa la siguiente URL de Git `https://github.com/0xsequence/sequence-unity.git?path=/Packages/Sequence-Unity`
  </Step>

  <Step title="Configure the SDK">
    Inicia sesión y crea un proyecto en [Sequence Builder](https://sequence.build).

    Descarga tu archivo de configuración desde Builder como se muestra a continuación.

    Alternativamente, obtén tu `Project Access Key` y `WaaS Configuration Key` desde `Onboard > Embedded Wallet`,
    y agrégalos a tu [Archivo de Configuración de Sequence del Proyecto](/sdk/unity/setup).

    <Frame>
      <img src="/images/unity/builder_config_file.png" />
    </Frame>
  </Step>

  <Step title="Integrate your Login Process">
    Comienza estableciendo una sesión de billetera usando un OTP por correo electrónico. Importa el `Setup` Sample desde el Package Manager UI
    que colocará un conjunto de plantillas en tu proyecto en un directorio `Resources/`.
    Crea el [Login Boilerplate](/sdk/unity/bootstrap_game#login) para enviar una contraseña de un solo uso a la dirección de correo electrónico especificada.

    Una vez que hayas integrado tus primeras funciones, puedes continuar con [proveedores de inicio de sesión adicionales](/sdk/unity/onboard/authentication/intro)

    como Google, Apple o PlayFab.

    ```csharp
    BoilerplateFactory.OpenSequenceLoginWindow(parent);
    ```

    <Frame>
      <img src="/images/unity/unity_boilerplate_login.png" />
    </Frame>
  </Step>

  <Step title="Test Sequence's Features">
    El SDK de Unity de Sequence incluye una variedad de [Boilerplates](/sdk/unity/bootstrap_game) para ayudarte a comenzar rápidamente tu juego.
    Una vez que todo esté configurado, puedes crear prefabs para mostrar un Perfil de Jugador, Inventario o Tienda en el Juego.
    Revisa [cómo integrar un Perfil de Jugador.](/sdk/unity/bootstrap_game#player-profile)

    ```csharp
    BoilerplateFactory.OpenSequencePlayerProfile(parent, wallet, chain);
    ```

    <Frame>
      <img src="/images/unity/unity_boilerplate_profile.png" />
    </Frame>
  </Step>
</Steps>
