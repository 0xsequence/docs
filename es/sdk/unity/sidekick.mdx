# Resumen
Sequence Sidekick permite interactuar fácilmente con funciones blockchain directamente desde el Editor de Unity, sin necesidad de ejecutar el juego ni salir del entorno de Unity.

Pruebe funciones blockchain con wallets, contratos y más, todo desde Unity. Para usos avanzados y detalles de la API, consulte la [documentación completa de Sequence Sidekick](https://docs.sequence.xyz/solutions/sidekick/overview).

## Requisitos previos
Antes de usar la integración con Unity, debe instalar y configurar el servicio Sequence Sidekick en su sistema. Siga la guía oficial aquí:

**[Guía de configuración de Sequence Sidekick](https://docs.sequence.xyz/solutions/sidekick/overview)**

Esta guía explica cómo descargar e iniciar Sidekick.

## Instrucciones para configurar Unity
Una vez que Sidekick esté configurado, siga estos pasos para integrarlo con Unity:

### 1. Cree un ScriptableObject SidekickConfig
En el Editor de Unity:
- Abra el menú `Assets > Create > Sequence > SidekickConfig`.
- Cree un nuevo asset `SidekickConfig`.
- En el Inspector, establezca la ruta a su ejecutable local de Sidekick.

Esta ruta le indica a la integración de Unity cómo iniciar el servicio Sidekick desde el editor.

### 2. Use la pestaña SequenceDev
Después de configurar la ruta de Sidekick:
- Abra la pestaña `SequenceDev` en la barra de herramientas del Editor de Unity.
- Desde aquí puede **Iniciar** y **Detener** el servicio Sidekick.

Esto da control total sobre la ejecución de Sidekick dentro del entorno de Unity.

### 3. Agregue SidekickEditorController a su escena
Para comenzar a interactuar con la blockchain:
- Agregue el script `SidekickEditorController` en cualquier escena.
- Este componente permite la comunicación entre Unity y el backend de Sidekick.

> No necesita entrar en el modo Play para usar las funciones de Sidekick.
