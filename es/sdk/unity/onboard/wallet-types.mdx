# Tipos de wallet
El Sequence Unity SDK soporta dos tipos de wallet:

## Wallet embebida
La [Wallet embebida](/sdk/headless-wallet) es una wallet que está integrada en tu juego. Es una wallet no custodial que es controlada por el usuario. El usuario puede firmar transacciones y mensajes con esta wallet. La Wallet embebida es el tipo de wallet recomendado para la mayoría de los juegos.

## Wallet universal
La [Wallet universal](/sdk/typescript/guides/frontend/connect-wallet) es una wallet que puede conectarse a cualquier wallet externa. Es una wallet no custodial que es controlada por el usuario. El usuario puede firmar transacciones y mensajes con esta wallet. La Wallet universal es recomendada para juegos que desean admitir wallets externas.
