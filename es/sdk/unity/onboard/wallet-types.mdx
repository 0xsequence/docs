# Tipos de Billetera

El SDK de Unity de Sequence soporta dos tipos de billeteras:

## Billetera Integrada

La [Billetera Integrada](/sdk/headless-wallet) es una billetera que está integrada en tu juego. Es una billetera no custodial que es controlada por el usuario. El usuario puede firmar transacciones y mensajes con esta billetera. La Billetera Integrada es el tipo de billetera recomendado para la mayoría de los juegos.

## Billetera Universal

La [Universal Wallet](/sdk/typescript/guides/frontend/connect-wallet) es una billetera que puede conectarse a cualquier billetera externa. Es una billetera no custodial que es controlada por el usuario. El usuario puede firmar transacciones y mensajes con esta billetera. La Universal Wallet es recomendada para juegos que quieren soportar billeteras externas.
