# Conectando wallets externas
Aunque la mayoría de las interacciones de los usuarios ocurrirán usando la wallet embebida, algunos de sus usuarios pueden tener wallets existentes que deseen conectar con sus juegos. Esto le permite validar que un usuario posee uno o más tokens sin requerir que los transfiera a su wallet embebida.

Puede conectar una wallet externa con una Embedded Wallet a través de una interfaz web. Esto le da acceso de solo lectura a cualquier wallet externa que haya vinculado a su Embedded Wallet.

Primero, debe configurar una aplicación web para facilitar esta conexión. Por favor, [lea esta guía](/solutions/wallets/developers/embedded-wallet/wallet-linking) y regrese una vez que haya configurado su aplicación web.

Ahora que ya configuró su aplicación web, puede dirigir a los jugadores a su URL. Una vez que inicien sesión en su aplicación web, podrán vincular sus wallets EOA.

### Vincule usuarios a su aplicación web

```csharp
Application.OpenURL("https://your-web-app.io/");
```

Una vez que una wallet está vinculada, simplemente puede llamar a las siguientes APIs de Unity para obtener todas las wallets vinculadas o desvincular una wallet.

### Obtener wallets vinculadas

```csharp
EOAWalletLinker linker = new EOAWalletLinker(wallet, Chain.Polygon);
LinkedWalletData[] wallets = await linker.GetLinkedWallets();
```

### Desvincular wallet

```csharp
EOAWalletLinker linker = new EOAWalletLinker(wallet, Chain.Polygon);
bool result = await linker.UnlinkWallet("0xabc...");
```
