# Conectando Billeteras Externas

Si bien la mayoría de las interacciones del usuario ocurrirán usando la billetera integrada, algunos de sus usuarios pueden tener billeteras existentes que deseen conectar con sus juegos. Esto le permite validar que un usuario posee un token(s) sin requerir que lo transfiera a su billetera integrada.

Puede [conectar una billetera externa con una billetera integrada](/sdk/web/overview) a través de una interfaz web.

Primero, querrá configurar un servidor para facilitar esta conexión. Por favor [lea esta guía](/sdk/web/overview) y vuelva una vez que haya configurado su servidor (puede omitir los primeros pasos sobre la configuración del Builder y la integración de la billetera integrada en una aplicación web ya que ya lo ha hecho en el lado de Unity; de manera similar, puede omitir las secciones sobre la firma de la prueba de autenticación de sesión).

Ahora que has configurado tu servidor, puedes crear un `EOAWalletLinker`.

```csharp
EOAWalletLinker linker = new EOAWalletLinker(wallet, "yourServersBaseUrl/generateNonce");
```

Puedes generar la URL para vincular la billetera con

```csharp
string url = await linker.GenerateEOAWalletlink(chain);
```

o puedes generarla y abrirla con el navegador predeterminado del dispositivo (a través de [Application.OpenURL](https://docs.unity3d.com/ScriptReference/Application.OpenURL.html))

```csharp
linker.OpenEOAWalletLink(chain); // note: this is an awaitable async method.
```
