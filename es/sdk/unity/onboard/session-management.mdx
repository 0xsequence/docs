---
title: Gestión de Sesiones
---

Una vez que hayas autenticado a tu usuario con las APIs de Sequence y establecido una sesión, hay varios métodos disponibles para ti para gestionar la sesión.

## Listar Sesiones

Use el `ListSessions` método para encontrar todas las sesiones activas asociadas con este usuario/billetera.

## Eliminar Sesión

Si desea eliminar/finalizar una sesión, puede usar el `DropSession` método y proporcionar el id de la sesión que desea eliminar.

## Eliminar Esta Sesión

El `DropThisSession` método se puede usar si desea eliminar/finalizar la sesión actual. En otras palabras, si desea cerrar la sesión del usuario.

<Note>Si está usando el `LoginPanel` y no ha habilitado `StoreSessionPrivateKeyInSecureStorage` o está en una plataforma no compatible (ver [Recuperación de Sesiones](/sdk/unity/onboard/recovering-sessions)), el `WaaSSessionManager` prefab adjunto al `LoginPanel` llamará automáticamente a este método `OnApplicationQuit`</Note>

## Obtener Lista de Cuentas

Si desea obtener las `Account`s asociadas con la billetera del usuario, puede usar el `GetAccountList` método.

## Obtener Token de Id

Si desea recuperar un token JWT de id desde las APIs de Sequence para validar la autenticación en su backend, puede usar el `GetIdToken` método y opcionalmente proporcionar un nonce.
