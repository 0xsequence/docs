---
title: Gestión de sesiones en Unity — Documentación de Sequence
---

Una vez que haya autenticado a su usuario con las APIs de Sequence y establecido una sesión, tiene disponibles varios métodos para gestionar la sesión.

## Listar sesiones
Utilice el método `ListSessions` para encontrar todas las sesiones activas asociadas a este usuario/wallet.

## Cerrar sesión
Si desea cerrar o finalizar una sesión, puede usar el método `DropSession` y proporcionar el id de la sesión que desea cerrar.

## Cerrar esta sesión
El método `DropThisSession` puede usarse si desea cerrar o finalizar la sesión actual. En otras palabras, si desea cerrar la sesión del usuario.

## Obtener lista de cuentas
Si desea obtener las `Account`s asociadas con la wallet del usuario, puede usar el método `GetAccountList`.

## Obtener Id Token
Si desea obtener un id token JWT de las APIs de Sequence para validar la autenticación en su backend, puede usar el método `GetIdToken` y opcionalmente proporcionar un nonce.
