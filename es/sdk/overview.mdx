## Características Principales

* **Soporte Multi-Plataforma**: Construye aplicaciones en plataformas web, móviles y de juegos
* **Integración de Billetera**: Conectores de billetera fáciles de usar para frameworks populares
* **Interacción con Contratos Inteligentes**: Herramientas para desplegar e interactuar con contratos inteligentes
* **Gestión de Transacciones**: Creación, firma y envío de transacciones simplificado
* **Seguridad de Tipos**: Fuerte soporte de tipado en implementaciones de TypeScript y Go
* **Documentación Extensa**: Guías detalladas y referencias de API para cada SDK

### Comparación de Características

Esta tabla proporciona una comparación de alto nivel de las características disponibles en los SDK de Sequence Headless Wallet, Web, Unity y Unreal.

| Característica              | Typescript Wallet SDK        | Web SDK (React)                       | Unity SDK                    | Unreal SDK                   |
| :-------------------------- | :--------------------------- | :------------------------------------ | :--------------------------- | :--------------------------- |
| **Plataforma Objetivo**     | Web (Headless)               | Web (React)                           | Unity                        | Unreal Engine                |
| **Componentes UI**          | No                           | Sí (Preconfigurados, Personalizables) | Prefabs                      | Prefabs                      |
| **Autenticación**           | Sí (Email, Social, Invitado) | Sí (Email, Social, Invitado)          | Sí (Email, Social, Invitado) | Sí (Email, Social, Invitado) |
| **Soporte de Billetera**    | Embedded / Ecosystem         | Embedded / Ecosystem / External       | Embedded / Ecosystem         | Embedded / Ecosystem         |
| **Inventario**              | No                           | Sí                                    | Sí                           | Sí                           |
| **On-Ramp**                 | No                           | Via Wallet UI                         | Via Prefabs                  | Via Prefabs                  |
| **Intercambios Integrados** | No                           | Sí                                    | Sí                           | Sí                           |
| **Checkout**                | No                           | Sí                                    | Sí                           | Sí                           |

**Nota:** Las características listadas como "Via Core SDK" o similar significan que aunque el SDK específico no implemente la característica directamente, se basa en los servicios subyacentes de Sequence o el SDK principal de TypeScript para proporcionar esa funcionalidad, a menudo facilitada por la experiencia integrada de la billetera.

## Comenzando

Elige el SDK que mejor se adapte a tu plataforma y necesidades de desarrollo:

* Para aplicaciones web:
  * Usa [Web-SDK](/sdk/web/overview) para aplicaciones React
  * Usa el [TypeScript/JavaScript SDK](/sdk/typescript/overview) para funcionalidad principal

Para desarrollo de juegos:

* Usa el [Unity SDK](/sdk/unity/overview) para juegos Unity
* Usa el [Unreal SDK](/sdk/unreal/introduction) para juegos Unreal Engine
  Para servicios backend, consulta el [Go SDK](/sdk/go/overview)

  Para aplicaciones móviles, explora el [React Native SDK](/sdk/mobile)
