Ofrecemos a sus usuarios una experiencia de incorporación fluida y segura. Con nuestras herramientas robustas construidas sobre la popular librería [wagmi](https://wagmi.sh/), desbloquee un mundo de posibilidades en el mundo de web3.

Web SDK está compuesto por 4 paquetes principales:
- @0xsequence/connect
- @0xsequence/checkout
- @0xsequence/hooks
- @0xsequence/wallet-widget

## Características clave

-   **Conexiones universales**: Conéctese fácilmente mediante inicio de sesión con redes sociales populares como Facebook, Google, Discord y más. Sus usuarios disfrutarán de un proceso de incorporación fluido y seguro.

-   **Wallets integrados**: Aproveche nuestro wallet no custodial integrado dentro del modal de conexión del Web SDK para poner web3 en segundo plano, como ventanas emergentes opcionales, para que los usuarios se enfoquen en la experiencia.

-   **Integración con Wallets Externos**: Integre fácilmente con los principales wallets web3 como WalletConnect y MetaMask. ¡Desate el poder de blockchain con solo unos clics!

-   **Inventario Web3**: Ofrezca a sus usuarios un inventario habilitado para web3, permitiéndoles gestionar sus monedas y coleccionables, todo dentro de su propia aplicación.

-   **Checkout de NFT**: Permita que los usuarios compren coleccionables dentro de Sequence Checkout usando tarjeta de crédito o criptomonedas.

-   **Personalización**: Personalice el modal de conexión del Web SDK con su propio logo, esquema de colores y configúrelo con los proveedores sociales y wallets que desea ofrecer.

¡Descubra el potencial de nuestros Web SDK probando nuestro [demo](https://0xsequence.github.io/web-sdk/)!
