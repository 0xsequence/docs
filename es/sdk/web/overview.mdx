Proporcionamos a sus usuarios una experiencia de incorporación fluida y segura. Con nuestras herramientas robustas construidas sobre el popular [wagmi](https://wagmi.sh/) biblioteca, desbloquea un mundo de posibilidades en el mundo de web3.

El SDK Web está compuesto por 3 paquetes principales:

* @0xsequence/connect
* @0xsequence/checkout
* @0xsequence/hooks
* @0xsequence/wallet-widget

## Características Principales

* **Conexiones Universales**: ¡Conéctate sin problemas a través de inicios de sesión sociales populares como Facebook, Google, Discord y más! Tus usuarios disfrutarán de un proceso de incorporación fluido y seguro.

* **Billeteras Integradas**: Aprovecha nuestra billetera integrada no custodial dentro del modal de conexión del SDK Web para mantener web3 en segundo plano, como ventanas emergentes/modales opcionales, para que los usuarios puedan concentrarse en la experiencia.

* **Integración de Billeteras Externas**: ¡Integra sin esfuerzo con las principales billeteras web3 como WalletConnect y MetaMask. ¡Libera el poder de la blockchain con solo unos clics!

* **Inventario Web3**: Proporciona a tus usuarios un inventario habilitado para web3, permitiéndoles gestionar sus monedas y coleccionables dentro de tu propia aplicación.

* **Pago de NFT**: Permite a los usuarios comprar coleccionables dentro de Sequence Checkout usando tarjeta de crédito o cripto.

* **Personalización**: Personaliza el modal de conexión del SDK Web con tu propio logotipo, esquema de colores y configúralo con tus proveedores sociales y billeteras que desees ofrecer.

Explora el potencial de nuestro SDK Web probando nuestra [demo](https://0xsequence.github.io/web-sdk/)!
