## Importar

```tsx
import { useSignInEmail } from '@0xsequence/connect'
```

## Uso

```tsx
import { useSignInEmail } from '@0xsequence/connect'

function App() {
  const email = useSignInEmail()
  
  return (
    <div>
      <h2>User Information</h2>
      {email ? (
        <div>
          <p>Connected with email: {email}</p>
        </div>
      ) : (
        <p>No email associated with connected wallet</p>
      )}
    </div>
  )
}
```

## Tipo de Retorno

```tsx
string | null
```

El hook devuelve:

* Un `string` que contiene la dirección de correo electrónico cuando una billetera está conectada y tiene un correo electrónico asociado
* `null` cuando:
  * No hay una billetera conectada
  * La billetera conectada no tiene un correo electrónico asociado
  * La billetera ha sido desconectada

## Gestión de Estado

El hook gestiona el estado del correo electrónico mediante:

1. Monitoreo del estado de conexión de la billetera
2. Recuperación del correo electrónico almacenado cuando una billetera se conecta
3. Borrado del correo electrónico cuando la billetera se desconecta
4. Persistencia del correo electrónico en el almacenamiento local usando la `WaasSignInEmail` clave

## Notas

Este hook está diseñado para funcionar con ambos:

* Sequence WaaS (Wallet-as-a-Service) wallets
* Tipos de billeteras universales

El hook se integra con el sistema de almacenamiento de Sequence para mantener la persistencia del correo electrónico a través de las sesiones, asegurando una limpieza adecuada cuando las billeteras son desconectadas.
