## Importar

```tsx
import { useStorage, useStorageItem } from '@0xsequence/connect'
```

### Uso

```tsx
import { useStorage, signEthAuthProof, validateEthProof } from '@0xsequence/connect'
import { useWalletClient, usePublicClient } from 'wagmi'

function App() {
  const { data: walletClient } = useWalletClient()
  const publicClient = usePublicClient()
  const storage = useStorage()
  
  const generateEthAuthProof = async () => {
    if (!walletClient || !publicClient || !storage) {
      return
    }
    
    try {
      // Use storage to generate an auth proof
      const proof = await signEthAuthProof(walletClient, storage)
      console.log('proof:', proof)
      
      const isValid = await validateEthProof(walletClient, publicClient, proof)
      console.log('isValid?:', isValid)
    } catch (e) {
      console.error(e)
    }
  }
  
  return (
    <button onClick={generateEthAuthProof}>
      Generate EthAuth Proof
    </button>
  )
}
```

### Tipo de Retorno

`Storage<StorageItem> | null`

Devuelve la instancia de Storage si está disponible, o null si no está configurada.

## useStorageItem

Hook para recuperar un elemento específico del almacenamiento de Sequence Connect.

### Uso

```tsx
import { useStorageItem } from '@0xsequence/connect'

function App() {
  const { data: authToken, isLoading } = useStorageItem('authToken')
  
  if (isLoading) {
    return <div>Loading...</div>
  }
  
  return (
    <div>
      {authToken ? 'Authenticated' : 'Not authenticated'}
    </div>
  )
}
```

### Parámetros

| Parámetro | Tipo                | Descripción                                         |
| --------- | ------------------- | --------------------------------------------------- |
| `key`     | `keyof StorageItem` | La clave del elemento de almacenamiento a recuperar |

### Tipo de Retorno

`UseQueryResult<StorageItem[K]>`

Devuelve un resultado de react-query que contiene los datos del elemento de almacenamiento, con las siguientes propiedades:

| Propiedad   | Tipo             | Descripción                                         |
| ----------- | ---------------- | --------------------------------------------------- |
| `data`      | `StorageItem[K]` | Los datos del elemento de almacenamiento recuperado |
| `isLoading` | `boolean`        | Si los datos están cargando actualmente             |
| `isError`   | `boolean`        | Si ocurrió un error durante la obtención de datos   |
| `error`     | `Error`          | El objeto de error si ocurrió un error              |
| ...         | ...              | Otras propiedades de react-query                    |

## Notas

Estos hooks proporcionan acceso a la capa de almacenamiento utilizada por Sequence Connect para persistir datos de autenticación, estado de la cartera y otras necesidades de almacenamiento del lado del cliente. El `useStorage` hook se usa comúnmente para operaciones que requieren acceso directo a la capa de almacenamiento, como generar pruebas de autenticación, mientras que `useStorageItem` proporciona una forma conveniente de acceder a elementos específicos con integración react-query.
