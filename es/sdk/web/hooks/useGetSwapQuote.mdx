## Importar

```tsx
import { useGetSwapQuote } from '@0xsequence/hooks'
```

## Uso

```tsx
import { useGetSwapQuote } from '@0xsequence/hooks'

function SwapComponent() {
  const { data: swapQuote, isLoading } = useGetSwapQuote({
    userAddress: '0x123...',
    buyCurrencyAddress: '0x456...',
    sellCurrencyAddress: '0x789...',
    buyAmount: '1000000000000000000', 
    chainId: 1,
    includeApprove: true
  })

  if (isLoading) return <div>Loading...</div>

  return (
    <div>
      {swapQuote && (
        <div>
          Currency: {swapQuote.currencyAddress}
          Price: {swapQuote.price}
          Max Price: {swapQuote.maxPrice}
          Transaction Value: {swapQuote.transactionValue}
        </div>
      )}
    </div>
  )
}
```

## Tipo de Retorno: `UseQueryResult<SwapQuote>`

El hook devuelve todas las propiedades de React Query's `UseQueryResult` con datos de cotización de intercambio. Aquí está la estructura detallada:

```tsx
interface SwapQuote {
  currencyAddress: string
  currencyBalance: string
  price: string
  maxPrice: string
  to: string
  transactionData: string
  transactionValue: string
  approveData: string
}
```

### Propiedades

#### data

`SwapQuote | undefined`

El objeto de cotización de intercambio contiene:

* `currencyAddress`: Dirección de la moneda a intercambiar
* `currencyBalance`: Saldo de la moneda en la billetera del usuario
* `price`: El precio actual para el intercambio
* `maxPrice`: Precio máximo permitido para el intercambio (incluye deslizamiento)
* `to`: La dirección del contrato objetivo para el intercambio
* `transactionData`: Datos de transacción codificados para ejecutar el intercambio
* `transactionValue`: El valor a enviar con la transacción
* `approveData`: Datos de transacción de aprobación codificados (si includeApprove es verdadero)

#### isLoading

`boolean`

Estado de carga para la obtención de datos.

#### isError

`boolean`

Estado de error que indica si la consulta falló.

#### error

`Error | null`

Cualquier error que ocurrió durante la obtención de datos.

## Parámetros

El hook acepta dos parámetros:

### args: `UseGetSwapQuoteArgs`

```tsx
interface UseGetSwapQuoteArgs {
  userAddress: string
  buyCurrencyAddress: string
  sellCurrencyAddress: string
  buyAmount: string
  chainId: number
  includeApprove?: boolean
  slippagePercentage?: number
}
```

| Parámetro             | Tipo      | Descripción                                              |
| --------------------- | --------- | -------------------------------------------------------- |
| `userAddress`         | `string`  | La dirección de la billetera del usuario                 |
| `buyCurrencyAddress`  | `string`  | La dirección de la moneda a comprar                      |
| `sellCurrencyAddress` | `string`  | La dirección de la moneda a vender                       |
| `buyAmount`           | `string`  | La cantidad de moneda a comprar (en unidades base)       |
| `chainId`             | `number`  | El ID de la cadena donde ocurrirá el intercambio         |
| `includeApprove`      | `boolean` | (Opcional) Si incluir datos de transacción de aprobación |
| `slippagePercentage`  | `number`  | (Opcional) Porcentaje máximo de deslizamiento permitido  |

### options: `HooksOptions`

```tsx
interface HooksOptions {
  disabled?: boolean
  retry?: boolean
}
```

| Parámetro  | Tipo      | Descripción                                                        |
| ---------- | --------- | ------------------------------------------------------------------ |
| `disabled` | `boolean` | (Opcional) Deshabilitar que la consulta se ejecute automáticamente |
| `retry`    | `boolean` | (Opcional) Si reintentar consultas fallidas                        |
