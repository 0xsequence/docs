## Descripción general
[@0xsequence/connect v6](https://www.npmjs.com/package/@0xsequence/connect/v/6.0.0-beta.4) es el SDK de React Hooks para el Ecosystem Wallet.
Combina inicio de sesión con redes sociales, passkeys, recuperación de cuentas, vinculación de wallets, gestión de sesiones y permisos para ofrecer a sus usuarios una experiencia completa similar a la de Web2.
Con un tamaño de paquete reducido, es la forma más rápida de agregar una experiencia completa de smart wallet no custodial del ecosistema a su DApp.

### Características clave

- Inicio de sesión con redes sociales (Email, Google, Apple)
- Passkeys
- Recuperación de cuenta
- Vinculación de wallets
- Gestión de sesiones inteligentes
- Temas personalizables
- Componentes de UI integrados
