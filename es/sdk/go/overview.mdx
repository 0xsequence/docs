## Go

Un SDK completo de Sequence también está disponible en Go: [https://github.com/0xsequence/go-sequence](https://github.com/0xsequence/go-sequence).

[go-sequence](https://github.com/0xsequence/go-sequence) es el equivalente de [0xsequence](https://github.com/0xsequence/sequence.js)
pero para backends Go / Golang. De hecho, toda la infraestructura de Sequence está escrita en Go y está construida con [go-sequence](https://github.com/0xsequence/go-sequence).

## ethkit

Como parte de las herramientas de código abierto de Sequence, nuestro equipo también ha construido [ethkit](https://github.com/0xsequence/ethkit), que
es un kit de desarrollo de Ethereum para backends Go. [ethkit](https://github.com/0xsequence/ethkit) soporta billeteras EOA,
y puedes pensar en ello como `ethers.js` pero para Go.
