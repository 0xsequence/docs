## Go
También hay disponible un SDK completo de Sequence en Go: [https://github.com/0xsequence/go-sequence](https://github.com/0xsequence/go-sequence).

[go-sequence](https://github.com/0xsequence/go-sequence) es el equivalente de [0xsequence](https://github.com/0xsequence/sequence.js)
pero para backends en Go / Golang. De hecho, toda la infraestructura de Sequence está escrita en Go y se construye con [go-sequence](https://github.com/0xsequence/go-sequence).

## ethkit
Como parte de las herramientas open source de Sequence, nuestro equipo también ha creado [ethkit](https://github.com/0xsequence/ethkit), que
es un toolkit de desarrollo Ethereum para backends en Go. [ethkit](https://github.com/0xsequence/ethkit) es compatible con wallets EOA,
y puede pensarse como `ethers.js` pero para Go.
