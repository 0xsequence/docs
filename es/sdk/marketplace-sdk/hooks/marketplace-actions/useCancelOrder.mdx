---
title: useCancelOrder
description: El hook useCancelOrder permite a los usuarios cancelar una publicación de un NFT que está actualmente a la venta en el Marketplace. Gestiona el proceso de cancelación y la ejecución de la transacción.
sidebarTitle: useCancelOrder
---

```ts
import { useCancelOrder } from '@0xsequence/marketplace-sdk/react';

## Into your React component:

const { cancelOrder } = useCancelOrder({
  chainId,
  collectionAddress
});

const onClickCancel = () => {
  cancelOrder({
    marketplace,
    orderId,
  });
};

return <button onClick={onClickCancel}>Cancel Order</button>;
```
