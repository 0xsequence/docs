description: Hook para obtener los listados de un coleccionable específico. Obtiene los listados activos (ventas) de un token específico desde el marketplace, con soporte para filtrado y paginación.
sidebarTitle: useListListingsForCollectible
category: Data Fetching
---

## Parámetros

| Nombre   | Type | Description                                                                                        |
| -------- | ---- | -------------------------------------------------------------------------------------------------- |
| `params` |      | Parámetros de configuración                                                                        |
| `params` |      | .chainId - El ID de la cadena (debe ser un número, por ejemplo, 1 para Ethereum, 137 para Polygon) |
| `params` |      | .collectionAddress - La dirección del contrato de la colección                                     |
| `params` |      | .collectibleId - El ID específico del token para obtener los listados                              |
| `params` |      | .filter - Parámetros opcionales de filtrado (marketplace, monedas, etc.)                           |
| `params` |      | .page - Parámetros opcionales de paginación                                                        |
| `params` |      | .query - Configuración opcional de React Query                                                     |

## Devuelve
Resultado de la consulta con los datos de los listados del coleccionable

## Ejemplo
Uso básico:

```typescript
const { data, isLoading } = useListListingsForCollectible({
chainId: 137,
collectionAddress: '0x...',
collectibleId: '123'
})
```

Con paginación:

```typescript
const { data } = useListListingsForCollectible({
chainId: 1,
collectionAddress: '0x...',
collectibleId: '456',
page: {
page: 2,
pageSize: 20
}
})
```

Con filtrado:

```typescript
const { data } = useListListingsForCollectible({
chainId: 137,
collectionAddress: '0x...',
collectibleId: '789',
filter: {
marketplace: [MarketplaceKind.sequence_marketplace_v2],
currencies: ['0x...'] // Specific currency addresses
}
})
```
