---
title: useCollectible
description: Hook para obtener los metadatos de un coleccionable específico. Este hook recupera los metadatos de un NFT individual de una colección, incluyendo propiedades como nombre, descripción, imagen, atributos, etc.
sidebarTitle: useCollectible
category: Data Fetching
---

## Parámetros

| Nombre   | Type | Description                                                                                        |
| -------- | ---- | -------------------------------------------------------------------------------------------------- |
| `params` |      | Parámetros de configuración                                                                        |
| `params` |      | .chainId - El ID de la cadena (debe ser un número, por ejemplo, 1 para Ethereum, 137 para Polygon) |
| `params` |      | .collectionAddress - La dirección del contrato de la colección                                     |
| `params` |      | .collectibleId - El ID del token del coleccionable                                                 |
| `params` |      | .query - Configuración opcional de React Query                                                     |

## Devuelve
Resultado de la consulta con los metadatos del coleccionable

## Ejemplo
Uso básico:

```typescript
const { data: collectible, isLoading } = useCollectible({
chainId: 137,
collectionAddress: '0x631998e91476da5b870d741192fc5cbc55f5a52e',
collectibleId: '12345'
})
```

Con opciones personalizadas de consulta:

```typescript
const { data } = useCollectible({
chainId: 137,
collectionAddress: '0x631998e91476da5b870d741192fc5cbc55f5a52e',
collectibleId: '12345',
query: {
enabled: Boolean(collectionAddress && tokenId),
staleTime: 30_000
}
})
```
