## Parámetros

| Nombre | Type | Description                            |
| ------ | ---- | -------------------------------------- |
| `args` |      | Los argumentos para obtener el balance |

## Devuelve
Resultado de la consulta con los datos del balance

## Ejemplo

```typescript
const { data, isLoading, error } = useBalanceOfCollectible({
  collectionAddress: "0x123...",
  collectableId: "1",
  userAddress: "0x456...",
  chainId: 1,
  query: {
    enabled: true,
    refetchInterval: 10000,
  },
});
```
