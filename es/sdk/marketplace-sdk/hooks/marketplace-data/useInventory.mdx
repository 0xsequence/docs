---
title: useInventory
description: Obtiene el inventario de coleccionables de un usuario para una colección específica con scroll infinito. Este hook combina datos tanto de la API del marketplace como del indexador para mostrar una vista completa de los coleccionables poseídos. Detecta automáticamente las colecciones LAOS ERC-721 y las gestiona adecuadamente. El hook garantiza que se devuelvan todos los tokens poseídos, incluso si no están listados en el marketplace.
sidebarTitle: useInventory
category: Data Fetching
---

## Parámetros

| Nombre | Type | Description                                                            |
| ------ | ---- | ---------------------------------------------------------------------- |
| `args` |      | Configuración para obtener el inventario                               |
| `args` |      | .accountAddress - La dirección de la wallet para obtener el inventario |
| `args` |      | .collectionAddress - La dirección del contrato de la colección         |
| `args` |      | .chainId - El ID de la red blockchain                                  |
| `args` |      | .query - Configuración opcional de la consulta                         |
| `args` |      | .query.enabled - Si se habilita la consulta (por defecto: true)        |

## Devuelve
returns.isFetchingNextPage - Verdadero mientras se obtiene la siguiente página

## Ejemplo
Uso básico con paginación:

```typescript
const {
data,
fetchNextPage,
hasNextPage,
isLoading
} = useInventory({
accountAddress: '0x...',
collectionAddress: '0x...',
chainId: 137
});
const allCollectibles = data?.pages.flatMap(page => page.collectibles) ?? [];
return (
<div>
{allCollectibles.map(item => (
<CollectibleCard
key={item.metadata.tokenId}
metadata={item.metadata}
balance={item.balance}
/>
))}
{hasNextPage && (
<button onClick={() => fetchNextPage()}>
Load More
</button>
)}
</div>
);
```

Con filtrado y visualización de balance:

```typescript
const { data } = useInventory({
accountAddress: userAddress,
collectionAddress: collection.address,
chainId: collection.chainId,
query: {
enabled: !!userAddress && isConnected
}
});
// Get all items across pages
const inventory = data?.pages.flatMap(p => p.collectibles) ?? [];
// Show ERC1155 balances
inventory.forEach(item => {
if (item.contractType === ContractType.ERC1155) {
console.log(`Token ${item.metadata.tokenId}: ${item.balance} owned`);
}
});
```
