[`@0xsequence/marketplace-sdk`](https://www.npmjs.com/package/@0xsequence/marketplace-sdk) le ayuda a integrar marketplaces de NFT en sus aplicaciones. Soporta tanto NFTs (ERC721) como tokens semi-fungibles (ERC1155), proporcionando una interfaz sencilla para trabajar con el marketplace de Sequence.

Características principales:
- API de marketplace simple e intuitiva
- Cree y gestione listados y ofertas de NFT
- Monitoree actividades y órdenes del marketplace
- Hooks y componentes React listos para usar
- Desarrollado con TypeScript
- Obtención de datos optimizada con React Query y Wagmi

¿Listo para comenzar? Consulte nuestra guía de [Primeros Pasos](/sdk/marketplace-sdk/getting-started).
