¿Por qué usar el Marketplace SDK? Ofrece una forma sencilla de integrar el marketplace de Sequence con un esfuerzo mínimo. Aproveche sus potentes hooks para obtener datos del marketplace y ejecutar acciones esenciales como listados, compras, ofertas y aceptación de ofertas.

# Hooks del Marketplace SDK
Marketplace SDK ofrece una colección de hooks optimizados para integrar de manera fluida y eficiente la funcionalidad de Marketplace en sus aplicaciones. Estos hooks permiten gestionar la experiencia del usuario de forma segura y sin fricciones.

## Hooks de acción
Los hooks de acción son esenciales para interactuar con el Marketplace en su aplicación. Son útiles para crear listados, hacer ofertas y realizar acciones de compra y venta.

<CardGroup cols={2}>
  <Card title="useBuyModal" icon="code" href="/sdk/marketplace-sdk/hooks/marketplace-actions/useBuyModal">
    El hook useBuyModal permite a los usuarios comprar un NFT que está listado en el Marketplace. Se encarga del proceso de compra y la ejecución de la transacción.
  </Card>

  <Card title="useCreateListingModal" icon="code" href="/sdk/marketplace-sdk/hooks/marketplace-actions/useCreateListingModal">
    El hook useCreateListingModal se utiliza para listar un artículo en un listado en el Marketplace. Proporciona la funcionalidad necesaria para crear y gestionar un nuevo listado.
  </Card>

  <Card title="useMakeOfferModal" icon="code" href="/sdk/marketplace-sdk/hooks/marketplace-actions/useMakeOfferModal">
    El hook useMakeOfferModal permite a los usuarios hacer una oferta por un NFT. Facilita la creación y el envío de ofertas dentro del Marketplace.
  </Card>

  <Card title="useSellModal" icon="code" href="/sdk/marketplace-sdk/hooks/marketplace-actions/useSellModal">
    El hook useSellModal se utiliza para vender un NFT aceptando una oferta existente. Proporciona la funcionalidad necesaria para completar el proceso de venta.
  </Card>

  <Card title="useCancelOrder" icon="code" href="/sdk/marketplace-sdk/hooks/marketplace-actions/useCancelOrder">
    El hook useCancelOrder permite a los usuarios cancelar una publicación de un NFT que está actualmente a la venta en el Marketplace. Gestiona el proceso de cancelación y la ejecución de la transacción.
  </Card>
</CardGroup>
