¿Por qué usar el SDK del Marketplace? Proporciona una forma perfecta de integrar el marketplace de Sequence con un esfuerzo mínimo. Aprovecha sus potentes hooks para recuperar datos del marketplace y ejecutar acciones esenciales como listados, compras, ofertas y aceptación de ofertas.
