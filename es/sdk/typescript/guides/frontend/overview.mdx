## Instalación

Sequence.js es un paquete modular de web3 stack y billetera inteligente para cadenas Ethereum. Puedes pensar en él de manera similar a ethers.js, pero adaptado al stack de Sequence que agiliza las funciones comunes necesarias para el desarrollo de juegos habilitados para blockchain. El SDK de sequence.js está compuesto por varios subpaquetes combinados en un único metapaquete llamado `0xsequence`.
Para agregar el SDK a tu proyecto, añade una dependencia de 0xsequence a tu package.json:

```sh
npm install 0xsequence ethers
```

o

```sh
pnpm install 0xsequence ethers
```

o

```sh
yarn add 0xsequence ethers
```

Si sabes que tu proyecto solo requiere subpaquetes específicos, también puedes depender directamente de ellos con`npm add @0xsequence/<sub-package name>`.

Tu package.json debería verse algo así (la versión actual puede variar):

```json
{
  "name": "my-dapp",
  ...
  "dependencies": {
    "0xsequence": "^1.0.0",
    "ethers": ^5.7.0"
    ...
  }
}
```

#### Distribución CDN como JS nativo

Opcionalmente, `0xsequence` puede ser instalado como una fuente de script JS (distribución UMD) vía:

```js
<script src="https://cdn.ethers.io/lib/ethers-5.7.umd.min.js"></script>
<script src="https://unpkg.com/0xsequence@latest/dist/0xsequence.umd.min.js"></script>
```
