## Instalación
Sequence.js es un paquete modular de web3 y smart wallet para cadenas de Ethereum. Puede considerarse similar a ethers.js, pero adaptado al stack de Sequence, que simplifica funciones comunes necesarias para el desarrollo de juegos con blockchain. El SDK de sequence.js está compuesto por varios subpaquetes combinados en un solo meta-paquete llamado `0xsequence`.\r\nPara agregar el SDK a su proyecto, añada una dependencia de 0xsequence a su package.json:

```sh
npm install 0xsequence ethers
```

o

```sh
pnpm install 0xsequence ethers
```

o

```sh
yarn add 0xsequence ethers
```

Si sabe que su proyecto solo requiere subpaquetes específicos, también puede agregarlos directamente con
`npm add @0xsequence/<sub-package name>`.

Su package.json debería verse algo así (la versión real puede variar):

```json
{
  "name": "my-dapp",
  ...
  "dependencies": {
    "0xsequence": "^1.0.0",
    "ethers": ^5.7.0"
    ...
  }
}
```

#### Distribución por CDN como JS nativo
Opcionalmente, `0xsequence` puede instalarse como un script JS (distribución UMD) mediante:

```js
<script src="https://cdn.ethers.io/lib/ethers-5.7.umd.min.js"></script>
<script src="https://unpkg.com/0xsequence@latest/dist/0xsequence.umd.min.js"></script>
```
