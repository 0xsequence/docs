### Soporte para otros lenguajes de backend

Si tus servicios de backend están escritos en un lenguaje diferente a JS o Go, aún puedes integrar Sequence fácilmente, ya que Sequence es realmente solo una biblioteca de cliente Ethereum estándar con algunas características adicionales. En muchos casos, las características adicionales se aprovechan mejor en el lado del cliente / dapp.

Si tu situación, por ejemplo, es un backend en Python o Java donde te gustaría verificar firmas de una Sequence Wallet, entonces puedes llamar a la función estándar [EIP1271](https://eips.ethereum.org/EIPS/eip-1271) para la dirección de la cuenta desde tu backend.

Si deseas utilizar las capacidades de Meta-Transacción de Sequence, consulta [Construyendo un Servidor de Retransmisión con Sequence](/guides/building-relaying-server).
