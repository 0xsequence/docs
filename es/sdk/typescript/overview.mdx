### Soporte para otros lenguajes de backend
Si sus servicios backend están escritos en un lenguaje distinto a JS o Go, igual puede integrar Sequence fácilmente, ya que Sequence es básicamente una librería estándar de cliente Ethereum con algunas funciones extra. En muchos casos, estas funciones adicionales se aprovechan mejor en el lado del cliente o la dapp.

Por ejemplo, si su backend es en Python o Java y desea verificar firmas de un Sequence Wallet, puede llamar a la función estándar [EIP1271](https://eips.ethereum.org/EIPS/eip-1271) para la dirección de cuenta desde su backend.

Si desea usar las capacidades de Meta-Transacciones de Sequence, consulte [Building Relaying Server with Sequence](/guides/building-relaying-server).
