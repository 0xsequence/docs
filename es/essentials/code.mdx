## Básico

### Código en línea
Para indicar que una `palabra` o `frase` es código, encierre el texto entre acentos graves (\`).

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

### Bloque de código
Use [bloques de código delimitados](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) encerrando el código entre tres acentos graves y, después de los acentos iniciales, escriba el lenguaje de programación de su fragmento para obtener resaltado de sintaxis. Opcionalmente, también puede escribir el nombre de su código después del lenguaje de programación.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
