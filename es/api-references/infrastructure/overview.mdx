La API Sequence Infrastructure contiene una variedad de servicios que potencian la infraestructura de Sequence y que puede aprovechar en su propia aplicación. Esto incluye Abstracción de Moneda, endpoints de Criptografía, vinculación de wallet y más.

Consulte nuestras Referencias de API [aquí](/api-references/infrastructure/endpoints) para comenzar.
