La API de Infraestructura de Sequence contiene una multitud de varios servicios que impulsan la infraestructura de Sequence que puedes aprovechar en tu propia aplicación. Esto incluye Abstracción de Moneda, endpoints de Criptografía, vinculación de billeteras y más.

Consulta nuestras Referencias de API [aquí](/api-references/infrastructure/endpoints) para comenzar.
