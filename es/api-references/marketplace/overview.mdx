El servicio de API del Sequence Marketplace ofrece una forma simple y rápida de interactuar con los protocolos del marketplace de Sequence. La API está diseñada para ser fácil de usar y tener todas las funciones que necesitas para construir un marketplace completamente funcional. Revisa nuestros diversos [ejemplos](/api-references/marketplace/examples/orderbook-transactions) o nuestra [guía de marketplace personalizado](/guides/custom-marketplace) para un recorrido de principio a fin.
