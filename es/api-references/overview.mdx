## Autenticación

La mayoría de las APIs de Sequence requieren autenticación usando:

* **Project Access Key**: Para solicitudes del lado del cliente, estas están listadas como`public`.
* **Service Token**: Para solicitudes del lado del servidor que requieren permisos elevados, estos requieren pasar un token bearer secreto y están listados como`secret`.

Puedes generar estas credenciales en el panel de[Sequence Builder](https://sequence.build).

## Límites de Tasa

Los límites de tasa de la API varían según el nivel y los MAUs disponibles. Verifica la configuración de tu proyecto en el panel de Sequence Builder para conocer tus límites específicos.
