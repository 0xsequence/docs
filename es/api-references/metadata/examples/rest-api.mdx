El servicio de API de Sequence Metadata ofrece una API simple y rápida para consultar metadatos de tokens y NFTs para cadenas compatibles con Ethereum.

En pocas palabras, la API de Metadata te permite consultar los metadatos de tokens de cualquier contrato ERC20, ERC721 o ERC1155 en una[cantidad de cadenas Ethereum soportadas](https://status.sequence.info).

Además, el servicio de Sequence Metadata está automáticamente integrado en el[Sequence Indexer](/api-references/indexer/overview).
Pero como demostramos a continuación, también es útil poder consultar los metadatos del token/contrato directamente :)

<Note>
  \[CONSEJO PRO: Endpoints RPC vs REST]

  El servicio Sequence Metadata está disponible tanto con endpoints RPC como REST. Recomendamos los endpoints RPC para tu aplicación, pero los endpoints REST están disponibles como opción por conveniencia.

  Para el endpoint RPC ver[Token metadata RPC](/api-references/metadata/examples/token-metadata)y[Contract metadata RPC](/api-references/metadata/examples/contract-metadata).
</Note>

## Ejemplo

Digamos que te gustaría consultar los metadatos de una carta Skyweaver 1155 en la red Polygon. Por supuesto, el ejemplo de abajo funcionará para cualquier combinación de red / contrato / token también.

Primero, el contrato de activos Skyweaver 1155 está ubicado en la dirección`0x631998e91476DA5B870D741192fc5Cbc55F5a52E`
en la red Polygon. Segundo, vamos a construir el endpoint de búsqueda de metadatos para consultar el contrato de Skyweaver para el ID de token`20`:

[Solicitud](https://metadata.sequence.app/tokens/polygon/0x631998e91476DA5B870D741192fc5Cbc55F5a52E/20)

**Respuesta:**

Si deseas consultar varios tokens al mismo tiempo, puedes incluir más IDs de token separados por comas[https://metadata.sequence.app/tokens/polygon/0x.../20,21,22](https://metadata.sequence.app/tokens/polygon/0x631998e91476DA5B870D741192fc5Cbc55F5a52E/20,21,22)-- haz clic para ver la respuesta JSON para consultar los IDs de token`20`,`21`, y`22`en una sola solicitud por lotes.

Siéntete libre de modificar la URL de metadata.sequence.app anterior para tu propio contrato, o otro proyecto popular para ver cómo regresan las respuestas. Puedes cambiar la red a[una de nuestras redes soportadas](/)
y especificar cualquier contrato y/o ID de token.

## Uso

El servicio de API de Metadata es accesible ya sea con una simple[interfaz REST](#rest-endpoints), o un[cliente RPC](#rpc-client).
Para el cliente RPC, ofrecemos clientes para navegador web, node y Go. Recomendamos el cliente RPC para la mayoría de las integraciones, pero hacemos disponible la interfaz REST por conveniencia.

## Endpoints REST

Como se mencionó en el[ejemplo](#example)anterior. El formato general del endpoint REST es:

```
GET https://metadata.sequence.app/tokens/<network>/<contractAddress>[/<tokenID>]
```

Donde`<network>`debe ser una de las`Chain ID`o`Chain Handle`de[las redes soportadas](/).

**Obtener metadatos a nivel de contrato:**

```
GET https://metadata.sequence.app/tokens/<network>/<contractAddress>
```

[Ejemplo](https://metadata.sequence.app/tokens/polygon/0x631998e91476DA5B870D741192fc5Cbc55F5a52E)

**Obtener metadatos a nivel de token:**

```
GET https://metadata.sequence.app/tokens/<network>/<contractAddress>/<tokenID>[,<tokenID>,...]
```

Ejemplos:

* [ID de token 20](https://metadata.sequence.app/tokens/polygon/0x631998e91476DA5B870D741192fc5Cbc55F5a52E/20)
* [ID de token 20 y 21](https://metadata.sequence.app/tokens/polygon/0x631998e91476DA5B870D741192fc5Cbc55F5a52E/20,21)

## Cliente RPC

La interfaz RPC de Metadata ofrece todas las capacidades del servicio de Metadata.

Proporcionamos SDKs para[Web / node.js](https://github.com/0xsequence/sequence.js)y[Go](https://github.com/0xsequence/go-sequence).
O si deseas integrar el servicio de Metadata con otro lenguaje objetivo, simplemente sigue la referencia de la API a continuación
para implementar las solicitudes HTTP. Además, lee el código fuente del cliente Typescript como [implementación de referencia del cliente RPC de Metadata](https://github.com/0xsequence/sequence.js/blob/master/packages/metadata/src/metadata.gen.ts) también.

## Métodos RPC de Metadata

**Punto final de la API de Metadata:** [https://metadata.sequence.app](https://metadata.sequence.app)

**Métodos RPC de Metadata:**

* `GetTokenMetadata` - obtener metadata de tokens ERC721 o ERC1155 de un solo contrato
* `GetTokenMetadataBatch` - obtener metadata de tokens ERC721 o ERC1155 de un lote de contratos
* `GetContractInfo` - obtener metadata de contrato de una dirección de contrato ERC20, ERC721 o ERC1155
* `GetContractInfoBatch` - obtener metadata de contrato de un lote de direcciones de contratos ERC20, ERC721 o ERC1155
* `SearchContracts` - obtener información de metadata de contrato basada en criterios de búsqueda para tokens ERC20, ERC721 y ERC1155
