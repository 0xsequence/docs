El Sequence Indexer es un servicio web3 modular que facilita la consulta de datos de tokens y NFT de blockchain desde redes compatibles con Ethereum. El Indexer indexa automáticamente cada token ERC20, ERC721 y ERC1155 de las cadenas compatibles con Ethereum.

El Sequence Indexer es una herramienta poderosa que puede usarse para construir billeteras, juegos y dapps que necesitan consultar datos en cadena. Está diseñado para ser rápido, confiable y fácil de usar. Aprovecha el [Sequence Node Gateway](/api-references/node-gateway) que monitorea la salud de los nodos, enruta inteligentemente las solicitudes a los nodos disponibles y almacena en caché las respuestas de solicitudes más recientes. Esto significa que cuando los proveedores de nodos fallan ya no es un problema, ya que el Node Gateway automáticamente enrutará a un proveedor saludable asegurando que el Indexer siempre se mantenga al día con la cadena sin tiempo de inactividad.

Otro beneficio de usar el Indexer es que está utilizando bases de datos de próxima generación basadas en árboles LSM desarrolladas internamente para almacenar los datos. Esto lo hace super rápido para consultar e indexar datos. Los datos están disponibles en tiempo real con baja latencia. Puedes consultarlo usando una simple API REST o uno de nuestros SDKs.

La API del Indexer te permite consultar todos los saldos de tokens, historial y otra información para todas las cadenas compatibles con Ethereum. Para hacerlo un lugar único para todas tus necesidades de datos de tokens, hemos incluido soporte de [Sequence Metadata](/api-references/metadata/overview) para todos los tokens. Esto significa que puedes renderizar fácilmente tokens en tus aplicaciones, juegos o billeteras sin necesidad de obtener metadatos de una API separada.

De hecho, la Sequence Wallet usa el Indexer internamente para poder renderizar sin problemas toda la información de tokens en cualquier billetera. Pero por supuesto, el Indexer es una pieza modular de infraestructura, y puedes usarlo directamente desde tu dapp, juego, o incluso desde un servidor.

**Características:**

* API súper rápida para consultar todos los saldos de tokens, historial, metadatos y NFTs con soporte multi-cadena
* Indexación en tiempo real de transacciones ERC20, ERC721 y ERC1155 en cadenas compatibles con EVM
* Detecta automáticamente todos los tokens en la cadena, sin necesidad de un registro de contratos
* Resistente a fallos de nodos y reorganizaciones de la cadena
* Escucha fácilmente eventos específicos y transacciones en la cadena con precisión mediante una API simple
* Soporte integrado de metadatos de tokens / NFT para renderizar fácilmente tokens en tus aplicaciones / juegos
* Alto tiempo de actividad y disponibilidad
