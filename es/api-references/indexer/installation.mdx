Sequence Indexer es una API simple para consultar cualquier dato de token y NFT de blockchain. A continuación encontrarás instrucciones sobre cómo integrar la API de Sequence Indexer en tus Aplicaciones Web, Juegos y backends. En caso de que te lo hayas perdido, por favor también consulta la [Descripción General del Indexador](/api-references/indexer/overview).

## Instalación

El Sequence Indexer está construido como una API HTTP con puntos finales RPC que puedes llamar directamente desde tu Aplicación Web, Juego o backend del servidor. A continuación encontrarás información sobre el esquema del punto final RPC con ejemplos de comandos curl, junto con ejemplos tanto en Javascript/Typescript como en Go.

Proporcionamos SDKs para [Web / node.js](https://github.com/0xsequence/sequence.js) y [Go](https://github.com/0xsequence/go-sequence). O si deseas integrar el Indexador con otro lenguaje objetivo, simplemente sigue la referencia de la API a continuación para implementar las solicitudes HTTP. Además, lee el código fuente del cliente Typescript como [implementación de referencia del cliente de la API del Indexador](https://github.com/0xsequence/sequence.js/blob/master/packages/indexer/src/indexer.gen.ts) también.

### Instalación Web / node.js

<CodeGroup>
  ```bash npm
  npm install 0xsequence ethers
  ```

  o

  ```bash pnpm
  pnpm install 0xsequence ethers
  ```

  o

  ```bash yarn
  yarn add 0xsequence ethers
  ```
</CodeGroup>

<Note>
  Este código requiere una Clave de Acceso API de [Sequence Builder](https://sequence.build).
</Note>

```ts
import { SequenceIndexer } from '@0xsequence/indexer'

// see https://docs.sequence.xyzhttps://status.sequence.info for list of
// indexer hosts for the chain you'd like to query
const indexer = new SequenceIndexer('https://mainnet-indexer.sequence.app', 'AQAAAAAAAF_JvPALhBthL7VGn6jV0YDqaFY')

// see examples below for the kinds of queries you can make
const tokenBalances = await indexer.getTokenBalances(...)
```

**NOTA:** si estás usando `@0xsequence/indexer` desde node.js, recomendamos usar node v18.x o más reciente.

<br />

### Instalación de Go

```bash Terminal
go get -u github.com/0xsequence/go-sequence@latest
```

luego en tu aplicación,

```go Go
import (
	"github.com/0xsequence/go-sequence/indexer"
)

// see https://docs.sequence.xyzhttps://status.sequence.info for list of
// indexer hosts for the chain you'd like to query
seqIndexer := indexer.NewIndexer("https://polygon-indexer.sequence.app", "AQAAAAAAAF_JvPALhBthL7VGn6jV0YDqaFY")

// see examples below for the kinds of queries you can make
accountAddress := "ACCOUNT_ADDRESS"
includeMetadata := true
metadataOptions := indexer.MetadataOptions{
	VerifiedOnly: true,
}

_, tokenBalances, err := seqIndexer.GetTokenBalances(context.Background(), &accountAddress, nil, nil, &includeMetadata, &metadataOptions, nil, nil)
```

### Instalación de Unity o Unreal

El Sequence Indexer está integrado directamente dentro de los respectivos [Sequence Unity](/sdk/unity/) y [Sequence Unreal](/sdk/unreal/introduction) SDKs.
