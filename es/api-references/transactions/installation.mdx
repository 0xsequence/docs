Proporcionamos SDKs para [Web / node.js](https://github.com/0xsequence/sequence.js) y [Go](https://github.com/0xsequence/go-sequence). O si prefieres integrar el Relayer con otro lenguaje objetivo, simplemente sigue la referencia de API a continuación para implementar las solicitudes HTTP. Además, lee el código fuente del cliente Typescript como [implementación de referencia del cliente de la API de Transactions](https://github.com/0xsequence/sequence.js/blob/master/packages/relayer/src/rpc-relayer/relayer.gen.ts) también.

### Instalación Web / node.js

```sh
npm install 0xsequence ethers
```

o

```sh
pnpm install 0xsequence ethers
```

o

```sh
yarn add 0xsequence ethers
```

### Instalación Go

```bash Terminal
go get -u github.com/0xsequence/go-sequence@latest
```

luego en tu aplicación,

```go Go
import (
	"github.com/0xsequence/go-sequence/relayer"
)
```
