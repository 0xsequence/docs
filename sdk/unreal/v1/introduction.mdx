The Sequence Unreal Embedded Wallet SDK provides full Sequence [Embedded Wallet](/solutions/wallets/embedded-wallet/overview) and [Indexer](/api/indexer/overview) integration
for games built on the Unreal Framework. We provide full Blueprint support for a seamless integration experience.

- Android
- iOS
- PC Standalone
- Mac Standalone

## Supported Login Methods

Start the integration using our [Session Subsystem for Blueprints](/sdk/unreal/v1/authentication).

- Email with One-Time-Password
- Google- & Apple Single Sign On
- PlayFab Login & Registration
- Guest Sign In
