The Sequence Unreal Embedded Wallet SDK provides full Sequence [Embedded Wallet](/solutions/wallets/embedded-wallet/overview) and [Indexer](/api-references/indexer/overview) integration for games built on the Unreal Framework. We provide full Blueprint support for a seamless integration experience.

Get started with our [Quickstart Guide.](/sdk/unreal/quickstart)

## Supported Platforms

Please view our [Platform Requirements](/sdk/unreal/platforms)

- Unreal Engine 5.2 and later
- Android
- iOS
- PC Standalone
- Mac Standalone

## Supported Login Methods

Start the integration using our [Session Subsystem for Blueprints](/sdk/unreal/authentication).

- Email with One-Time-Password
- Google- & Apple Single Sign On
- PlayFab Login & Registration
- Guest Sign In
