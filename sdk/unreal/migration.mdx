### Config file

- Add a DefaultChain variable to your SequenceConfig.ini file.

### Sign-In UI

- Remove references from the old GameMode and Pawn in your project.
- Use the [Sign In Boilerplate](/sdk/unreal/bootstrap_game)

### Blueprint Nodes

- Remove references to your any event listener node.
- Refresh your Sequence Blueprint nodes to show delegates.
- etc...
