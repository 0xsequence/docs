Create a `SequenceConnect` object and specify the Chain and Ecosystem you want to let users connect to.

```csharp
EcosystemType ecosystem = EcosystemType.Sequence;
SequenceConnect connect = new SequenceConnect(ecosystem);
```

### Try our Built-In Demo

Import the `Ecosystem Wallet Demo` sample from Unity's Package Manager. Additionally, checkout our boilerplate
script `SequenceEcosystemWalletWindow` for integration details.
