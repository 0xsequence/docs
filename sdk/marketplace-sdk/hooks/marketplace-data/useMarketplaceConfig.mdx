---
title: useMarketplaceConfig
description: The useMarketplaceConfig hook retrieves your Marketplace's configuration, including collections, social links, title, and other settings from Builder.
sidebarTitle: useMarketplaceConfig
---

```ts
import { useMarketplaceConfig } from "@0xsequence/marketplace-sdk/react";

## Into your React component:

const data = useMarketplaceConfig();
```

    <ParamField path="collections" type="MarketplaceCollection[]">

    <ParamField path="walletOptions" type="MarketplaceWalletOptions">
