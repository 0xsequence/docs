# Getting Started

Let's create a simple React application that can display and interact with NFTs. We'll set up wallet connection and basic NFT viewing functionality.

<Steps>
  <Step title="Create a Project in Sequence Builder">
    Before you begin coding, you'll need to set up your project:
    1. Go to [Sequence Builder](https://sequence.build)
    2. Create a new project
    3. Follow our [White-label Marketplace Guide](https://docs.sequence.xyz/solutions/marketplaces/white-label-marketplace/overview)
    4. Note down your Project ID and API Access Key
  </Step>

  <Step title="Create a New React Project">
    We'll use Vite with React and TypeScript for this guide. You can use any package manager, but we'll use pnpm in our examples:

    ```bash
    # Create a new project
    pnpm create vite@latest my-marketplace-app -- --template react-ts

    # Navigate to the project directory
    cd my-marketplace-app

    # Install dependencies
    pnpm install
    ```
  </Step>

  <Step title="Install Required Dependencies">
    Install the Marketplace SDK and its peer dependencies:

    ```bash
    pnpm add @0xsequence/marketplace-sdk @0xsequence/connect wagmi viem @tanstack/react-query
    ```
  </Step>

  <Step title="Set Up Providers">
    Create a new file `src/providers.tsx` to configure the necessary providers:

    ```tsx
    import {
      MarketplaceProvider,
      MarketplaceQueryClientProvider,
      ModalProvider,
      createWagmiConfig,
      getQueryClient,
      marketplaceConfigOptions,
    } from "@0xsequence/marketplace-sdk/react";
    import { useQuery } from "@tanstack/react-query";
    import { WagmiProvider } from "wagmi";
    import {
      SequenceConnectProvider,
      type ConnectConfig,
    } from "@0xsequence/connect";
    import type { SdkConfig } from "@0xsequence/marketplace-sdk";

    interface ProvidersProps {
      children: React.ReactNode;
    }

    export default function Providers({ children }: ProvidersProps) {
      const sdkConfig = {
        projectId: "YOUR_PROJECT_ID",
        projectAccessKey: "YOUR_PROJECT_ACCESS_KEY",
        shadowDom: true,
      } satisfies SdkConfig;

      const queryClient = getQueryClient();
      const { data: marketplaceConfig, isLoading } = useQuery(
        marketplaceConfigOptions(sdkConfig),
        queryClient
      );

      if (isLoading) {
        return <div>Loading configuration...</div>;
      }

      if (!marketplaceConfig) {
        return <div>Failed to load marketplace configuration</div>;
      }

      const connectConfig: ConnectConfig = {
        projectAccessKey: sdkConfig.projectAccessKey,
        signIn: {
          projectName: marketplaceConfig.settings.title,
        },
      };

      const wagmiConfig = createWagmiConfig(marketplaceConfig, sdkConfig);

      return (
        <WagmiProvider config={wagmiConfig}>
          <MarketplaceQueryClientProvider>
            <SequenceConnectProvider config={connectConfig}>
              <MarketplaceProvider config={sdkConfig}>
                {children}
                <ModalProvider />
              </MarketplaceProvider>
            </SequenceConnectProvider>
          </MarketplaceQueryClientProvider>
        </WagmiProvider>
      );
    }
    ```

    <Check>
      Make sure to replace `YOUR_PROJECT_ID` and `YOUR_PROJECT_ACCESS_KEY` with your actual project credentials. You can find your Project ID at `https://sequence.build/project/{YOUR_PROJECT_ID}/overview` and your Project Access Key at `https://sequence.build/project/{YOUR_PROJECT_ID}/settings/apikeys` in the [Sequence Builder](https://sequence.build) dashboard.
    </Check>

    <Note>
      The `shadowDom: true` setting in the SDK configuration is required when not using Tailwind CSS. This ensures that the styles are properly applied to the components.
    </Note>
  </Step>

  <Step title="Update Main Entry Point">
    Update your `main.tsx` to use the Providers component:

    ```tsx
    import React from 'react'
    import ReactDOM from 'react-dom/client'
    import App from './App'
    import Providers from './providers'
    import './index.css'

    ReactDOM.createRoot(document.getElementById('root')!).render(
      <React.StrictMode>
        <Providers>
          <App />
        </Providers>
      </React.StrictMode>
    )
    ```
  </Step>

  <Step title="Create Connect Button Component">
    Create a new component `src/components/ConnectButton.tsx` for wallet connection:

    ```tsx
    import { useOpenConnectModal } from '@0xsequence/connect';
    import { useAccount } from 'wagmi';

    export default function ConnectButton() {
      const { setOpenConnectModal } = useOpenConnectModal();
      const { address } = useAccount();

      return (
        <button 
          onClick={() => setOpenConnectModal(true)}
        >
          {address ? `${address.slice(0, 6)}...${address.slice(-4)}` : "Connect Wallet"}
        </button>
      );
    }
    ```
  </Step>

  <Step title="Create NFT Card Component">
    Create a new component `src/components/NFTCard.tsx` to display and interact with NFTs:

    ```tsx
    import {
      useCollectible,
      useMakeOfferModal,
    } from "@0xsequence/marketplace-sdk/react";
    import type { Address } from "viem";

    const collectionAddress = "YOUR_COLLECTION_ADDRESS" as Address;
    const collectibleId = "1" as string;
    const chainId = 80002 as number; // Polygon Amoy Testnet

    export default function NFTCard() {
      const { data: nft } = useCollectible({
        chainId,
        collectionAddress,
        collectibleId,
      });

      const { show } = useMakeOfferModal();

      if (!nft) return null;

      return (
        <div>
          <img src={nft.image} alt={nft.name} />
          <h3>{nft.name}</h3>
          <button
            onClick={() => {
              show({
                chainId,
                collectionAddress,
                collectibleId,
              });
            }}
          >
            Make Offer
          </button>
        </div>
      );
    }
    ```

    <Note>
      To get your collection address and chain ID:
      1. Go to your marketplace at `https://sequence.build/project/{YOUR_PROJECT_ID}/marketplace/market` and copy the collection address you created.
      2. For the chain ID:
         - If you know the network where your collection is deployed, use that chain ID
         - If unsure, visit [Sequence Chainlist](https://sequence.xyz/chains) and find the chain ID where your collection is deployed (look for networks with "Supported" status)
      3. Make sure you have minted at least one collectible in your collection, then use that collectible's ID as the `collectibleId`
    </Note>

    <Tip>
      While Sequence Connect is recommended for the best experience, you can use other wallet providers with the Marketplace SDK. For custom wallet connectors, see our guide on [custom connectors](/sdk/web/custom-connectors).
    </Tip>
  </Step>

  <Step title="Update App Component">
    Update your `App.tsx` to use the new components:

    ```tsx
    import ConnectButton from './components/ConnectButton'
    import NFTCard from './components/NFTCard'

    export default function App() {
      return (
        <div>
          <ConnectButton />
          <NFTCard />
        </div>
      )
    }
    ```
  </Step>

  <Step title="Run Your Application">
    Start your development server:

    ```bash
    pnpm run dev
    ```

    Your marketplace application will be running at `http://localhost:5173`. You can now:
    1. Connect your wallet using the connect button
    2. View NFT details
    3. Make offers on NFTs using the Make Offer button
  </Step>
</Steps>
