# Quickstart
:::info
Looking for [Unity](/sdk/unity/overview), [Unreal](/sdk/unreal/overview) or [Mobile](/sdk/mobile)? We have you covered with our dedicated SDKs.
:::

You can quickly get started with our React boilerplate by running the following from your terminal:

::::steps
### Prerequisites

Ensure you have the pnpm package manager installed on your machine. If you don't have, please follow the instructions for your OS [here](https://pnpm.io/installation).

### Clone Starter Repo
Run the following from your terminal to clone the starter repo.

```bash [React]
git clone https://github.com/0xsequence/embedded-wallet-react-boilerplate.git
```


### Install Packages and Start

```bash
cd embedded-wallet-react-boilerplate && pnpm install && pnpm dev
```

### Connect and Authenticate
Navigate to `localhost:4444` and click connect to try out the authentication.


### Use Sequence Embedded Wallets!

Now that you have a working authentication, you can take look at our examples for the most common functions:
- [Sign messages](/solutions/wallets/embedded-wallet/examples/use-wallets#sign-messages)
- [Send transactions](/solutions/wallets/embedded-wallet/examples/use-wallets#send-transactions)
- [Verify a user on your backend](/solutions/wallets/embedded-wallet/examples/verification#prerequisite-deployment-of-a-parent-wallet)
::::


:::tip
We are using example variables in the .env file, so we recommend creating a project on Sequence Builder and [setting up your own wallet configuration](https://sequence.build/project/default/wallet/embedded-wallet) and update the environment variables to control your social providers, customize email templates, and more.
:::
