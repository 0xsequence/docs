# Getting Started with Sequence Kit

In this guide, we will walk you through installing Sequence Kit, triggering the connection modal, and displaying the Inventory in your application.

## Installing Sequence Kit Packages

Sequence Kit is modular, allowing you to install only the necessary packages. To get started, install the `kit` core package, the `connectors` package for interfacing with various wallet providers. Additionally, install other dependencies such as wagmi, viem, and 0xsequence.

```bash
npm install @0xsequence/kit @0xsequence/kit-connectors wagmi ethers@5.7.2 viem 0xsequence @tanstack/react-query
# or
pnpm install @0xsequence/kit @0xsequence/kit-connectors wagmi ethers@5.7.2 viem 0xsequence @tanstack/react-query
# or
yarn add @0xsequence/kit @0xsequence/kit-connectors wagmi ethers@5.7.2 viem 0xsequence @tanstack/react-query
```

## Integrate a Project Access Key

Using the following variable in your code, direct yourself to [sequence.build](https://sequence.build), create a project, and claim a Public Project Access Key, using [this walkthrough](/solutions/builder/getting-started).

```
const projectAccessKey = '<access-key>'
```

## Create a Query Client

Using `QueryClient` which is a dependency of `wagmi` that makes fetching, caching, synchronizing and updating server state in your web applications a easy.

```
import { QueryClient } from '@tanstack/react-query'
...
const queryClient = new QueryClient() 
```

## Create Wagmi Chains Array

Import the required networks you're using for your application, to later be used in creating your Wagmi config.

```
import { mainnet, polygon, Chain } from 'wagmi/chains'
import { http } from 'wagmi'
...
const chains = [mainnet, polygon] as [Chain, ...Chain[]]

const transports: any = {}

chains.forEach(chain => {
  transports[chain.id] = http()
})
```

## Create Default Connector
Import the `getDefaultConnectors` function from the `@0xsequence/kit-connectors` package, and include a Wallet Connect ID obtained from [here](https://cloud.walletconnect.com/app), a default chain ID, app name, and `projectAccessKey`.

```
import { getDefaultConnectors } from '@0xsequence/kit-connectors'
...
const connectors = getDefaultConnectors({
  walletConnectProjectId: 'wallet-connect-id',
  defaultChainId: 137,
  appName: 'demo app',
  projectAccessKey
})
```

## Setup Wrapper Elements

```
import MyPage from './components/MyPage'
import { WagmiConfig } from 'wagmi'
import { QueryClientProvider } from '@tanstack/react-query'
import { KitProvider } from '@0xsequence/kit'
import { projectAccessKey, config, queryClient } from './config'

function App() {
  return (
    <WagmiConfig config={config}>
      <QueryClientProvider client={queryClient}> 
        <KitProvider config={{projectAccessKey: projectAccessKey}}>
          <MyPage />
        </KitProvider>
      </QueryClientProvider>
    </WagmiConfig>
  );
}
```

## Full Code

### config.ts

```jsx
import { QueryClient } from '@tanstack/react-query'
import { mainnet, polygon, Chain } from 'wagmi/chains'
import { getDefaultConnectors } from '@0xsequence/kit-connectors'
import { createConfig, http } from 'wagmi'

const queryClient = new QueryClient() 

const chains = [mainnet, polygon] as [Chain, ...Chain[]]

const transports: any = {}

chains.forEach(chain => {
  transports[chain.id] = http()
})
  
const projectAccessKey = '<access-key>'

const connectors = getDefaultConnectors({
  walletConnectProjectId: 'wallet-connect-id',
  defaultChainId: 137,
  appName: 'demo app',
  projectAccessKey
})

//@ts-ignore
const config = createConfig({
  transports,
  connectors,
  chains
})

export {
  projectAccessKey,
  config,
  queryClient
}
```

### main.tsx

```jsx
import MyPage from './components/MyPage'
import { WagmiConfig } from 'wagmi'
import { QueryClientProvider } from '@tanstack/react-query'
import { KitProvider } from '@0xsequence/kit'
import { projectAccessKey, config, queryClient } from './config'

function App() {
  return (
    <WagmiConfig config={config}>
      <QueryClientProvider client={queryClient}> 
        <KitProvider config={{defaultTheme: 'light', projectAccessKey: projectAccessKey}}>
          <MyPage />
        </KitProvider>
      </QueryClientProvider>
    </WagmiConfig>
  );
}
```
Please note: it is best to have the provider configuration variables in a file imported into your `main.tsx`, and to definitely not include the provider configurations in any React components, as this will break the app.

# Setting Up Inventory
To install the optional Inventory feature, add the `KitWalletProvider` below the `KitProvider` wrapper.

```jsx
import { KitWalletProvider } from '@0xsequence/kit-wallet'

// ...

const App = () => {
  return (
    <WagmiConfig config={config}>
      <QueryClientProvider client={queryClient}> 
        <KitProvider config={kitConfig}>
          <KitWalletProvider> // <-- Added code
            <Page />
          </KitWalletProvider>
        </KitProvider>
      </QueryClientProvider>
    </WagmiConfig>
  )
}
```

# Toggling the Connect Modal

<div class="text--center">
  <img src="/img/kit/sign-in-modal.png" alt="Toggle the Connect Modal in Sequence Kit"/>
</div>

Invoke the connect modal using the `useOpenConnectModal` hook.

```jsx
import { useOpenConnectModal } from '@0xsequence/kit'
import { useDisconnect, useAccount } from 'wagmi'


const MyReactComponent = () => {
  const { setOpenConnectModal } = useOpenConnectModal()

  const { isConnected } = useAccount()

  const onClick = () => {
    setOpenConnectModal(true)
  }

  return (
    <>
      {!isConnected && (
        <button onClick={onClick}>
          Sign in
        </button>
      )}
    </>
  )
}
```

The modal will automatically close once the user signs in. You can utilize the `useAccount` hook from wagmi to detect the user's connection status.

# Invoking the Inventory modal

<div class="text--center">
  <img src="/img/kit/inventory-modal.png" alt="Invoke the Inventory Modal from Sequence Kit to display collectibles in the user's wallet" />
</div>

Open the Inventory using the `useOpenWalletModal` react hook.

```jsx
import { useOpenWalletModal } from '@0xsequence/kit-wallet'

const MyComponent = () => {
  const { setOpenWalletModal } = useOpenWalletModal()

  const onClick = () => {
    setOpenWalletModal(true)
  }

  return (
    <button onClick={onClick}>open inventory</button>
  )
}
```

Users can then select on the collectible in order to send to another address.
