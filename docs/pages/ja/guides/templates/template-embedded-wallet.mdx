## MOCKDATA + ja-translated Getting Started

### ja-translated SDK Installation
MOCKDATA + ja-translated We provide TypeScript and Unity SDKs for the Embedded Wallet authentication system. You can install the TypeScript SDK with:

```bash
pnpm install @0xsequence/waas
```

ja-translated For more information on the Unity Embedded Wallet SDK, please refer to the [Unity SDK documentation](/sdk/unity/overview)

### MOCKDATA + ja-translated Project Setup
MOCKDATA + ja-translated Sequence Embedded Wallet is currently only available as a closed beta. To start using the Embedded Wallet SDKs, you'll need to contact the Sequence team and acquire API credentials. Once this is done, you will be provided with two keys:
- ja-translated Project access key
- MOCKDATA + ja-translated Embedded Wallet configuration key

### ja-translated Library Setup
MOCKDATA + ja-translated To start using Sequence Embedded Wallet SDK, you'll need to create a new instance of the `waas` class:

```typescript
import { SequenceWaaS } from '@0xsequence/waas'

const waas = new SequenceWaaS({
  projectAccessKey: `${process.env.PROJECT_ACCESS_KEY}`,
  waasConfigKey: `${process.env.WAAS_CONFIG_KEY}`,
  network: 'mumbai'
}, defaults.TEST)
```

ja-translated Note that the library is operational, but it can't be used to interact with any wallet until you have authenticated **as a user**.
