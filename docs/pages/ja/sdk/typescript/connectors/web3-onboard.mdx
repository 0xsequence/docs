## MOCKDATA + ja-translated Installation
MOCKDATA + ja-translated The Sequence module is available through npm:

```shell
npm install @web3-onboard/sequence 0xsequence ethers
```

ja-translated or

```shell
yarn add @web3-onboard/sequence 0xsequence ethers
```

## ja-translated Usage

```js
import sequenceModule from '@web3-onboard/sequence'

const sequenceOptions = {
  network: 137,
  appName: 'My app'
}

const sequenceOnboard = sequenceModule({})

const onboard = Onboard({
  wallets: [
    sequenceOnboard,
    ...restWallets
  ],
  ...restofOnboardParams
})

```

## MOCKDATA + ja-translated Parameters

### ja-translated network (optional)
MOCKDATA + ja-translated The default network that the Sequence wallet will attempt to connect to.

### ja-translated appName (optional)
ja-translated The name of the app which will show up in Sequence's prompt upon an attempt to connect.
