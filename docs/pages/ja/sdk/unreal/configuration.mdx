# MOCKDATA + ja-translated Configuration
ja-translated ::::steps

## ja-translated Private Dependency Modules
MOCKDATA + ja-translated In your Project's Build.cs file ([where to find](https://forums.unrealengine.com/t/where-are-build-cs-and-target-cs/302665)) please check the private Dependency module such that it includes "SequencePlugin":

```cpp
    public SequenceTest(ReadOnlyTargetRules Target) : base(Target)
    {
        PCHUsage = PCHUsageMode.UseExplicitOrSharedPCHs;
		PublicDependencyModuleNames.AddRange(new string[] { "Core", "CoreUObject", "Engine", "InputCore" });
		PrivateDependencyModuleNames.AddRange(new string[] { "SequencePlugin" });
	}
```

## ja-translated Create a config file
MOCKDATA + ja-translated Navigate to `[YourProjectDirectory]/Config`, create a file named `SequenceConfig.ini`, and add the following lines:

```
[/Script/Sequence.Config]

FallbackEncryptionKey = "" 
WaaSConfigKey = "" 
ProjectAccessKey = "" 
GoogleClientID = "" 
AppleClientID = "" 
FacebookClientID = "" 
DiscordClientID = ""
PlayFabTitleID = ""
RedirectUrl = "https://api.sequence.app"
```

## ja-translated Add your project keys
MOCKDATA + ja-translated Fill in `[YourProjectDirectory]/Config/SequenceConfig.ini` with the appropriate values for your project.

ja-translated `WaaS Config Key` - You can get this key in [Sequence Builder](https://sequence.build/) under **Onboard > Embedded Wallet**

MOCKDATA + ja-translated `Project Access Key` - You can get this key in [Sequence Builder](https://sequence.build/) under **Settings > API Keys**

ja-translated `Fallback Encryption Key` - Create a 32-character long alphanumeric encryption key.

MOCKDATA + ja-translated ::::

MOCKDATA + ja-translated :::tip
Make sure to get the WaaS keys from **Production Mode**.
:::

ja-translated :::warning
Don't forget to [configure your embedded wallet settings in the Builder](/solutions/builder/embedded-wallet/index)!
:::

## MOCKDATA + ja-translated Security
ja-translated In order to prevent tampering with data you must encrypt your packaged project using Unreals packaging settings. You can refer to [these docs](https://dev.epicgames.com/documentation/en-us/unreal-engine/packaging-unreal-engine-projects?application_version=5.3) for more information.

MOCKDATA + ja-translated :::tip
You can use a password manager (or any other secure randomness source) to generate this encryption key
:::

ja-translated :::warning
Only alphanumeric characters are accepted!
:::
