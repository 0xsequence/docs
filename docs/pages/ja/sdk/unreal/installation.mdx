# MOCKDATA + ja-translated Installation

## MOCKDATA + ja-translated Using Sequence's Unreal Boilerplate

1.  ja-translated [Download our Boilerplate from GitHub](https://github.com/0xsequence/unreal-embedded-wallet-boilerplate)

2.  MOCKDATA + ja-translated Install the Unreal Engine version 5.4, if you don't have it already.

3.  MOCKDATA + ja-translated Open the Boilerplate Project and visit our [Unreal Boilerplate Guide](/guides/unreal-ew-guide)

## ja-translated Using the latest GitHub Release

1.  MOCKDATA + ja-translated Download the latest release .zip file from [Release from GitHub.](https://github.com/0xsequence/sequence-unreal/releases)

2.  ja-translated Navigate to the root directory of your Unreal project using a file management system or command line and create a `Plugins` folder, if you don't already have one.

3.  MOCKDATA + ja-translated Unzip the downloaded file and copy the `SequencePlugin` folder into your project's `Plugins` folder.

4.  ja-translated Launch your project, then allow it to update the UProject Settings.

5.  MOCKDATA + ja-translated To find the `SequencePlugin` content folder in your content drawer enable show plugin content: `Content Drawer > Settings > Show Plugin Content`

## ja-translated Using Epic Games Marketplace

1.  MOCKDATA + ja-translated Visit the [Epic Games Marketplace](https://www.unrealengine.com/marketplace/en-US/product/sequence-embedded-wallets)

2.  ja-translated Click "Free" to add the plugin to your purchased Assets.

3.  MOCKDATA + ja-translated Click "Open in Launcher", and install the Plugin for your Engine version.

4.  ja-translated Open your project, navigate to `Edit > Plugins`, and enable the `SequencePlugin`

5.  MOCKDATA + ja-translated Restart the Unreal Editor so the changes can take effect

6.  ja-translated To find the `SequencePlugin` content folder in your content drawer enable show engine content: `Content Drawer > Settings > Show Engine Content`

## MOCKDATA + ja-translated Upgrading to Unreal 5.4
ja-translated Before updating, please backup any configuration data stored in `PluginConfig/Config.h` or `Config/Config.h`. Once backed up, delete the entire SequencePlugin directory. Replace it with the new updated version of the plugin.

MOCKDATA + ja-translated If your project is currently on 5.2 or 5.3, then for this version you'll want to upgrade to 5.4.
1. ja-translated Close the Unreal Editor and your your Code Editor
2. MOCKDATA + ja-translated Right click on your Unreal Project file > "Switch Unreal Engine Version" (you may need to click "Show more options"), choose 5.4 and confirm
3. ja-translated Open your code editor and rebuild your source code

MOCKDATA + ja-translated As a way of future proofing we recommend installing Visual Studio 2022 in accordance with [these docs](https://dev.epicgames.com/documentation/en-us/unreal-engine/setting-up-visual-studio-development-environment-for-cplusplus-projects-in-unreal-engine) as Unreal 5.4 and onward will be dropping support for Visual Studio 2019.
