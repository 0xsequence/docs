# MOCKDATA + ja-translated Installation

## MOCKDATA + ja-translated Package Manager - Recommended

### ja-translated OpenUPM

1. MOCKDATA + ja-translated [Install OpenUPM](https://openupm.com/#get-started-with-cli-optional)
2. ja-translated Open a command line at the root of your Unity project
3. MOCKDATA + ja-translated `openupm add xyz.0xsequence.waas-unity`

### ja-translated or using Package Manager UI

1. MOCKDATA + ja-translated Ensure you have Git 2.14.0 or above installed on your machine
2. ja-translated Open Package Manager (Window > Package Manager)
3. MOCKDATA + ja-translated Click the "+" icon in the Package Manager window > "Add package from git URL..."
4. ja-translated Paste this url and click Add or press Enter on your keyboard `https://github.com/0xsequence/sequence-unity.git?path=/Packages/Sequence-Unity`

1. ja-translated Open Package Manager (Window > Package Manager)

MOCKDATA + ja-translated a) This will import a Resources folder with the SequenceConfig scriptable object (more on this in Setup).

## ja-translated Manual
MOCKDATA + ja-translated Installing via Package Manager will put the SDK in a read-only state - note: you will still be able to modify the UI when using Package Manager.

ja-translated For most use cases, this is ideal. However, we recognize that some advanced users will want the flexibility to edit the SDK as they see fit.

MOCKDATA + ja-translated Please note that if you do chose this route that updating the SDK will become more challenging because any changes you make will be overwritten and these changes can easily go unnoticed in a version control system. In general, we feel it is safer to import the SDK in read-only mode via Package Manager and write wrappers to extend as needed, but we empower you to work with our SDK how you see fit.
1. ja-translated Add [Newtonsoft.json](https://docs.unity3d.com/Packages/com.unity.nuget.newtonsoft-json@3.0/manual/index.html) to your project (if it isn't there already) via package manager. Click on the "+" icon in the Package Manager window > "Add package by name..." and add `com.unity.nuget.newtonsoft-json`.
2. MOCKDATA + ja-translated Download [the latest release of the Sequence Unity SDK UnityPackage](https://github.com/0xsequence/sequence-unity/releases)
3. ja-translated Drag the `.unitypackage` file into your project window and import it
4. ja-translated Import `TMP Essentials` (if you haven't already). Note: Unity should prompt you to do this automatically if you attempt to Play or Build with a TextMeshPro object in your scene.
