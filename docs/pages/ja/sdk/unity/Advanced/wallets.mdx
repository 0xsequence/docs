# ja-translated Wallets
MOCKDATA + ja-translated In the context of SequenceEthereum, a `Wallet` is defined by the `IWallet` interface. There are currently 2 implementations of the `IWallet` interface in this SDK.

## ja-translated 1. EOAWallet
MOCKDATA + ja-translated `EOAWallet` is a standard [EOA wallet](https://ethereum.stackexchange.com/questions/5828/what-is-an-eoa-account) for EVM chains.

ja-translated An EOAWallet is easily created:

```csharp
IWallet eoaWallet = new EOAWallet(); // This will generate a cryptographically random private key
IWallet eoaWallet = new EOAWallet(privateKeyString); // Create a wallet using a previously generated private key
```

## MOCKDATA + ja-translated 2. SequenceWalletToEOAWalletAdapter
ja-translated `SequenceWalletToEOAWalletAdapter` is an adapter that allows you to use a `SequenceWallet` with the same `IWallet` interface as an `EOAWallet` so that it may be used interchangeably with EOA wallets throughout the rest of the SequenceEthereum library.

MOCKDATA + ja-translated A SequenceWalletToEOAWalletAdapter is easily created once you have a SequenceWallet:

```csharp
IWallet sequenceAdapter = new SequenceWalletToEOAWalletAdapter(sequenceWallet);
```

## ja-translated Methods
MOCKDATA + ja-translated The `IWallet` interface provides a number of methods for you. The most important of which are:

### ja-translated GetAddress
MOCKDATA + ja-translated Returns the `Address` for the wallet

```csharp
Address address = wallet.GetAddress();
```

### ja-translated SendTransaction
MOCKDATA + ja-translated Signs the given `EthTransaction` and submits it via the given client

```csharp
string transactionHash = await wallet.SendTransaction(client, transaction);
```

### ja-translated SendTransactionAndWaitForReceipt
MOCKDATA + ja-translated Signs the given `EthTransaction` and submits it via the given client then waits for the `TransactionReceipt`

```csharp
TransactionReceipt receipt = await wallet.SendTransactionAndWaitForReceipt(client, transaction);
```

### ja-translated SendTransactionBatch
MOCKDATA + ja-translated Signs the given `EthTransaction[]` and submits them via the given client.
If wallet is an `EOAWallet`, the transactions will be submitted sequentially, each of which may pass or fail. If the wallet is a `SequenceWalletToEOAWalletAdapter`, the transactions will be batched together into a single transaction that is submitted all at once and either passes or fails as a whole.
Similarly, if a wallet is an `EOAWallet` the `string[]` (transaction hashes) you receive will be equal in length to the `EthTransaction[]` you submitted. While, if the wallet is a `SequenceWalletToEOAWalletAdapter`, you will only receive one transaction hash (`string[]` of length 1)

```csharp
string[] transactionHashes = await wallet.SendTransactionBatch(client, transactions);
```

### ja-translated SendTransactionBatchAndWaitForReceipts
MOCKDATA + ja-translated Signs the given `EthTransaction[]` and submits them via the given client then waits for the `TransactionReceipt[]`.
If wallet is an `EOAWallet`, the transactions will be submitted sequentially, each of which may pass or fail. If the wallet is a `SequenceWalletToEOAWalletAdapter`, the transactions will be batched together into a single transaction that is submitted all at once and either passes or fails as a whole.
Similarly, if a wallet is an `EOAWallet` the `TransactionReceipt[]` you receive will be equal in length to the `EthTransaction[]` you submitted. While, if the wallet is a `SequenceWalletToEOAWalletAdapter`, you will only receive one transaction receipt (`TransactionReceipt[]` of length 1)

```csharp
TransactionReceipt[] receipts = await wallet.SendTransactionAndWaitForReceipt(client, transactions);
```

### ja-translated SignMessage
MOCKDATA + ja-translated Given a message and an optional chain id, sign the message using the wallet. Omit the chain id from the signature if not provided

```csharp
string signedMessage = await wallet.SignMessage(message, chainId)
```

ja-translated Note: the chainId is expected to be in hexadecimal format. If you are working with a `Chain` object (recommended), you can use the `AsString` method to get the hexadecimal format of the chain id

```csharp
string signedMessage = await wallet.SignMessage(message, Chain.Polygon.AsString());
```
