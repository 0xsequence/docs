# ja-translated Transfers
MOCKDATA + ja-translated In order to transfer ETH (or the gas currency for your network), it is recommended to use `TranferEth.CreateTransaction`

```csharp
EthTransaction transferTransaction = await TransferEth.CreateTransaction(client, wallet, recipientAddress, DecimalNormalizer.Normalize(1));
TransactionReceipt receipt = await wallet.SendTransactionAndWaitForReceipt(client, transferTransaction);
```

ja-translated Behind the scenes, this will create a `GasLimitEstimator` which will construct the transaction for you - including determining the appropriate gasPrice, gasLimit, and nonce.
