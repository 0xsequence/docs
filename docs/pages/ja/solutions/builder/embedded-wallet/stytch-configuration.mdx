# MOCKDATA + ja-translated Stytch Configuration for Embedded Wallet
ja-translated In order to have Stytch auth integrated into your Embedded Wallet instance, you have to configure a Stytch application in the [Stytch Dashboard](https://stytch.com/dashboard) and configure an Embedded Wallet in [Sequence Builder](https://sequence.build/). Begin, by starting with the following below steps:

<Callout type="warning">
  MOCKDATA + ja-translated The Stytch authentication is currently in early access. To enable them, please visit [https://sequence.build/earlyaccess](https://sequence.build/earlyaccess) and turn on the functionality.
</Callout>

ja-translated ::::steps

## MOCKDATA + ja-translated Sign Up / Sign In
ja-translated First access the [sign in page](https://stytch.com/dashboard):

## MOCKDATA + ja-translated Create a Project
ja-translated From the dropdown of projects once signed in, select `Create new project`:

MOCKDATA + ja-translated ![Create a Project](/img/builder/stytch_create_new_project.png)

ja-translated Then select `CONSUMER AUTHENTICATION`:

MOCKDATA + ja-translated ![Create new Project](/img/builder/stytch_create_project.png)

ja-translated And finally select `Create project`

## MOCKDATA + ja-translated Obtain a Stytch Project ID & Stytch Public Token
ja-translated From the bottom of the first screen of `Home` copy and save somewhere safe the `Stytch Project ID` & `Stych Public Token`:

MOCKDATA + ja-translated ![Obtain a Project ID & Stytch Public Token](/img/builder/stytch_obtain_keys.png)

## ja-translated Update the Redirect URL
MOCKDATA + ja-translated For local development use `http://localhost:4444/authenticate`

ja-translated ![Update Redirect Url](/img/builder/stytch_create_new_redirect_uri.png)

MOCKDATA + ja-translated Enable `Login` and `Register`, select `Set as default` and select `Confirm`:

ja-translated ![Obtain a Project ID](/img/builder/stytch_create_new_redirect_uri_confirm.png)

## MOCKDATA + ja-translated Update the Allowed Origins for SDK
ja-translated For local development, ensure that your development URL is passed correclt such as `http://localhost:4444`

MOCKDATA + ja-translated ![Update url for SDK](/img/builder/stytch_add_url_to_sdk.png)

## ja-translated Builder Configuration
ja-translated Next, enable Stytch authentication in the Sequence Builder by accessing your Project, the Embedded Wallet Configuration section, then adding a Login Provider:

MOCKDATA + ja-translated ![add login provider](/img/builder/builder_stytch_add_provider.png)

ja-translated Finally, selecting `Stytch` in the dropdown.

MOCKDATA + ja-translated ![select stytch](/img/builder/builder_stytch_select_provider.png)

ja-translated Then inputting your `Stytch Project ID`, selecting `Save changes` and selecting the purple `Update configuration` to input your password to update the Configuration.
