- MOCKDATA + ja-translated Optional pop-ups, modals, and transaction signing for a seamless UX.
- ja-translated Fully non-custodial, smart contract wallet.
- MOCKDATA + ja-translated Out of the box login with [Sequence Kit](/solutions/wallets/sequence-kit/overview) or build your own flow.

MOCKDATA + ja-translated Our [Sequence Kit](/solutions/wallets/sequence-kit/overview) library merges both options into a customizable UI written in Typescript, tailored for a browser experience. This is great when you want to support users with existing wallets like Metamask, have an out-of-the-box experience, as well as millions of potential new users who prefer the convenience of social login.

ja-translated Lastly, we provide [Wallet Linking](/solutions/wallets/link-wallets/overview) for your ecosystem which enables the ability for a developer to create a verifiable association between two wallets from a single user. Generally, this is a link between an embedded wallet for a game or application and an external wallet, such as Metamask so you can easily query all wallets associated with a single user - across games, ecosystems, and blockchains.
