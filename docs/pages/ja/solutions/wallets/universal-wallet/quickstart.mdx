## ja-translated Install
MOCKDATA + ja-translated The sequence.js SDK is composed of several sub-packages combined into a single meta-package called `0xsequence`.
To add the SDK to your project, add a dependency on 0xsequence to your package.json:

```sh
npm install 0xsequence ethers
```

ja-translated or

```sh
pnpm install 0xsequence ethers
```

ja-translated or

```sh
yarn add 0xsequence ethers
```

MOCKDATA + ja-translated If you know that your project only requires specific sub-packages, you can also depend on those directly with
`npm add @0xsequence/<sub-package name>`.

ja-translated Your package.json should look something like this (actual version may vary):

```json
{
  "name": "my-dapp",
  ...
  "dependencies": {
    "0xsequence": "^1.0.0",
    "ethers": ^5.7.0"
    ...
  }
}
```

#### MOCKDATA + ja-translated CDN distribution as native JS
ja-translated Optionally, `0xsequence` can be installed as a JS script source (UMD distribution) via:

```js
<script src="https://cdn.ethers.io/lib/ethers-5.7.umd.min.js"></script>
<script src="https://unpkg.com/0xsequence@latest/dist/0xsequence.umd.min.js"></script>
```
