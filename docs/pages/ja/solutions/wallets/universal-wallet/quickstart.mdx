# MOCKDATA + ja-translated Quickstart
ja-translated :::info
Looking for [Unity](/sdk/unity/overview), [Unreal](/sdk/unreal/introduction) or [Mobile](/sdk/mobile)? We have you covered with our dedicated SDKs.
:::

ja-translated ::::steps

### MOCKDATA + ja-translated Clone Starter Repo
ja-translated Run the following from your terminal to clone the starter repo.

MOCKDATA + ja-translated :::code-group

```bash [React]
npx sequence-cli boilerplates create-embedded-wallet-react-starter
```

```bash [Next]
npx sequence-cli boilerplates create-kit-embedded-wallet-nextjs-starter
```

### MOCKDATA + ja-translated Copy example .env file

```bash
cd embedded-wallet-react-boilerplate && cp .env.example .env
```

### ja-translated Install Packages and start app
MOCKDATA + ja-translated :::code-group

```bash [pnpm]
pnpm install && pnpm dev
```

```bash [npm]
npm install && npm run dev
```

```bash [yarn]
yarn install && yarn dev
```

ja-translated :::

### MOCKDATA + ja-translated Connect and Authenticate
ja-translated Navigate to `localhost:4444` and click connect to try out the authentication.
