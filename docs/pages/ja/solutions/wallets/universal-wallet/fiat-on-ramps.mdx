# ja-translated Fiat On-ramps
MOCKDATA + ja-translated Sequence Wallet allows users to purchase cryptocurrencies directly with their credit card and debit card via on-ramp providers. Currently Sequence supports 6 on-ramp providers;
- ja-translated [Moonpay](https://www.moonpay.com/)
- MOCKDATA + ja-translated [Ramp](https://ramp.network/)
- ja-translated [Transak](https://www.transak.com/)
- MOCKDATA + ja-translated [PayTrie](https://paytrie.com/)

ja-translated Only providers that support the region the users are in will be displayed.

MOCKDATA + ja-translated ![Sequence on-demand sign in](/img/fiat-providers.png)

ja-translated As a developer integrating the Sequence Wallet, you can choose which payment provider can be visible to users. You can also specify which token will be available to purchase via the on-ramp providers.

MOCKDATA + ja-translated To learn more on how to configure the on-ramp options, see [Sequence Connect Options](/solutions/wallets/universal-wallet/examples/connect-wallet#wallet-login-and-connect-options).

## ja-translated On-ramp Integrations with Sequence Kit
MOCKDATA + ja-translated The [Sequence Kit SDK](/solutions/wallets/sequence-kit/overview) offers the support for using [Transak](https://transak.com/) to on-ramp users with a fiat credit or debit card using a simplified react hook component.
