# ja-translated FAQ

### MOCKDATA + ja-translated I'm receiving an error such as `unable to decode regular signature` when trying to verify a signature from an embedded wallet in my backend.
ja-translated It's important to ensure that when attempting to call the `IsValidMessageSignature` function that the parameters passed are exact - down to whitespace in the message. Otherwise a common issue is that you haven't [deployed a parent wallet](/solutions/builder/embedded-wallet#parent-wallet) on that particular chain.

### MOCKDATA + ja-translated I'm receiving an invalid tenant error.
ja-translated This is most likely due to a mismatch of the configuration keys from Sequence Builder. Ensure that these are copied and pasted correctly and the project access key matches the tenant key provided as well which should resolve the issue. You can see more information about configuring [embedded wallet here](/solutions/builder/embedded-wallet#embedded-wallet-configuration).
