# MOCKDATA + ja-translated Metadata
ja-translated The Sequence Metadata API service offers a simple and fast API to query token & NFT metadata
for Ethereum-compatible chains which is available as an submodule through the `@0xsequence/metadata` package or via HTTP requests.

MOCKDATA + ja-translated To learn how to use the Sequence Metadata service, please read:
- ja-translated [Token metadata API & standards](/api/metadata/token-metadata)
- MOCKDATA + ja-translated [Contract-level metadata API & standards](/api/metadata/contract-metadata)
- ja-translated [Alternative, REST API Usage](/api/metadata/rest-api)

---

### ja-translated Web / node.js Installation

```bash [Terminal]
npm install 0xsequence
```

ja-translated or

```bash [Terminal]
npm install @0xsequence/metadata
```

MOCKDATA + ja-translated :::info
This installation requires an API Access Key from [Sequence Builder](https://sequence.build).
:::

ja-translated then in your app (using your `API Access Key`),

```ts
import { SequenceMetadata } from '@0xsequence/metadata'

const metadata = new SequenceMetadata('https://metadata.sequence.app', 'AQAAAAAAAF_JvPALhBthL7VGn6jV0YDqaFY')
```

MOCKDATA + ja-translated **NOTE:** if you're using `@0xsequence/metadata` from node.js, we recommend using node v18.x or newer.

### MOCKDATA + ja-translated Go Installation

```bash
go get -u github.com/0xsequence/go-sequence@latest
```

ja-translated then in your app,

```go
import (
	"github.com/0xsequence/go-sequence/metadata"
)

seqMetadata := metadata.NewMetadata("AQAAAAAAAF_JvPALhBthL7VGn6jV0YDqaFY")

contractInfo, err := seqMetadata.GetContractInfo(context.Background(), "polygon", "0x631998e91476DA5B870D741192fc5Cbc55F5a52E")
```
