# シーケンスビルダーでのプロジェクト管理
プロジェクトの作成は、数回のクリックでシームレスに行えます。このプロセスにはKYCは必要なく、ユーザーは世界的な[規制とポリシー](https://support.sequence.xyz/en/article/sequence-list-of-restricted-regions-1eked2s/)に準拠し、プラットフォームの利用規約に同意する必要があります。

##### プロジェクト名
`プロジェクト名`を選択します - 楽しいもの、真面目なもの、またはウォレットにリンクされた他のプロジェクトの中で新しいプロジェクトを区別するためのユニークなもの。

##### アバター
プロジェクトにカスタム`アバター`をアップロードし、プロジェクトの一部として含めたいアイデンティティの側面を表示するか、構築しているテーマに特化したものを表示します。[常にコラボレーターを更新できます](/support/builder/project-settings#watch-how-to-invite-collaborators)し、インターネット全体で調整することができるため、アバターは信頼へのステップストーンとなることができます。

##### ネットワークの設定
1つまたは複数の`ネットワーク`を、11の提供されているプロダクション/メインネットワークまたは8つのテストネットワークから、トグルを簡単に切り替えることで単一のプロジェクトに設定します。プロジェクトのネットワークは、いつでも設定で更新できます。

これら3つの値すべて：`プロジェクト名`、`アバター`、設定された`ネットワーク`は後で変更可能です。
