# トークンディレクトリ
トークンがさまざまなブロックチェーンに追加されるとき、デフォルトではこれらのトークンはデフォルトのウォレット設定に含まれません。多くのトークンが検証されるまでスパムと見なされる可能性があるためです。

契約を検証する最良の方法は、シーケンスビルダーを通じてデプロイすることで、自動的にトークンディレクトリに追加されます。

また、外部契約やコレクションの検証もサポートしています。これを行うには、[こちらからお問い合わせください](https://go.crisp.chat/chat/embed/?website_id=9ef4395b-6bb1-4645-8e02-6071d89290a1)。

それ以外の場合、テスト用にトークンをデプロイしている場合は、シーケンスウォレットを設定して未検証のトークンを表示することもできます。

## トークンをウォレットに表示させる方法
ビルダー内のシーケンスウォレットから、[設定 > 一般](https://sequence.app/settings/general)に移動し、`未検証のトークンを非表示`をオフにします。

<Callout type="warning">
  Note <br />
  This will show all unverified tokens which can be utilized for testing purposes.
</Callout>
