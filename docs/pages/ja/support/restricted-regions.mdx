# シーケンスの制限地域リスト
カナダ、アメリカ、その他の世界的な管轄区域によって義務付けられた規制およびポリシーに準拠するため、シーケンスは以下の地域ではアクセスできません。暗号通貨のオンランプを支援する外部の第三者のマネーサービスプロバイダーが、独自の裁量で特定の地理的場所への資金提供を拒否する場合があることにご注意ください。
