# サポート
シーケンスは、スタックに関する問題に対処し、新しいプロジェクトの質問に答えるために、複数のタイムゾーンでフロントラインサポートチームを持っています。これにより、ゲームやアプリケーションでのWeb3体験を統合することができます。

チームメンバーの一人が答えを知らない場合、シーケンスの開発チームのバックアップサポートを受けて、問題の根本を追求します。
