Sequence Indexerは、Ethereum互換ネットワークからブロックチェーントークンとNFTデータを簡単にクエリできるモジュラーなweb3サービスです。IndexerはEthereum互換チェーンのすべてのERC20、ERC721、ERC1155トークンを自動的にインデックス化します。

Sequence Indexerは、オンチェーンデータをクエリする必要のあるウォレット、ゲーム、dappsを構築するための強力なツールです。高速で信頼性が高く、使いやすいように設計されています。[Sequence Node Gateway](/api-references/node-gateway)を活用しており、これはノードの健全性を監視し、利用可能なノードにリクエストをインテリジェントにルーティングし、最新のリクエスト応答をキャッシュします。これにより、ノードプロバイダーがダウンしても問題ありません。Node Gatewayが自動的に健全なプロバイダーにルーティングし、Indexerが常にダウンタイムなくチェーンの最新状態を維持できるようにします。

Indexerを使用する他の利点は、社内で構築された次世代のLSMツリーベースのデータベースを使用してデータを保存していることです。これにより、データのクエリとインデックス化が非常に高速になります。データは低レイテンシーでリアルタイムに利用可能です。シンプルなREST APIまたは当社のSDKの1つを使用してクエリを実行できます。

Indexer APIを使用すると、すべてのEthereum互換チェーンのすべてのトークン残高、履歴、その他の情報をクエリできます。すべてのトークンデータニーズのためのワンストップの場所にするために、[Sequence Metadata](/api-references/metadata/overview)のサポートをすべてのトークンに含めています。これにより、別のAPIからメタデータを取得する必要なく、アプリ、ゲーム、またはウォレットでトークンを簡単にレンダリングできます。

実際、Sequence WalletはバックグラウンドでIndexerを使用しており、ウォレット内のすべてのトークン情報をシームレスにレンダリングできます。もちろん、Indexerはモジュラーなインフラストラクチャーであり、dapp、ゲーム、さらにはサーバーから直接使用することができます。

**機能:**

* マルチチェーンに対応した、すべてのトークン残高、履歴、メタデータ、NFTを照会できる超高速APIです
* EVM互換チェーン全体でのERC20、ERC721、ERC1155トランザクションのリアルタイムインデックス作成
* コントラクトレジストリを必要とせず、チェーン上のすべてのトークンを自動的に検出
* ノード障害やチェーンの再編成に対する耐性
* シンプルなAPIで、チェーン上の特定のイベントやトランザクションを正確に監視可能
* アプリやゲームでトークンを簡単にレンダリングできる組み込みのトークン/NFTメタデータサポート
* 高稼働率と可用性
