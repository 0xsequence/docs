私たちは以下のSDKを提供しています：[Web / node.js](https://github.com/0xsequence/sequence.js) と [Go](https://github.com/0xsequence/go-sequence)。
または、別の言語でRelayerを統合したい場合は、以下のAPIリファレンスに従ってHTTPリクエストを実装してください。また、Transactions APIクライアントの[リファレンス実装](https://github.com/0xsequence/sequence.js/blob/master/packages/relayer/src/rpc-relayer/relayer.gen.ts)としてTypeScriptクライアントのソースコードもご覧ください。

### Web / node.js インストール

```sh
npm install 0xsequence ethers
```

または

```sh
pnpm install 0xsequence ethers
```

または

```sh
yarn add 0xsequence ethers
```

### Go インストール

```bash Terminal
go get -u github.com/0xsequence/go-sequence@latest
```

その後アプリケーションで、

```go Go
import (
	"github.com/0xsequence/go-sequence/relayer"
)
```
