## 認証

ほとんどのSequence APIは、以下のいずれかを使用した認証が必要です：

* **Project Access Key**：クライアントサイドのリクエストの場合、これらは`public`として表示されます。
* **Service Token**：昇格された権限が必要なサーバーサイドのリクエストの場合、シークレットベアラートークンの受け渡しが必要で、`secret`として表示されます。

これらの認証情報は[Sequence Builder](https://sequence.build)ダッシュボードで生成できます。

## レート制限

APIレート制限は、ティアと利用可能なMAUによって異なります。具体的な制限については、Sequence Builderダッシュボードのプロジェクト設定をご確認ください。
