# Indexer Gateway

Sequence Indexer Gateway APIは、複数のチェーンにまたがってクエリを同時に実行するための強力なツールです。Indexerを活用することで、Indexer Gatewayは対応するEthereumチェーン全体でデータを検索およびフィルタリングすることができます。

このAPIはIndexerをモデルにしているだけでなく、実際にバックエンドでIndexerを使用しているため、既存のIndexerで動作するコードをわずかな修正で複数チェーン対応にすることができます。

## 対応ネットワークとエンドポイント

Indexer Gatewayのエンドポイントは、ネットワークID（例：`[1, 137,
10]`）またはネットワーク名（例：`["mainnet", "polygon", "optimism"]`）のリストを受け付けます。

こちらで[対応ネットワークの完全なリストをご確認いただけます](https://status.sequence.info)。
