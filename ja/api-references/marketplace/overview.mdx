Sequence Marketplace APIサービスは、シーケンスマーケットプレイスプロトコルと対話するためのシンプルで高速な方法を提供します。このAPIは使いやすく設計されており、完全に機能するマーケットプレイスを構築するために必要なすべての機能を備えています。様々な[例](/api-references/marketplace/examples/orderbook-transactions)または[カスタムマーケットプレイスガイド](/guides/custom-marketplace)で完全なウォークスルーをご確認ください。
