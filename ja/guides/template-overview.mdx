市場投入までの時間を短縮するため、オープンソースのコードデモ、例、ボイラープレートでSequenceの統合を始めましょう。

## オンボーディング

### Web SDK搭載の組み込みウォレット用Reactボイラープレート

シンプルなReactアプリケーションでWeb SDKと統合された組み込みウォレットを紹介するボイラープレートの例。

Sequence-CLIで実行:

```bash
npx sequence-cli boilerplates create-embedded-wallet-react-starter
```
