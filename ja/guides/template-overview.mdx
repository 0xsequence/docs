オープンソースのコードデモやサンプル、ボイラープレートを活用して、Sequence の導入を市場投入までの時間を短縮しましょう。

## オンボーディング

### React 用 Web SDK ＋ Embedded Wallet ボイラープレート

Web SDK と組み合わせた Embedded Wallet をシンプルな React アプリで紹介するサンプルボイラープレートです。

Sequence-CLI で実行：

```bash
npx sequence-cli boilerplates create-embedded-wallet-react-starter
```

### バックエンドでウォレット検証を行う Embedded Wallet ボイラープレート

### Unreal Engine 用 Embedded Wallet ボイラープレート
