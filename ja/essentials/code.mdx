## 基本

### インラインコード

単語や`word`フレーズ`phrase`をコードとして示すには、バッククォート(\`)で囲みます。

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

### コードブロック

使用する[フェンスで囲まれたコードブロック](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)は、3つのバッククォートでコードを囲み、先頭のバッククォートの後にスニペットのプログラミング言語を記述することで、シンタックスハイライトが得られます。オプションで、プログラミング言語の後にコードの名前を書くこともできます。

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
