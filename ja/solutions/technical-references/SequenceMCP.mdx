# Sequence MCPサーバーのセットアップ方法
Sequence MCPサーバーを利用すると、Claudeのような強力なAIエージェントをお好みのIDEで直接活用できます。この統合により、ブロックチェーンデータのクエリやスマートコントラクトとのやり取りなど、Sequence関連の開発作業にインテリジェントな支援を受けられます。

## 事前準備
始める前に、以下が用意されていることを確認してください：
- Node.jsがシステムにインストールされていること
- 対応IDE（CursorまたはClaude Desktop）
- npmまたはyarnパッケージマネージャー

## インストール手順

1.  ターミナルを開き、次のコマンドを実行してください：
    ```bash
    npx @mintlify/mcp@latest add sequence-0fb8d9e6
    ```

2.  プロンプトが表示されたら、希望するMCPクライアントを選択してください：
    - Cursor
    - Claude Desktop

3.  インストールプロセスは、選択したアプリケーション用にSequence MCPサーバーを自動的に設定します。

## MCPサーバーの利用方法
インストール後は、IDE内でAIアシスタンスを利用できます。手順は以下の通りです：
1. IDE（CursorまたはClaude Desktop）を開く
2. Sequence開発に関するクエリを入力し始める
3. AIエージェントがリクエストを処理し、関連するサポートを提供します

### クエリ例
次のような質問ができます：
- 「アドレス 0x8e3E38fe7367dd3b52D1e281E4e8400447C8d8B9 のトークン残高とメタデータを取得するスクリプトを作成してください」
- 「Sequenceの埋め込みウォレットを統合するにはどうすればよいですか？」
- 「ウォレットのトランザクション履歴を取得する方法を教えてください」

クエリが処理中の場合、次のような表示がされます：

```
> Calling MCP tool
```
