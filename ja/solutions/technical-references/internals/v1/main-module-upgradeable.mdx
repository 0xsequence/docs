---
title: Main Module Upgradeable & 設定移行
description: Main Module Upgradeableはウォレットシステムの重要なコントラクトで、ウォレットアドレスを変更せずにシームレスなアップグレードを可能にします。
---

Main Module Upgradeableはウォレットのロジックを保持するメインコントラクトです。アップグレード可能であり、ウォレットアドレスを変えずにロジックを変更できます。

ウォレットがデプロイされると、`MainModule`が初期コード実装として使われます。このモジュールは、ウォレットのカウンターファクチュアルアドレスを使ってウォレット設定の`imageHash`を検証します。ウォレット設定が更新されると、`MainModule`は自動的にウォレットコード実装を`MainModuleUpgradeable`に置き換え、ストレージの初期化も行います。
