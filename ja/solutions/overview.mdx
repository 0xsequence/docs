[Sequence](https://sequence.xyz/)は、オンチェーンアプリを迅速にリリースするためのモジュール型Web3開発者向けプラットフォームおよびインフラスタックです。**ウォレット**、**リアルタイムインデクサ**、**決済ソリューション**などの機能を活用し、既存のスタックに組み込めます。まずは\*\*[Sequence Builder](https://sequence.build/)\*\*でプロジェクトを作成し、お好みのフレームワーク（[React](/sdk/web/overview)、[React Native](/sdk/mobile)、[Unity](/sdk/unity/overview)、[Unreal](/sdk/unreal/overview)）で開発を始めましょう。

## はじめに

<CardGroup cols={2}>
  <Card title="最初のプロジェクトを作成" icon="folder-plus" href="/solutions/builder/getting-started">
    Sequence Builderでプロジェクトをセットアップし、APIキーを取得して、アプリに必要なコア機能（ウォレット、インデクサ、決済）を有効化しましょう。
  </Card>

  <Card title="埋め込みウォレットデモ" icon="gem" href="https://0xsequence.github.io/kit/?mode=waas">
    Web SDKを使った埋め込み型ウォレットのブラウザデモをお試しください。カスタマイズ可能なUI、ブランド対応のフロー、すぐに使える認証機能を備えています。
  </Card>

  <Card title="クイックスタート：ウォレット" icon="wallet" href="/solutions/wallets/embedded-wallet/overview">
    ノンカストディアルでポップアップのないウォレット体験を、ソーシャル／メールサインインで追加できます。EOAウォレットともリンクして併用可能です。
  </Card>

  <Card title="クイックスタート：リアルタイムインデクサ" icon="database" href="/solutions/builder/indexer">
    低遅延APIで、EVMチェーン全体の残高、送金、コントラクトイベントをクエリできます。
  </Card>

  <Card title="クイックスタート：決済" icon="cart-shopping" href="/sdk/web/guides/checkout">
    カードや暗号資産決済を受け付け、オンランプを有効化し、ドロップイン型Checkout SDKで購入をスムーズに。
  </Card>

  <Card title="コミュニティに参加" icon="discord" href="https://discord.com/invite/sequence">
    サポートが必要な場合やフィードバックの共有、他の開発者との交流はこちらから。
  </Card>
</CardGroup>

## コア構成要素

<CardGroup cols={3}>
  <Card title="ウォレット" icon="wallet" href="/solutions/wallets/embedded-wallet/overview">
    ガス不要オプションや最小限のユーザー負担を備えた、組み込み型ノンカストディアルウォレット。Web2サインイン対応、EOAウォレット連携、取引ポップアップなしでUXをシンプルに保ちます。
  </Card>

  <Card title="リアルタイムインデクサ" icon="database" href="/solutions/builder/indexer">
    EVMチェーン向けの高性能インデックス：残高、トークン価格、送金、コントラクトイベントをリアルタイムかつ大規模にクエリ可能。
  </Card>

  <Card title="決済" icon="coins" href="/sdk/web/guides/checkout">
    カード＋暗号資産のチェックアウト、法定通貨からのオンランプ、シームレスなアプリ内購入。シンプルでモジュール式のAPIで迅速に資金をオンチェーン化。
  </Card>
</CardGroup>
