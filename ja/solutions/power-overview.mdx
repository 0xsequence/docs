---
title: Sequence でのパワー
description: Sequence の実績のある機能であなたのブロックチェーンゲームをパワーアップ。監査済みのコントラクトをデプロイし、NFT メタデータを管理し、web3 アクティビティを追跡し、トランザクションを管理し、Webhook に簡単に接続。ユーザーのためにガス代を抽象化し、ノードインフラを簡単に管理。
sidebarTitle: 概要
---

Sequence には、ブロックチェーン対応のゲームを構築し、数百万人のプレイヤーにスケールするために不可欠な、実績のある機能が多数あります。

## コントラクトのデプロイ

<CardGroup cols={2}>
  <Card title="ERC20" icon="coins" href="/solutions/collectibles/contracts/deploy-ERC20-currency">
    Sequence Builder を通じて ERC20 代替可能トークンをデプロイするための、標準的で監査済みのスマートコントラクト。
  </Card>

  <Card title="ERC721" icon="image" href="/solutions/collectibles/contracts/deploy-an-item-collection">
    Sequence Builder を通じて標準的で監査済みの非代替性トークン（ERC721）をデプロイ。
  </Card>

  <Card title="ERC1155" icon="layer-group" href="/solutions/collectibles/contracts/deploy-an-item-collection">
    私たちが共同執筆に携わった、標準的で監査済みの準代替性トークン（ERC1155）を Sequence Builder を通じてデプロイ。
  </Card>

  <Card title="Link any contracts" icon="link" href="/solutions/builder/contracts">
    ABI を提供してカスタムコントラクトをアップロードし、Sequence Builder から管理。
  </Card>
</CardGroup>

## ゲームを構築

<CardGroup cols={2}>
  <Card title="Analytics" icon="chart-line" href="/solutions/builder/analytics">
    ゲーム内のアクティビティ、リテンション、その他あらゆる側面を追跡。
  </Card>

  <Card title="Metadata Management" icon="database" href="/solutions/collectibles/metadata">
    UI を使用するか、メタデータ API を使用してスケールでコレクションのメタデータを更新。
  </Card>

  <Card title="Indexer" icon="magnifying-glass" href="/solutions/builder/indexer">
    ゲームやアプリで使用するウォレットとコントラクトを高パフォーマンスでクエリ。
  </Card>

  <Card title="Webhooks" icon="bell" href="/solutions/builder/webhooks">
    HTTPS コールバックを提供することで、ブロックチェーン全体の任意のイベントをリッスンする Webhook をデプロイ。
  </Card>
</CardGroup>
