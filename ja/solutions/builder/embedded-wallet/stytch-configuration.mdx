sidebarTitle: Stych設定
---

Embedded WalletインスタンスにStytch認証を統合するには、[Stytch Dashboard](https://stytch.com/dashboard)でStytchアプリケーションを設定し、[Sequence Builder](https://sequence.build/)でEmbedded Walletを設定する必要があります。以下の手順から始めましょう：

<Warning>
  Stytch認証は現在アーリーアクセス段階です。これを有効にするには、[https://sequence.build/earlyaccess](https://sequence.build/earlyaccess)にアクセスして機能をオンにしてください。
</Warning>

<Steps>
  <Step title="Sign Up / Sign In">
    まず[サインインページ](https://stytch.com/dashboard)にアクセスします：
  </Step>

  <Step title="Create a Project">
    サインイン後、プロジェクトのドロップダウンから選択します`Create new project`：

    <Frame>
      ![Create a Project](/images/builder/stytch_create_new_project.png)
    </Frame>

    次に選択します`CONSUMER AUTHENTICATION`：

    <Frame>
      ![Create new Project](/images/builder/stytch_create_project.png)
    </Frame>

    最後に選択します`Create project`
  </Step>

  <Step title="Obtain a Stytch Project ID & Stytch Public Token">
    最初の画面の下部から`Home`をコピーして安全な場所に保存してください`Stytch Project ID`と`Stych Public Token`：

    <Frame>
      ![Obtain a Project ID & Stytch Public Token](/images/builder/stytch_obtain_keys.png)
    </Frame>
  </Step>

  <Step title="Update the Redirect URL">
    ローカル開発には`http://localhost:4444/authenticate`

    <Frame>
      ![Update Redirect Url](/images/builder/stytch_create_new_redirect_uri.png)
    </Frame>

    有効化`Login`と`Register`を選択し、`Set as default`を選択して`Confirm`：

    <Frame>
      ![Obtain a Project ID](/images/builder/stytch_create_new_redirect_uri_confirm.png)
    </Frame>
  </Step>

  <Step title="Update the Allowed Origins for SDK">
    ローカル開発の場合、開発URLが正しく渡されていることを確認してください。例えば`http://localhost:4444`

    <Frame>
      ![Update url for SDK](/images/builder/stytch_add_url_to_sdk.png)
    </Frame>
  </Step>

  <Step title="Builder Configuration">
    次に、Sequence Builderでプロジェクトにアクセスし、Embedded Wallet設定セクションでStytch認証を有効にし、ログインプロバイダーを追加します：

    <Frame>
      ![add login provider](/images/builder/builder_stytch_add_provider.png)
    </Frame>

    最後に`Stytch`をドロップダウンから選択します。

    <Frame>
      ![select stytch](/images/builder/builder_stytch_select_provider.png)
    </Frame>

    その後、`Stytch Project ID`を入力し、`Save changes`を選択し、紫色の`Update configuration`を選択してパスワードを入力し、設定を更新します。
  </Step>
</Steps>
