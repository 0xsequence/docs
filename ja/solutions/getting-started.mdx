サインアップとプロジェクト作成は簡単で、数分以内に完了します。Builderへのサインアップから最初のプロジェクト作成までサポートします。

さっそく始めましょう！

## Sequence Builderにサインアップする

<Steps>
  <Step title="sequence.buildにアクセス">
    [Sequence.Build](https://sequence.build/)にアクセスし、`Start Building` または `Sign in` ボタンをクリックして始めてください。

    <Frame>
      ![Sequence builder ランディングページ](/images/builder/builder_landing_page_signup_project.png)
    </Frame>
  </Step>

  <Step title="サインイン方法を選択">
    初めてBuilderにサインインすると、ウォレットが自動的に作成されます。

    すでにSequenceウォレットをお持ちの場合は、そのウォレットでサインインしていると、利用するオプションが表示されます。

    ウォレットがない場合は、いくつかのサインイン方法が表示されます。ソーシャルサインインやメールアドレスで簡単に始められます。

    <Frame>
      ![Sequence builder サインイン選択](/images/builder/builder_choose_signin_signup_project.png)
    </Frame>
  </Step>

  <Step title="sequence.buildにサインイン">
    サインイン方法を選択すると、利用規約が表示されます。`Terms of Use`、`Privacy Policy`、`Master Services Agreement` を読むには、それぞれのリンクをクリックしてください。

    <Frame>
      ![Sequence builder 利用規約の同意](/images/builder/builder_accept_terms_signup_project.png)
    </Frame>

    すべて読んだら、続行するためにクリックしてください。

    これでBuilderにサインインできました！
  </Step>

  <Step title="プロジェクトの作成">
    Builderでの操作はすべてプロジェクト単位で行われます。さっそくプロジェクトを作成しましょう。

    プロジェクトを始める最も簡単な方法は、`+ Create Project` をクリックすることです。

    <Frame>
      ![Sequence builder ダッシュボード](/images/builder/builder_dashboard_signup_project.png)
    </Frame>
  </Step>

  <Step title="プロジェクトの詳細を入力">
    モーダルが表示され、プロジェクトの詳細を入力できます。プロジェクト名を入力し、必要に応じてアバターも追加できます。

    次に、デプロイを開始したいネットワークを選択します。後から変更も可能です。

    <Frame>
      ![Sequence builder 詳細入力](/images/builder/builder_enter_details_signup_project.png)
    </Frame>
  </Step>

  <Step title="プロジェクトを作成する">
    `Create new Project` ボタンをクリックしてください。

    <Frame>
      ![Sequence builder プロジェクト作成完了](/images/builder/builder_project_created_signup_project.png)
    </Frame>

    完了すると、プロジェクトのダッシュボードが表示されます。
  </Step>

  <Step title="APIアクセスキーを取得する">
    最後に、開発を始めるにはBuilderからアクセスキーを取得する必要があります。左上の `project` をクリックし、サイドナビの `Settings` を選択し、`API Access Keys` カードをクリックしてください。

    <Frame>
      ![プロジェクト設定を開く](/images/builder/builder_open_access_key_settings.png)
    </Frame>

    `API Access Keys` の `settings` で、開発環境用の `dev` アクセスキーを選択します。`prod` キーは本番環境用です。

    <Frame>
      ![APIキー設定](/images/builder/builder_settings_api_keys.png)
    </Frame>
  </Step>

  <Step title="（開発用オプション）サービスアカウントの作成">
    Sequenceスタックの一部サービスを統合するには、サービスアカウントを作成して状態の更新や読み取りを行う必要があります（例：[Collections API](/guides/metadata-guide) や [Webhooks](/api-references/indexer/examples/webhook-listener) など）。その際は `Secret API Key` が必要です。

    プロジェクトを選択し、サイドバーの `Settings` ページから `API Access Keys` カードを選択してAPIアクセスキーのセクションに移動します。

    <Frame>
      ![プロジェクト設定を開く](/images/builder/builder_open_access_key_settings.png)
    </Frame>

    ページ下部にスクロールすると、`Secret API Access Key(s)` セクションがあります。

    ここで `+ Add Service Account` を選択します。

    <Frame>
      ![シークレットAPIキー](/images/builder/builder_service_accounts_create_secret_api_key.png)
    </Frame>

    `Read` キーまたは `Write` シークレットAPIアクセスキーを作成できます。

    Sequenceシステムの更新を行う場合は、`Write` を選択し、`+ Add Service Account` をクリックして統合に追加してください。

    <Frame>
      ![書き込みキーの作成](/images/builder/builder_service_accounts_create_write_key.png)
    </Frame>

    JWTトークンをコピーして安全な場所、またはプロジェクト内に保存してください。

    <Warning>
      注意：生成された `Secret API Access Key` は今後Builderの画面から再表示できませんので、必ず安全な場所に保存してください。
    </Warning>
  </Step>
</Steps>
