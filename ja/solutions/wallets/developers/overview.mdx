Sequenceは、シンプルなdAppとプリビルトウォレットから、数千のウォレットをプログラムで管理する複雑なバックエンドサービスまで、あらゆるものを構築できる強力な多層ツール群を提供しています。
私たちのツールは、用途に応じて2つの主要カテゴリに分かれています。
- 🌐 **フロントエンド向け:** Web SDKは、ユーザー向けアプリケーション構築のためのオールインワンReact Hooksソリューションです。
- ⚙️ **バックエンド向け:** Sequence.jsは、サーバーサイド処理やカスタム・低レベルウォレット統合のための基盤ライブラリ群です。

## SDK同士の連携イメージ
Web SDKはSequence.jsをベースに構築されており、フロントエンド開発向けに高レベルで使いやすい抽象化を提供します。プロジェクトの構成に応じて、両方を組み合わせて使うことも、個別に使うことも可能です。

**Web SDK**は、dAppに完全なノンカストディアル型スマートウォレット体験を最速で追加できる方法です。
Reactでユーザー向けアプリケーションを構築する場合は、ここから始めてください。Web SDKは迅速な統合のために設計されており、必要な機能をシンプルなReact hooksで提供します。
主な特長：
- **完全なウォレットUI**：接続、資産表示、トランザクション管理ができる美しいプリビルトのウォレットモーダル。
- **シームレスなオンボーディング**：ソーシャルログイン、メール、パスキーに標準対応。
- **簡単なトランザクション操作**：メッセージ署名、トランザクション送信、ガス代負担のためのhooks。
- **強力な抽象化**：Smart Sessions、バッチトランザクション、法定通貨オンランプなどを標準サポート。
- **UIコンポーネント**：NFT決済やスワップなどのReactコンポーネントをすぐに利用可能。

**おすすめ対象**：
- フロントエンドdApp開発者	DeFiプロトコル、NFTマーケットプレイス、Web3ゲーム、その他ユーザー向けアプリケーションの構築に。

➡️ はじめる：[Web SDKを5分で統合する](/solutions/wallets/developers/web-sdk)

**Sequence.js**は、Sequenceウォレット基盤への直接的かつプログラム的なアクセスを提供する強力で柔軟なライブラリ群です。バックエンドでウォレット管理や複雑なトランザクションの制御、ゼロからカスタムウォレット体験を構築する際にご利用いただけます。

**主な機能**
- **サーバーサイド署名**：バックエンドから安全にトランザクションを承認・送信（例：バッチミントやエアドロップなど）。
- **独自ウォレットの構築**：API経由でスマートウォレットを作成・設定。
- **カスタムスマートセッション**：独自の複雑なスマートセッションを構築。
- **ガスレス操作**：中央管理のガスタンクからユーザーのトランザクションをスポンサー。
- **カスタムクライアントの基盤**：非ReactフロントエンドやモバイルアプリのコアロジックとしてSequence.jsを利用可能。

**おすすめ対象**：
- バックエンド開発者・パワーユーザー	セッションキーを管理するゲームサーバー、NFTをエアドロップするバックエンドサービス、非Reactフレームワーク向けのカスタムウォレットなどの構築に。
