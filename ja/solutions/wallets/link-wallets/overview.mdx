# ウォレットリンクの概要

ウォレットリンクは、開発者が単一のユーザーの2つのウォレット間で検証可能な関連付けを作成することを可能にします。一般的に、これはゲームやアプリケーションの組み込みウォレットとMetamaskなどの外部ウォレットとのリンクです。これにより、ゲームにおける以下のような問題が解決されます：

* トークンゲーティングや報酬配布のために価値のある資産を移動させることなく、ユーザーがNFTを保有していることを検証できます。
* 外部ウォレットからゲームの組み込みウォレットへの資金や流動性の移行が容易になります。
* ガバナンス権を提供するために、ユーザーが保有するERC20トークンの数を検証できます。

Sequence SDKsを使用することで、モバイル、Web、PC、コンソールなどほぼすべてのプラットフォームから組み込みウォレットセッションを開始でき、ユーザーはシームレスなゲーム体験を楽しむことができます。ただし、必要に応じて、追加のウォレットをリンクするために外部ブラウザセッションにユーザーを誘導します。
