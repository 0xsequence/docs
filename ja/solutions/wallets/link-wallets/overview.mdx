# ウォレット連携の概要
ウォレット連携を利用すると、開発者は1人のユーザーが持つ2つのウォレット間で検証可能な関連付けを作成できます。一般的には、ゲームやアプリのEmbedded WalletとMetamaskなどの外部ウォレットを連携します。これにより、ゲームで以下のような課題を解決できます：
- NFTを保有していることを証明しつつ、貴重な資産を移動せずにトークンゲートや報酬配布を行いたい場合。
- 外部ウォレットからゲーム用Embedded Walletへの資金や流動性のスムーズな移行。
- ユーザーが保有する複数のERC20トークンを検証し、ガバナンス権を付与する場合。

Sequence SDKを使えば、Embedded Walletのセッションはモバイル、Web、PC、コンソールなどほぼすべてのプラットフォームで開始でき、ユーザーはシームレスなゲーム体験を楽しめます。必要に応じて、ユーザーは外部ブラウザセッションに誘導され、追加のウォレットを連携できます。
