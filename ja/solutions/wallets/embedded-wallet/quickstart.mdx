<Steps>
  <Step title="スターターリポジトリをクローン">
    ターミナルで以下のコマンドを実行し、スターターリポジトリをクローンします。

    <CodeGroup>
      ```bash React
      npx sequence-cli boilerplates create-embedded-wallet-react-starter
      ```

      ```bash Next
      npx sequence-cli boilerplates create-kit-embedded-wallet-nextjs-starter
      ```
    </CodeGroup>
  </Step>

  <Step title="接続と認証">
    ブラウザで `http://localhost:4444` にアクセスし、<b>Connect</b> をクリックして認証プロセスを開始します。

    これで Embedded Wallet をボイラープレートでテストできます。次はご自身のプロジェクトを設定しましょう。
  </Step>

  <Step title="プロジェクトの設定">
    <a href="https://sequence.build/project/default/embedded-wallet?tab=config">Builder 上のプロジェクトの埋め込みウォレット設定ページ</a>にアクセスし、リカバリーウォレットアドレスを入力してパスワードを設定し、<b>Create Configuration</b> をクリックしてください。この設定は後から変更できませんのでご注意ください。
  </Step>

  <Step title="環境変数の更新">
    クローンしたディレクトリに移動し、この設定を `.env` ファイルにコピー＆ペーストして既存の設定を置き換えてください。

    <a href="https://sequence.build/project/default/embedded-wallet?tab=integration">インテグレーションページ</a>にある新しい `PROJECT_ACCESS_KEY` と `WAAS_CONFIG_KEY` を必ず更新してください。

    ```tsx
    # Update with your project access key
    VITE_PROJECT_ACCESS_KEY="AQAAAAAAAKJcPk5v95BJE-Q7GwdlY9hPuAM"
    # Update with your WAAS_CONFIG_KEY
    VITE_WAAS_CONFIG_KEY="eyJwcm9qZWN0SWQiOjQxNTY0LCJycGNTZXJ2ZXIiOiJodHRwczovL3dhYXMuc2VxdWVuY2UuYXBwIn0="
    VITE_GOOGLE= NULL
    VITE_APPLE= NULL
    VITE_WALLET_CONNECT_ID= NULL
    VITE_CHAINS= 80002
    VITE_DEFAULT_CHAIN= 80002
    ```
  </Step>

  <Step title="新しい設定のテスト">
    `.env` ファイルを保存し、ルートディレクトリで `pnpm dev` を実行して `http://localhost:4444` に戻ります。メールでウォレットにサインインし、ウォレットの動作を再度テストしてください。

    これで完了です！Overview ページでウォレット接続が確認でき、キーはプロジェクトごとにスコープされています。
  </Step>
</Steps>
