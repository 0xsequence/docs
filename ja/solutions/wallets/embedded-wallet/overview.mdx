Sequence Embedded Wallet は、ユーザーがソーシャルプロバイダーやメール、ゲストとして認証できる非カストディアル型ウォレットソリューションです。[Web](/sdk/headless-wallet/quickstart)、[Unity](/sdk/unity/overview)、[Unreal](/sdk/unreal/introduction) など、さまざまな SDK でご利用いただけます。

これらのヘッドレスウォレットはユーザーごとにオンデマンドで作成でき、アプリケーションを Ethereum 互換ブロックチェーンと統合する際に、RPC API など馴染みのある仕組みを利用できます。また、不要なポップアップやユーザー向け確認画面などの煩わしさがなく、真にネイティブでシームレスなゲーム体験を提供できます。

Sequence Embedded Wallet は本物の非カストディアル型ウォレットを提供し、作成された各ウォレットはその所有者の同意がなければアクセスできないことをシステムが保証します。これは、[AWS Nitro Enclaves](https://aws.amazon.com/ec2/nitro/nitro-enclaves/) の強力なセキュリティ環境を活用し、機密データを誰もアクセスできない安全な環境で処理することで実現しています（Sequence でさえアクセスできません）。

また、ご希望に応じて複数の体験間でウォレットを共有できるため、ユーザーはどのソーシャルログインを利用しても、ゲームやプラットフォームをまたいで一貫したウォレットアドレスを持つことができます。

## デモを試す

  Sequence Embedded Wallet のシームレスなオンボーディングを[こちら](https://waas-demo.sequence.xyz/)でお試しください。
</Note>

実装の準備ができたら、[クイックスタートガイド](/sdk/headless-wallet/quickstart)に従うか、お好みのパッケージマネージャーから SDK をインストールしてください。

<CodeGroup>
  ```bash pnpm
  pnpm install @0xsequence/waas
  ```

  ```bash npm
  npm install @0xsequence/waas
  ```

  ```bash yarn
  yarn add @0xsequence/waas
  ```
</CodeGroup>
