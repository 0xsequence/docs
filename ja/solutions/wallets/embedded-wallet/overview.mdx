Sequence 組み込みウォレットは、ソーシャルプロバイダー、メール、またはゲストとして認証できる非カストディアルウォレットソリューションです。これは様々なSDKを通じて利用可能です：[Web](/sdk/headless-wallet/quickstart)、[Unity](/sdk/unity/overview)そして[Unreal](/sdk/unreal/introduction)。

これらのヘッドレスウォレットはユーザーごとにオンデマンドで作成でき、RPCのAPIのような馴染みのある構造を使用しながら、Ethereum互換のブロックチェーンとアプリケーションを統合することができます。さらに、ポップアップモーダル、ユーザーの確認、その他の気を散らすものが不要なため、真にネイティブでシームレスなゲーム体験を提供することができます。

Sequence 組み込みウォレットは真の非カストディアルウォレットを提供し、作成された各ウォレットはウォレット所有者の同意がある場合にのみアクセスできることをシステムが保証します。これは[AWS Nitro Enclaves](https://aws.amazon.com/ec2/nitro/nitro-enclaves/)の力を活用して、機密データを誰も - Sequenceでさえも - アクセスできない安全な環境で処理することで可能になっています。

これらは必要に応じて体験間で共有されるため、ユーザーは使用するソーシャルログインに関係なく、ゲームやプラットフォーム間で一貫したウォレットアドレスを持つことができます。

## デモを試す

<Note>
  Sequence 組み込みウォレットのシームレスなオンボーディングを[こちら](https://waas-demo.sequence.xyz/)
</Note>

実装の準備ができたら、[クイックスタートガイド](/sdk/headless-wallet/quickstart)に従うか、お好みのパッケージマネージャーからSDKをインストールしてください：

<CodeGroup>
  ```bash pnpm
  pnpm install @0xsequence/waas
  ```

  ```bash npm
  npm install @0xsequence/waas
  ```

  ```bash yarn
  yarn add @0xsequence/waas
  ```
</CodeGroup>
