## 主な機能

* **マルチプラットフォーム対応**：Web、モバイル、ゲームプラットフォーム全体でアプリケーションを構築
* **ウォレット統合**：人気のフレームワーク向けの使いやすいウォレットコネクタ
* **スマートコントラクトの操作**：スマートコントラクトのデプロイと操作のためのツール
* **トランザクション管理**：簡素化されたトランザクションの作成、署名、送信
* **型安全性**：TypeScriptとGo実装での強力な型サポート
* **広範なドキュメント**：各SDKの詳細なガイドとAPIリファレンス

### 機能比較

この表は、Sequence Headless Wallet、Web、Unity、およびUnreal SDKで利用可能な機能の高レベルな比較を提供します。

| 機能             | Typescript Wallet SDK | Web SDK (React)    | Unity SDK         | Unreal SDK        |
| :------------- | :-------------------- | :----------------- | :---------------- | :---------------- |
| **対象プラットフォーム** | Web (Headless)        | Web (React)        | Unity             | Unreal Engine     |
| **UIコンポーネント**  | いいえ                   | はい（事前構築、カスタマイズ可能）  | プレハブ              | プレハブ              |
| **認証**         | はい（メール、ソーシャル、ゲスト）     | はい（メール、ソーシャル、ゲスト）  | はい（メール、ソーシャル、ゲスト） | はい（メール、ソーシャル、ゲスト） |
| **ウォレットサポート**  | 組み込み / エコシステム         | 組み込み / エコシステム / 外部 | 組み込み / エコシステム     | 組み込み / エコシステム     |
| **インベントリ**     | いいえ                   | はい                 | はい                | はい                |
| **オンランプ**      | いいえ                   | ウォレットUI経由          | プレハブ経由            | プレハブ経由            |
| **統合スワップ**     | いいえ                   | はい                 | はい                | はい                |
| **チェックアウト**    | いいえ                   | はい                 | はい                | はい                |

**注意：**「Core SDK経由」または同様の表記がある機能は、特定のSDKが直接その機能を実装していなくても、基盤となるSequenceサービスまたはコアTypeScript SDKを通じてその機能を提供していることを意味し、多くの場合、統合されたウォレット体験によって実現されています。

## はじめに

あなたのプラットフォームと開発ニーズに最適なSDKを選択してください：

* Webアプリケーションの場合：
  * 使用 [Web-SDK](/sdk/web/overview) Reactアプリケーション向け
  * 使用 [TypeScript/JavaScript SDK](/sdk/typescript/overview) コア機能向け

ゲーム開発の場合：

* 使用 [Unity SDK](/sdk/unity/overview) Unityゲーム向け
* 使用 [Unreal SDK](/sdk/unreal/introduction) Unreal Engineゲーム向け
  バックエンドサービスについては、[Go SDK](/sdk/go/overview)

  モバイルアプリについては、[React Native SDK](/sdk/mobile)
