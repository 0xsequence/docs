UnrealまたはUnityとの統合については、[Unreal](/sdk/unreal/quickstart)または[Unity](/sdk/unity/installation)をご覧ください。

<Steps>
  <Step title="スターターリポジトリのクローン">
    ターミナルで以下のコマンドを実行してスターターリポジトリをクローンしてください。

    <CodeGroup>
      ```bash Javascript
      npx sequence-cli boilerplates create-email-embedded-wallet-react-starter
      ```

      テストキーの使用を選択してください。
    </CodeGroup>
  </Step>

  <Step title="接続と認証">
    `localhost:4444`にアクセスし、メールアドレスを入力して認証をお試しください。
  </Step>

  <Step title="埋め込みウォレットキーの追加">
    Sequence Builderにアクセスし、[埋め込みウォレットを設定](/solutions/builder/embedded-wallet/configuration)して、プロジェクト専用のキーを取得し、ユーザーを認証しましょう。.envファイルの既存の設定キーを置き換えるだけです。
  </Step>

  <Step title="Sequence Embedded Walletsを使いましょう！">
    認証が動作したら、よく使われる機能のサンプルもご覧いただけます：

    - [メッセージの署名](/sdk/headless-wallet/use-wallets#sign-messages)
    - [トランザクションの送信](/sdk/headless-wallet/use-wallets#send-transactions)
    - [コントラクトの呼び出し](/sdk/headless-wallet/use-wallets#call-any-contract)
    - [バックエンドでユーザーを検証](/sdk/headless-wallet/verification#prerequisite-deployment-of-a-parent-wallet)
  </Step>
</Steps>

<Tip>
  現在は.envファイルのサンプル変数を使っていますが、[Sequence Builder](https://sequence.build/project/default/wallet/embedded-wallet)でプロジェクトを作成し、[独自のウォレット設定](/sdk/headless-wallet)を行って環境変数を更新することで、より多くのソーシャルプロバイダーでの認証やメールテンプレートのカスタマイズなどが可能です。
</Tip>
