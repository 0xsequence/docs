**トランザクションレシート：**&#x30C8;ランザクションは成功または失敗を示すレシートを返します。失敗したレシートはトランザクションの実行なしで生成されます。

**成功したトランザクションのレスポンス：**&#x6210;功したトランザクションはSentTransactionResponseを提供します。トランザクションハッシュにアクセスする前に、必&#x305A;**`isSentTransactionResponse`**&#x30BF;イプガードでこのレスポンスを確認してください。

```ts
import { Sequence, isSentTransactionResponse } from "@0xsequence/waas";

const waas = new SequenceWaaS(
  {
    projectAccessKey: `${process.env.PROJECT_ACCESS_KEY}`,
    waasConfigKey: `${process.env.WAAS_CONFIG_KEY}`,
    network: "arbitrum-nova",
  }
);

await waas.signIn({ idToken }, "Session name");

const tx = await waas.sendTransaction({
  chainId: 42161,
  transactions: [
    {
      to: "0xD72C236Be524Ec24F72329317e2785E687105B69",
      value: "0",
    },
  ],
});

if (isSentTransactionResponse(tx)) {
  console.log(tx);
}
```

**トランザクションの詳細を理解する**

**トランザクションレスポンスのフィールド：**

* **txHash**: ネットワークに送信されたトランザクションのハッシュ。
* **receipt.logs**: トランザクション中に出力されたログ。
* **receipt.status**: トランザクションがSUCCEEDEDまたはFAILEDを示します。
* **receipt.revertReason**: 該当する場合、トランザクションが取り消された理由を説明します。

完全なレスポンスオブジェクトを確認するには、[レスポンスペイロード](/solutions/wallets/embedded-wallet/architecture/response-payloads)セクションを参照してください。

## 失敗したトランザクションのレスポンス

In-Game Walletは、実際の送信前にネットワーク上でトランザクションをシミュレートすることで、失敗するトランザクションを事前に防ぎます。シミュレーションが失敗した場合、トランザクションは送信されず、FailedTransactionResponseが返されます。

完全なレスポンスオブジェクトを確認するには、ウォレットの[レスポンスペイロード](/solutions/wallets/embedded-wallet/architecture/response-payloads)セクションを参照してください。

```ts
import { Sequence, isSentTransactionResponse } from "@0xsequence/waas";

const waas = new SequenceWaaS(
  {
    projectAccessKey: `${process.env.PROJECT_ACCESS_KEY}`,
    waasConfigKey: `${process.env.WAAS_CONFIG_KEY}`,
    network: "arbitrum-nova",
  }
);

await waas.signIn({ idToken }, "Session name");

const tx = await waas.sendTransaction({
  chainId: 137,
  transactions: [
    {
      // This address always fails on Polygon, give it a try :D
      to: "0x839eE023B21f4Ffe2294025DE0AC30Ba7278D6Fd",
      value: "0",
    },
  ],
});

if (isSentTransactionResponse(tx)) {
  // ... This will never be executed
} else {
  // tx can only be `SentTransactionResponse` or `FailedTransactionResponse`
  console.log(tx);
}
```
