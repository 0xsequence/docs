[`@0xsequence/marketplace-sdk`](https://www.npmjs.com/package/@0xsequence/marketplace-sdk) は、NFT マーケットプレイスをアプリケーションに統合するためのツールです。NFT（ERC721）とセミファンジブルトークン（ERC1155）の両方に対応し、Sequence マーケットプレイスを簡単に扱えるインターフェースを提供します。

主な特徴：
- シンプルで直感的なマーケットプレイス API
- NFT のリスティングやオファーの作成・管理
- マーケットプレイスのアクティビティや注文の追跡
- すぐに使える React フックとコンポーネント
- TypeScript で構築
- React Query と Wagmi を活用した効率的なデータ取得

始める準備はできましたか？[はじめにガイド](/sdk/marketplace-sdk/getting-started) をご覧ください。
