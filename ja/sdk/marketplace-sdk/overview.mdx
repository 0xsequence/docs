なぜMarketplace SDKを使うのか？Sequenceのマーケットプレイスを手間なく統合できるシームレスな方法を提供します。強力なフックを活用して、マーケットプレイスデータの取得や出品、購入、オファー、オファー受諾などの主要な操作を実行できます。

# Marketplace SDKフック
Marketplace SDKは、マーケットプレイス機能をアプリケーションに効率的かつシームレスに統合できる最適化されたフックを提供します。これらのフックにより、スムーズで安全なユーザー体験の管理が可能です。

<CardGroup>
  <Card title="マーケットプレイスデータフック" icon="code" href="/sdk/marketplace-sdk/hooks/marketplace-data-hooks">
    主要なマーケットプレイスデータを取得するためのGETフックのコレクションです。UIでマーケットプレイス情報を取得・管理するのに便利です。
  </Card>

  <Card title="マーケットプレイスアクション" icon="code" href="/sdk/marketplace-sdk/hooks/marketplace-actions">
    マーケットプレイスアクションを統合して、出品、オファー、購入を管理しましょう。
  </Card>
</CardGroup>
