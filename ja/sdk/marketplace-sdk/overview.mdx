なぜMarketplace SDKを使うのか？Sequenceのマーケットプレイスを手間なく統合できるシームレスな方法を提供します。強力なフックを活用して、マーケットプレイスデータの取得や出品、購入、オファー、オファー受諾などの主要な操作を実行できます。

# Marketplace SDKフック
Marketplace SDKは、マーケットプレイス機能をアプリケーションに効率的かつシームレスに統合できる最適化されたフックを提供します。これらのフックにより、スムーズで安全なユーザー体験の管理が可能です。

## アクションフック
アクションフックは、アプリケーション内でマーケットプレイスとやり取りするために不可欠です。出品、オファー、購入や販売などの操作に役立ちます。

<CardGroup cols={2}>
  <Card title="useBuyModal" icon="code" href="/sdk/marketplace-sdk/hooks/marketplace-actions/useBuyModal">
    useBuyModalフックは、マーケットプレイスで販売中のNFTを購入するためのものです。購入手続きやトランザクションの実行を担当します。
  </Card>

  <Card title="useCreateListingModal" icon="code" href="/sdk/marketplace-sdk/hooks/marketplace-actions/useCreateListingModal">
    useCreateListingModalフックは、マーケットプレイスでアイテムを出品する際に使用します。新しい出品を作成・管理するために必要な機能を提供します。
  </Card>

  <Card title="useMakeOfferModal" icon="code" href="/sdk/marketplace-sdk/hooks/marketplace-actions/useMakeOfferModal">
    useMakeOfferModalフックは、ユーザーがNFTに対してオファーを出すことを可能にします。マーケットプレイス内でオファーの作成と送信をサポートします。
  </Card>

  <Card title="useSellModal" icon="code" href="/sdk/marketplace-sdk/hooks/marketplace-actions/useSellModal">
    useSellModalフックは、既存のオファーを受け入れてNFTを販売する際に使用します。販売手続きを完了するための機能を提供します。
  </Card>

  <Card title="useCancelOrder" icon="code" href="/sdk/marketplace-sdk/hooks/marketplace-actions/useCancelOrder">
    useCancelOrderフックは、Marketplaceで現在販売中のNFTのリスティングをキャンセルできるフックです。出品のキャンセル処理およびトランザクションの実行を行います。
  </Card>
</CardGroup>

## データフック
データフックは、アプリケーションで主要なMarketplace情報を取得するために不可欠です。リスティングやオファーなど、UIに必要なデータの取得に役立ちます。
