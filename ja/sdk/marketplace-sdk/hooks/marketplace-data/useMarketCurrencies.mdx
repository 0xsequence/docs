---
title: useMarketCurrencies
description: useMarketCurrenciesフックは、BuilderからMarketplace内の特定コントラクトアドレスで受け入れられている通貨リストを取得します。各通貨には、名前、contractAddress、シンボル、小数点、画像などの情報が含まれます。
sidebarTitle: useMarketCurrencies
---

```ts
import { useMarketCurrencies } from "@0xsequence/marketplace-sdk/react";

## Into your React component:

const { data: currenciesData } = useMarketCurrencies({
	chainId: 137,
	collectionAddress: "0x0e5566a108e617baedbebb44e3fcc7bf03e3a839",
	includeNativeCurrency: true,
	query: {
		enabled: true,
	},
});

console.log(currenciesData);
```
