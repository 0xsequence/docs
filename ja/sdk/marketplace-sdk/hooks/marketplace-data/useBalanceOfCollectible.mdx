## パラメータ

| 名前     | 型 | 説明         |
| ------ | - | ---------- |
| `args` |   | 残高取得のための引数 |

## 戻り値
残高データを含むクエリ結果

## 例

```typescript
const { data, isLoading, error } = useBalanceOfCollectible({
  collectionAddress: "0x123...",
  collectableId: "1",
  userAddress: "0x456...",
  chainId: 1,
  query: {
    enabled: true,
    refetchInterval: 10000,
  },
});
```
