---
title: useCollection
description: useCollectionフックは、BuilderからMarketplace内の特定コレクションアドレスに関する情報を取得します。返されるデータには、chainId、アドレス、シンボルなどの詳細が含まれます。
sidebarTitle: useCollection
---

```ts
import { useCollection } from "@0xsequence/marketplace-sdk/react";

## Into your React component:

const { data: collectionData } = useCollection({
  chainId: 137,
  collectionAddress: "0x0e5566a108e617baedbebb44e3fcc7bf03e3a839",
  query: {
    enabled: true,
  },
});

console.log(collectionData);
```
