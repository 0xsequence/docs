<Info>
  このパッケージは[Connect SDK](/sdk/web/wallet-sdk/ecosystem/getting-started)に依存しています。
</Info>

<Steps>
  <Step title="Connect SDKのインストールとセットアップ">
    手順は[こちら](/sdk/web/wallet-sdk/ecosystem/getting-started)をご覧ください。
  </Step>

  <Step title="Checkout SDKのインストール">
    ```bash
        npm install @0xsequence/checkout@v6-beta
        # or
        pnpm install @0xsequence/checkout@v6-beta
        # or
        yarn add @0xsequence/checkout@v6-beta
    ```
  </Step>

  <Step title="アプリをSequenceCheckoutProviderでラップしてください">
    ```typescript [main.tsx]
        import React from "react";
        import ReactDOM from "react-dom/client";
        import "./index.css";

        import App from "./App";
        import { config } from "./config";
        import { SequenceConnect } from "@0xsequence/connect";
        import { SequenceCheckoutProvider } from "@0xsequence/checkout";

        function Dapp() {
        return (
            <SequenceConnect config={config}>
                <SequenceCheckoutProvider>
                    <App />
                </SequenceCheckoutProvider>
            </SequenceConnect>
        );
        }

        ReactDOM.createRoot(document.getElementById("root")!).render(
        <React.StrictMode>
            <Dapp />
        </React.StrictMode>
        );
    ```
  </Step>

  <Step title="使い始める">
    これで、[useCreditCardCheckoutModal](/sdk/web/checkout-sdk/hooks/useCreditCardCheckoutModal) のようなフックを使い始めることができます。
  </Step>
</Steps>
