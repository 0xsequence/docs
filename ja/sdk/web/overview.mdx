私たちは、ユーザーにスムーズで安全なオンボーディング体験を提供します。人気のある[wagmi](https://wagmi.sh/) ライブラリを使用して、web3の世界での可能性を解き放ちましょう。

Web SDKは主に3つのパッケージで構成されています：

* @0xsequence/connect
* @0xsequence/checkout
* @0xsequence/hooks
* @0xsequence/wallet-widget

## 主な機能

* **ユニバーサル接続**：Facebook、Google、Discordなどの人気のあるソーシャルログインを通じてシームレスに接続！ユーザーは安全でスムーズなオンボーディングプロセスを体験できます。

* **組み込みウォレット**：Web SDKの接続モーダル内にある非カストディアル組み込みウォレットを活用して、オプションのポップアップやモーダルなどのweb3をバックグラウンドに置き、ユーザーが体験に集中できるようにします。

* **外部ウォレットの統合**：WalletConnectやMetaMaskなどの主要なweb3ウォレットと簡単に統合。わずか数クリックでブロックチェーンのパワーを解き放ちましょう！

* **Web3インベントリ**：ユーザーにweb3対応のインベントリを提供し、アプリケーション内でコインやコレクティブルを管理できるようにします。

* **NFTチェックアウト**：Sequence Checkout内でクレジットカードまたは暗号通貨を使用してコレクティブルを購入できるようにします。

* **カスタマイズ性**：Web SDK接続モーダルを独自のロゴ、カラースキームでブランド化し、対応したいソーシャルプロバイダーやウォレットで設定できます。

私たちのWeb SDKの可能性を[デモ](https://0xsequence.github.io/web-sdk/)で体験してみてください！
