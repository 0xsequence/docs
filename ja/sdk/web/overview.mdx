人気の[wagmi](https://wagmi.sh/)ライブラリを基盤とした堅牢なツールで、ユーザーにスムーズかつ安全なオンボーディング体験を提供します。Web3の世界で多彩な可能性を広げましょう。

Web SDKは主に4つのパッケージで構成されています：
- @0xsequence/connect
- @0xsequence/checkout
- @0xsequence/hooks
- @0xsequence/wallet-widget

## 主な特徴

-   **ユニバーサル接続**：Facebook、Google、Discordなどの人気ソーシャルログインでシームレスに接続！ユーザーはスムーズで安全なオンボーディングを体験できます。

-   **組み込みウォレット**：Web SDKの接続モーダル内で非カストディアルな組み込みウォレットを活用し、Web3を背景に隠して（任意のポップアップやモーダルなど）、ユーザーが体験に集中できるようにします。

-   **外部ウォレット連携**：WalletConnectやMetaMaskなど主要なWeb3ウォレットとも簡単に連携可能。数クリックでブロックチェーンの力を活用できます！

-   **Web3インベントリ**：ユーザーが自分のコインやコレクティブルをアプリ内で管理できるWeb3対応インベントリを提供します。

-   **NFTチェックアウト**：Sequence Checkoutでクレジットカードや暗号資産を使ってコレクティブルを購入できるようにします。

-   **カスタマイズ性**：Web SDKの接続モーダルに自社ロゴやカラースキームを設定し、利用したいソーシャルプロバイダーやウォレットを自由に構成できます。

[デモ](https://0xsequence.github.io/web-sdk/)を試して、Web SDKの可能性を体験してください！
