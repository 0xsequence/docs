title: useConfig
description: プロバイダーコンテキストからマーケットプレイスSDKの設定を取得します。このフックは、MarketplaceSdkProviderに渡されたコアSDK設定へアクセスできます。APIエンドポイントやチェーン設定、子コンポーネントやフックで必要となるSDK全体の設定が含まれます。
sidebarTitle: useConfig
---

## 戻り値
プロバイダーコンテキストから取得したSDK設定オブジェクト全体

## 例
基本的な使い方：

```typescript
const config = useConfig();
console.log(config.marketplaceApiUrl);
console.log(config.projectId);
```

APIコールでのconfig利用例：

```typescript
const config = useConfig();

const response = await fetch(`${config.marketplaceApiUrl}/collections`, {
  headers: {
    'X-Project-ID': config.projectId
  }
});
```
