## 新しい依存関係のインストール

すべてのフックライブラリの名前が変更されました。Web SDKを使い始めるために必要な新しい依存関係は以下の通りです：

<CodeGroup>
  ```bash New 
  npm install @0xsequence/connect wagmi ethers@6.13.0 viem 0xsequence @tanstack/react-query
  # or
  pnpm install @0xsequence/connect wagmi ethers@6.13.0 viem 0xsequence @tanstack/react-query
  # or
  yarn add @0xsequence/connect wagmi ethers@6.13.0 viem 0xsequence @tanstack/react-query
  ```

  ```bash Old
  npm install @0xsequence/kit @0xsequence/waas wagmi ethers@6.13.0 viem 0xsequence @0xsequence/design-system@^1.9.0 motion @tanstack/react-query
  # or
  pnpm install @0xsequence/kit @0xsequence/waas wagmi ethers@6.13.0 viem 0xsequence @0xsequence/design-system@^1.9.0 motion @tanstack/react-query
  # or
  yarn add @0xsequence/kit @0xsequence/waas wagmi ethers@6.13.0 viem 0xsequence @0xsequence/design-system@^1.9.0 motion @tanstack/react-query
  ```
</CodeGroup>

## アプリの設定

設定に関する主な違い：

<CodeGroup>
  ```tsx New
  import React from "react";
  import ReactDOM from "react-dom/client";
  import "./index.css"; 

  import App from "./App";
  import { SequenceConnect } from "@0xsequence/connect";
  import { SequenceCheckoutProvider } from "@0xsequence/checkout"
  import { config } from "./config";

  function Dapp() {
    return (
      <SequenceConnect config={config}>
        <SequenceCheckoutProvider>
            <App />
        </SequenceCheckoutProvider>
      </SequenceConnect>
    );
  }

  ReactDOM.createRoot(document.getElementById("root")!).render(
    <React.StrictMode>
      <Dapp />
    </React.StrictMode>
  );
  ```

  ```tsx Old
  import React from "react";
  import ReactDOM from "react-dom/client";
  import "./index.css";

  import App from "./App";
  import { SequenceKit } from "@0xsequence/kit";
  import { KitCheckoutProvider } from "@0xsequence/kit-checkout";
  import { config } from "./config";
  import '@0xsequence/design-system/styles.css'

  function Dapp() {
    return (
      <SequenceKit config={config}>
        <KitCheckoutProvider>
          <App />
        </KitCheckoutProvider>
      </SequenceKit>
    );
  }

  ReactDOM.createRoot(document.getElementById("root")!).render(
    <React.StrictMode>
      <Dapp />
    </React.StrictMode>
  );
  ```
</CodeGroup>

* `@0xsequence/kit`→`@0xsequence/connect`認証用
* `@0xsequence/kit`→`@0xsequence/wallet-widget`ウォレットUI用
* `@0xsequence/kit-checkout`→`@0xsequence/checkout`
* を使用する際に`@0xsequence/design-system/styles.css`を手動でインポートする必要はありません`@0xsequence/connect`
* `@0xsequence/connect`は`@0xsequence/design-system`v2と互換性があります
* `KitProvider`は現在`SequenceConnectProvider`
* `KitCheckoutProvider`は現在`SequenceCheckoutProvider`
* `KitWalletProvider`は現在`SequenceWalletProvider`

## アプリの接続

<CodeGroup>
  ```tsx New
  import './App.css'
  import { useOpenConnectModal } from '@0xsequence/connect'

  function App() {
    const {setOpenConnectModal} = useOpenConnectModal()
    
    return (
      <>
        <button onClick={() => setOpenConnectModal(true)}>Connect</button>
      </>
    )
  }

  export default App
  ```

  ```tsx Old
  import './App.css'
  import { useOpenConnectModal } from '@0xsequence/kit'

  function App() {
    const {setOpenConnectModal} = useOpenConnectModal()
    
    return (
      <>
        <button onClick={() => setOpenConnectModal(true)}>Connect</button>
      </>
    )
  }

  export default App
  ```
</CodeGroup>
