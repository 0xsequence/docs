## 概要
[@0xsequence/connect v6](https://www.npmjs.com/package/@0xsequence/connect/v/6.0.0)はEcosystem Wallet用のReact Hooks SDKです。
ソーシャル認証、パスキー、アカウント復旧、ウォレット連携、セッション・権限管理を組み合わせ、ユーザーにWeb2のような体験を提供します。
バンドルサイズも小さく、DAppに完全なノンカストディアル型エコシステムスマートウォレット体験を最速で追加できます。

### 主な特徴：

- ソーシャル認証（メール、Google、Apple、Epic Games、Twitchなど）
- パスキー対応
- アカウント復旧機能
- ウォレット連携
- スマートセッション管理
- カスタマイズ可能なテーマ設定
- 組み込みUIコンポーネント
