## インストール
Sequence.jsは、Ethereumチェーン向けのモジュラー型web3スタックおよびスマートウォレットパッケージです。ethers.jsに似ていますが、ブロックチェーン対応ゲーム開発に必要な機能を効率化したSequenceスタック向けに最適化されています。sequence.js SDKは複数のサブパッケージで構成されており、`0xsequence`というメタパッケージとしてまとめられています。
SDKをプロジェクトに追加するには、package.jsonに0xsequenceを依存関係として追加してください。

```sh
npm install 0xsequence ethers
```

または

```sh
pnpm install 0xsequence ethers
```

または

```sh
yarn add 0xsequence ethers
```

プロジェクトで特定のサブパッケージのみ必要な場合は、
`npm add @0xsequence/<sub-package name>`で直接依存させることもできます。

package.jsonは次のようになります（実際のバージョンは異なる場合があります）：

```json
{
  "name": "my-dapp",
  ...
  "dependencies": {
    "0xsequence": "^1.0.0",
    "ethers": ^5.7.0"
    ...
  }
}
```

#### CDN配信によるネイティブJSとしての利用
オプションとして、`0xsequence`はJSスクリプトソース（UMD配布）としてもインストールできます：

```js
<script src="https://cdn.ethers.io/lib/ethers-5.7.umd.min.js"></script>
<script src="https://unpkg.com/0xsequence@latest/dist/0xsequence.umd.min.js"></script>
```
