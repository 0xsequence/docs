## インストール

Sequence.jsは、Ethereumチェーン用のモジュラーweb3スタックとスマートウォレットパッケージです。これはethers.jsに似ていますが、ブロックチェーン対応のゲーム開発に必要な一般的な機能を効率化するSequenceスタックに合わせて調整されています。sequence.js SDKは、いくつかのサブパッケージで構成されており、単一のメタパッケージとして`0xsequence`にまとめられています。
SDKをプロジェクトに追加するには、package.jsonに0xsequenceの依存関係を追加してください：

```sh
npm install 0xsequence ethers
```

または

```sh
pnpm install 0xsequence ethers
```

または

```sh
yarn add 0xsequence ethers
```

プロジェクトが特定のサブパッケージのみを必要とする場合は、それらを直接依存関係として追加することもできます：`npm add @0xsequence/<sub-package name>`。

package.jsonは以下のようになるはずです（実際のバージョンは異なる場合があります）：

```json
{
  "name": "my-dapp",
  ...
  "dependencies": {
    "0xsequence": "^1.0.0",
    "ethers": ^5.7.0"
    ...
  }
}
```

#### ネイティブJSとしてのCDN配布

オプションとして、`0xsequence`はJSスクリプトソース（UMD配布）として以下のようにインストールできます：

```js
<script src="https://cdn.ethers.io/lib/ethers-5.7.umd.min.js"></script>
<script src="https://unpkg.com/0xsequence@latest/dist/0xsequence.umd.min.js"></script>
```
