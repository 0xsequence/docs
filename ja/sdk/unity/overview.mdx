# はじめに

Sequence Unity Embedded Wallet SDKは、完全なSequence [Embedded Wallet](/sdk/headless-wallet)と[Indexer](/api-references/indexer/overview)のUnityゲーム向け統合を提供し、Unity用に特別に構築された独自のSequenceEthereumライブラリと統合されています。そうです、Nethereumは必要ありません！

このSDKは[セマンティックバージョニング](https://semver.org/)（`major.minor.patch`）に従います。破壊的な変更は常に`major`バージョンの増加を引き起こし、破壊的でない新機能は`minor`バージョンの増加を引き起こし、バグ修正は`patch`バージョンの増加を引き起こします。

## 要件

Unity 2021.3.6f1以降

* Android
* iOS
* PCスタンドアロン -> （ソーシャルサインイン使用時はMonoビルドのみ -> システムコマンドはIL2CPPビルドで例外をスローします -> [OpenIdAuthentication.PlatformSpecificSetup](https://github.com/0xsequence/sequence-unity/blob/master/Assets/Sequence/SequenceSDK/Authentication/OpenIdAuthenticator.cs#L231)を参照）
* Macスタンドアロン -> （ソーシャルサインイン使用時はMonoビルドのみ -> 私たちのテストでは、MacOSは最初にシステムコマンドを実行しない限りカスタムURLスキームを自動的に認識しません。これらのシステムコマンドはMonoでのみ動作します -> [OpenIdAuthentication.PlatformSpecificSetup](https://github.com/0xsequence/sequence-unity/blob/master/Assets/Sequence/SequenceSDK/Authentication/OpenIdAuthenticator.cs#L231)を参照）
* WebGL
