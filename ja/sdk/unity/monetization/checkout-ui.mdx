SDKには、開始時に使用してゲーム用にカスタマイズすることをお勧めするデフォルトのチェックアウトUIが付属しています。

## 組み込みチェックアウトUIの使用

私たちは`CheckoutPanel`をBoilerplatesアセンブリの一部として提供しています。他のボイラープレートと同様に、これらは`BoilerplatesFactory`を通じてアクセスします。[Bootstrap Your Game](/sdk/unity/bootstrap_game)で詳細をご確認ください。

簡単に言えば、`CheckoutPanel`プレハブは`Assets/**/Resources/Checkout/CheckoutPanel`にあり、簡単に`CheckoutPanel`を使用して`BoilerplateFactory`を開くことができます：

```
(CheckoutPanel, CheckoutPage) checkoutUi = BoilerplateFactory.OpenCheckoutPanel(parentTransform, checkoutHelperInstance, fiatCheckoutInstance, optionalOnCloseAction);
```

ここでcheckoutHelperInstanceは`ICheckoutHelper`インターフェースを実装し、fiatCheckoutInstanceは`IFiatCheckout`インターフェースを実装します。

現在、`ICheckoutHelper`の実装が2つあります：`NftCheckout`と`ERC1155SaleCheckout`です。これらはそれぞれセカンダリーセールスマーケットプレイスとERC1155プライマリーセールス用です。ERC721プライマリーセールスの実装は近日公開予定です。

`SequenceCheckout`は現在利用可能な唯一の`IFiatCheckout`の実装です。セカンダリーセールスマーケットプレイスとERC1155プライマリーセールスの両方をサポートしています。ERC721プライマリーセールスのサポートは近日`SequenceCheckout`に追加予定です。
