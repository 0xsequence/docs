## パッケージマネージャー - 推奨

### OpenUPM

1. [OpenUPMのインストール](https://openupm.com/#get-started-with-cli-optional)
2. Unityプロジェクトのルートでコマンドラインを開く
3. `openupm add xyz.0xsequence.waas-unity`

### またはパッケージマネージャーUIを使用

1. マシンにGit 2.14.0以上がインストールされていることを確認してください
2. パッケージマネージャーを開く（Window > Package Manager）
3. パッケージマネージャーウィンドウの「+」アイコンをクリック > 「Add package from git URL...」
4. このURLを貼り付けて、AddをクリックするかキーボードのEnterを押してください`https://github.com/0xsequence/sequence-unity.git?path=/Packages/Sequence-Unity`

### セットアップ

1. パッケージマネージャーを開く（Window > Package Manager）
2. パッケージマネージャーから「Samples」をクリック
3. サンプルから「Setup」をインポート
   ![Setup](/images/unity/unity-import-setup.png)

a) これにより、SequenceConfigスクリプタブルオブジェクトを含むResourcesフォルダがインポートされます（詳細はセットアップを参照）。

4. インポート`TMP Essentials`（まだの場合）。注：シーン内にTextMeshProオブジェクトがある状態でPlayまたはBuildを試みると、Unityが自動的にプロンプトを表示します。

![Import TMPro](/images/unity/unity-import-tmpro.png)

### サンプル

このドキュメントに加えて、SDKの使用方法を理解するためのいくつかのサンプルも提供しています。

これらは以下の方法で見つけることができます：

1. パッケージマネージャーを開く
2. SDK「Sequence WaaS SDK」を見つける
3. 「Samples」をクリック
4. インストールしたいサンプルの横にある「Import」をクリックします。これによりAssetsの下にSamplesフォルダが作成され、そこにサンプルがインポートされます。サンプルはAssetsフォルダ内にあるため、完全に変更可能です。

![Samples](/images/unity/unity-package-manager-samples.png)

2番目のサンプル「Demo Scene」は、SDKの機能の一部を紹介するサンプルシーンです。これはドキュメントの有用な補足となります。

3番目のサンプル「Useful Scripts」には、SDKの統合を容易にする便利なスクリプトとPrefabが含まれています。

## マニュアル

パッケージマネージャーを介してインストールすると、SDKは読み取り専用状態になります - 注：パッケージマネージャーを使用している場合でも、UIを変更することは可能です。

ほとんどのユースケースでは、これが理想的です。ただし、一部の上級ユーザーは、必要に応じてSDKを編集する柔軟性を求めることがあることも認識しています。

このルートを選択した場合、SDKの更新がより困難になることにご注意ください。これは、行った変更が上書きされ、これらの変更はバージョン管理システムで簡単に見落とされる可能性があるためです。一般的に、パッケージマネージャーを介してSDKを読み取り専用モードでインポートし、必要に応じて拡張するためのラッパーを作成する方が安全だと考えていますが、お客様が最適と考える方法でSDKを使用していただくことを推奨しています。

1. 追加[Newtonsoft.json](https://docs.unity3d.com/Packages/com.unity.nuget.newtonsoft-json@3.0/manual/index.html)をプロジェクトに（まだない場合）パッケージマネージャーを介して追加します。パッケージマネージャーウィンドウの「+」アイコンをクリック > 「Add package by name...」をクリックして追加`com.unity.nuget.newtonsoft-json`。
2. ダウンロード[Sequence Unity SDK UnityPackageの最新リリース](https://github.com/0xsequence/sequence-unity/releases)
3. ドラッグ`.unitypackage`ファイルをプロジェクトウィンドウにドラッグしてインポート
4. インポート`TMP Essentials`（まだの場合）。注：シーン内にTextMeshProオブジェクトがある状態でPlayまたはBuildを試みると、Unityが自動的にプロンプトを表示します。
