`SequenceConnect`オブジェクトを作成し、ユーザーが接続できるチェーンとエコシステムを指定します。

```csharp
EcosystemType ecosystem = EcosystemType.Sequence;
SequenceConnect connect = new SequenceConnect(ecosystem);
```

### 組み込みデモをお試しください
Unityのパッケージマネージャーから`Ecosystem Wallet Demo`サンプルをインポートしてください。また、統合の詳細については、ボイラープレートスクリプト`SequenceEcosystemWalletWindow`もご確認ください。
