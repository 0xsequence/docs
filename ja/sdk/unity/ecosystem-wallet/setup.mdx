### エコシステムの設定
まず、[クイックスタートガイド](/sdk/unity/quickstart)に従って SequenceConfig ファイルを作成するか、Unity で `Create` > `Sequence` > `SequenceConfig` から新しいファイルを作成してください。

次に、必要な項目を入力します：
- **Builder API キー：** [Builder プロジェクト](https://sequence.build/)から取得したプロジェクトアクセスキー。
- **Wallet App Url：** 接続したいエコシステムの URL。不明な場合は、デフォルトで `https://v3.sequence-dev.app` をご利用ください。

### 組み込みデモをお試しください
Unityのパッケージマネージャーから`Ecosystem Wallet Demo`サンプルをインポートしてください。また、統合の詳細については、ボイラープレートスクリプト`SequenceEcosystemWalletWindow`もご確認ください。
