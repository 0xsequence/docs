<Steps>
  <Step title={"Install Sequence's Beta SDK"}>
    [SDKのGitHubリリースからv5をダウンロード](https://github.com/0xsequence/sequence-unity/releases/tag/v5-beta)し、Ecosystem Walletsにアクセスしましょう。`.unitypackage` ファイルとしてダウンロードし、Unityプロジェクトに追加してください。

    <Warning>
      Unity 向け Ecosystem Wallet は現在ベータ版のため、時折バグが発生する場合がありますのでご注意ください。
    </Warning>
  </Step>

  <Step title={"Configure your Ecosystem"}>
    まず、[SDK の GitHub リリースから v5 をダウンロード](https://github.com/0xsequence/sequence-unity/releases/tag/v5-beta)し、Ecosystem Wallets を利用できるようにします。`.unitypackage` ファイルとしてダウンロードし、Unity プロジェクトに追加してください。

    [クイックスタートガイド](/sdk/unity/quickstart)に従って SequenceConfig ファイルを作成するか、Unity の `Create` > `Sequence` > `SequenceConfig` から新規ファイルを作成してください。

    次に、必要な項目を入力します：

    - **Builder API キー：** [Builder プロジェクト](https://sequence.build/)から取得したプロジェクトアクセスキー。
    - **Wallet App Url：** 接続したいエコシステムの URL。不明な場合は、デフォルトで `https://v3.sequence-dev.app` をご利用ください。
  </Step>

  <Step title={"Try our Built-In Demo"}>
    Unity のパッケージマネージャーから `Ecosystem Wallet Demo` サンプルをインポートしてください。また、統合の詳細については、ボイラープレートスクリプト `SequenceEcosystemWalletWindow` もご参照ください。

  </Step>
</Steps>
