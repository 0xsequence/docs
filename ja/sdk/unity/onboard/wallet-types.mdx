# ウォレットタイプ

Sequence Unity SDKは2つのウォレットタイプをサポートしています：

## 組み込みウォレット

[組み込みウォレット](/sdk/headless-wallet)は、ゲームに組み込まれるウォレットです。これはユーザーが管理する非カストディアルウォレットです。ユーザーはこのウォレットでトランザクションやメッセージに署名できます。組み込みウォレットは、ほとんどのゲームで推奨されるウォレットタイプです。

## ユニバーサルウォレット

この[ユニバーサルウォレット](/sdk/typescript/guides/frontend/connect-wallet)は、外部ウォレットに接続できるウォレットです。これはユーザーが管理するノンカストディアルウォレットです。ユーザーはこのウォレットでトランザクションやメッセージに署名できます。ユニバーサルウォレットは、外部ウォレットをサポートしたいゲーム向けに推奨されています。
