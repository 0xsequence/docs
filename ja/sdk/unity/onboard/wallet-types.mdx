# ウォレットタイプ
Sequence Unity SDKは2種類のウォレットタイプをサポートしています。

## 埋め込み型ウォレット
[Embedded Wallet](/sdk/headless-wallet) は、ゲーム内に組み込まれるウォレットです。ユーザー自身が管理するノンカストディアル型ウォレットで、トランザクションやメッセージへの署名が可能です。ほとんどのゲームでは、このEmbedded Walletの利用を推奨しています。

## ユニバーサルウォレット
[Universal Wallet](/sdk/typescript/guides/frontend/connect-wallet) は、外部ウォレットとも接続できるウォレットです。ユーザー自身が管理するノンカストディアル型ウォレットで、トランザクションやメッセージへの署名が可能です。外部ウォレットのサポートが必要なゲームにはUniversal Walletを推奨します。
