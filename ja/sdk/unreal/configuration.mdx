title: 設定
description: web3ゲーミング向けSequenceインフラストラクチャスタックのUnreal SDKセットアップのドキュメント。
---

<Steps>
  <Step title="Private Dependency Modules">
    プロジェクトのBuild.csファイル（[場所の確認](https://forums.unrealengine.com/t/where-are-build-cs-and-target-cs/302665)）で、プライベート依存モジュールに「SequencePlugin」が含まれていることを確認してください：

    ```cpp
        public SequenceTest(ReadOnlyTargetRules Target) : base(Target)
        {
            PCHUsage = PCHUsageMode.UseExplicitOrSharedPCHs;
    		PublicDependencyModuleNames.AddRange(new string[] { "Core", "CoreUObject", "Engine", "InputCore" });
    		PrivateDependencyModuleNames.AddRange(new string[] { "SequencePlugin" });
    	}
    ```
  </Step>

  <Step title="Create a config file">
    移動して`[YourProjectDirectory]/Config`、ファイル名`SequenceConfig.ini`を作成し、以下の内容を追加してください：

    ```
    [/Script/Sequence.Config]

    WaaSConfigKey = "Go to https://sequence.build/ under Onboard > Embedded Wallet"
    ProjectAccessKey = "Go to https://sequence.build/ under Settings > API Keys"
    GoogleClientID = "" 
    AppleClientID = "" 
    FacebookClientID = "" 
    DiscordClientID = ""
    PlayFabTitleID = ""
    RedirectUrl = "https://api.sequence.app"
    ```
  </Step>
</Steps>

<Warning>
  忘れずに[Builderで組み込みウォレットの設定を構成してください](/solutions/builder/embedded-wallet/configuration)！
</Warning>
