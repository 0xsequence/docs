<Steps>
  <Step title="Unreal SDKのインストール">
    SequenceのUnreal SDKの最新バージョンを[GitHub](https://github.com/0xsequence/sequence-unreal/releases)からダウンロードしてインストールするか、[Unreal Project Boilerplate](https://github.com/0xsequence/unreal-embedded-wallet-boilerplate)をダウンロードして新しいプロジェクトを一から始めてください。
  </Step>

  <Step title="SDK の設定">
    [Sequence Builder](https://sequence.build) にサインインし、プロジェクトを作成します。

    `Onboard > Embedded Wallet`から`Project Access Key`と`WaaS Configuration Key`を取得し、[プロジェクトのSequence設定ファイル](/sdk/unreal/v1/configuration)に追加してください。
  </Step>

  <Step title="Eメールログインの統合">
    まず、メールOTPを使ってウォレットセッションを確立します。`Start Email Login Async`ノードを実行して、指定したメールアドレスにワンタイムパスワードを送信します。次に、受け取ったパスワードを使い、`Confirm Email Login with Code Async`ノードでログインを完了します。

    最初の機能を統合した後は、Google、Apple、PlayFabなどの[追加のログインプロバイダー](/sdk/unreal/v1/authentication)も利用できます。

    <Frame caption="Email Otp">
      <img src="/images/unreal/v1/email_otp.png" />
    </Frame>
  </Step>

  <Step title="Sequence の機能をテストする">
    SequenceのUnreal SDKには、ゲームをすぐに始められるさまざまなBlueprintが含まれています。すべての設定が完了したら、インベントリ、ウォレットビュー、ゲーム内ストアを表示するウィジェットを作成できます。[ウォレットビューの統合方法](/sdk/unreal/v1/bootstrap_game#my-wallet-view)もご覧ください。3つのBlueprintノードだけで資金の表示や送信が可能です。

    <Frame caption="Feature Wallet Demo">
      <img src="/images/unreal/v1/feature_wallet_demo.gif" />
    </Frame>
  </Step>
</Steps>
