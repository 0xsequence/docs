<Steps>
  <Step title="Unreal SDKのインストール">
    [Epic Games Marketplace](https://www.unrealengine.com/marketplace/en-US/product/sequence-embedded-wallets)からSDKをダウンロードしてください。
    詳細は[インストール手順](/sdk/unreal/installation)をご覧ください。または、
    [プロジェクト用ボイラープレート](https://github.com/0xsequence/unreal-embedded-wallet-boilerplate)を利用して、新しいUnrealプロジェクトをすぐに開始できます。
  </Step>

  <Step title="SDK の設定">
    [Sequence Builder](https://sequence.build) にサインインし、プロジェクトを作成します。

    下記の手順で Builder から設定ファイルをダウンロードしてください。

    または、`Onboard > Embedded Wallet` から `Project Access Key` と `WaaS Configuration Key` を取得し、
    [プロジェクトのSequence設定ファイル](/sdk/unreal/configuration)に追加してください。

    <Frame>
      <img src="/images/unreal/builder_config_file.png" />
    </Frame>
  </Step>

  <Step title="Eメールログインの統合">
    まず、メールOTPを使ってウォレットセッションを確立します。`Start Email Login Async`ノードを実行して、指定したメールアドレスにワンタイムパスワードを送信します。次に、受け取ったパスワードを使い、`Confirm Email Login with Code Async`ノードでログインを完了します。

    最初の機能が統合できたら、[追加のログインプロバイダー](/sdk/unreal/authentication)（Google、Apple、PlayFabなど）もご利用いただけます。

    <Frame caption="Email Otp">
      <img src="/images/unreal/email_otp.png" />
    </Frame>
  </Step>

  <Step title="Sequence の機能をテストする">
    SequenceのUnreal SDKには、ゲームをすぐに始められるさまざまなボイラープレートが含まれています。
    すべての設定が完了したら、インベントリ、ウォレットビュー、ゲーム内ストアを表示するウィジェットを作成できます。
    [ウォレットビューの統合方法](/sdk/unreal/bootstrap_game#my-wallet-view)もご覧ください。3つのBlueprintノードだけで、資金の表示や送信が可能です。

    <Frame caption="Feature Wallet Demo">
      <img src="/images/unreal/feature_wallet_demo.gif" />
    </Frame>
  </Step>
</Steps>
