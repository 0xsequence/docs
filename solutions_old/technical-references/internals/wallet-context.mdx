The Sequence Wallet stack utilises a different "context" for each version of the wallet.
This context is a set of addresses that are used to interact with the wallet. 
he context is used to deploy the wallet, interact with the wallet, and to interact with the wallet's modules.

For more information on the wallet context, see the specific version of the wallet you are using:
- [v1 Wallet Context](/solutions_old/technical-references/internals/v1/wallet-context)
- [v2 Wallet Context](/solutions_old/technical-references/internals/v2/wallet-context)
